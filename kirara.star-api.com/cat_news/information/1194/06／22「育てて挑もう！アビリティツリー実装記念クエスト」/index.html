「育てて挑もう！アビリティツリー実装記念クエスト」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

6月24日より
「育てて挑もう！アビリティツリー実装記念クエスト」を開催いたします。

開催期間

6月24日 17:00 〜 7月13日 23:59

イベント概要

本イベントは、収集クエストと【超難関】クエストで構成されています。
新機能「アビリティツリー」でキャラクターを強化して、
【超難関】クエストに挑戦してみましょう！

アビリティツリーに関する詳細は、
別途掲載のおしらせ「アビリティツリーの遊び方」をご確認ください。

育成が捗る収集クエストが登場！

開放期間
前半：6月24日 17:00 〜 7月13日 23:59
後半：7月 2日 0:00 〜 7月13日 23:59

収集クエストは前半と後半に分かれており、
前半ではキャラクターの強化に必要な新芽・大種、
後半ではぶきの強化に必要なクレスト・シンボルがドロップします。
この機会にたくさんの強化素材を集めましょう！

新登場！【超難関】クエスト！

開放期間
7月 2日 0:00 〜 7月13日 23:59

【超高難易度】クエストを超える難しさの【超難関】クエストが新登場します。
また、初回クリア報酬として「ポルカとコルクからの贈り物」を入手でき、
これを所持していることで8月5日に特別な称号を取得できます。
アビリティツリーでキャラクターを育成して挑んでみましょう！

イベントミッション実施！

実施期間
6月24日 17:00 〜 7月13日 23:59

限定交換期間
6月24日 17:00 〜 7月20日13:59

「イベント復刻コールチケット」最大9枚GET！

実施期間中、「アビリティツリー実装記念クエスト」の累計クリア数に応じて
過去のイベント報酬で登場した★4キャラクターと交換できる
「イベント復刻コールチケット」が最大9枚入手できます。

「イベント復刻コールチケット」の交換対象キャラクターは、
MENU＞トレード＞限定交換＞「アビリティツリー実装記念クエスト」
よりご確認ください。

「交換メダル」最大800個GET！

実施期間中、「アビリティツリー実装記念クエスト」の累計クリア数に応じて
「交換メダル」が最大800個入手できます。
また、集めた「交換メダル」は、各クラスの星結晶に交換することができます。

交換可能なアイテムについては、
MENU＞トレード＞限定交換＞「アビリティツリー実装記念クエスト」
よりご確認ください。
