開催期間

6月24日 17:00 ～ 7月13日 23:59

内容

アビリティツリー実装記念のミニイベントを開催いたします。
イベントは、新芽・大種・クレスト・シンボルなどの素材がドロップする
収集クエストと、初登場の【超難関】クエストで構成されています。
【超難関】クエストは【超高難易度】クエストを超える難しさとなっておりますので、
アビリティツリーでキャラクターを育成して挑んでみましょう。

イベントの詳細は、別途掲載のおしらせ
「育てて挑もう！アビリティツリー実装記念クエスト」をご確認ください。
