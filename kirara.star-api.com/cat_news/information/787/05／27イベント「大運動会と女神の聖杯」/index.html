5/27
イベント「大運動会と女神の聖杯」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

5月28日 17:00より、イベント「大運動会と女神の聖杯」を開催いたします。
本イベントは、全ユーザー協力型のイベントとなっております。

開催期間

5月28日 17:00 ～ 6月9日 15:59

イベント概要

本イベントは、
シナリオクエストと討伐クエスト、大討伐クエストで構成されています。
討伐クエストや大討伐クエストでは
イベント素材のほかにイベントポイントを集めることができ、
全ユーザーが獲得したイベントポイントの合計に応じて
召喚チケットやエトワリウムなど様々な報酬を受け取ることができます。

イベントの報酬や遊び方の詳細は、
別途掲載のおしらせ「イベント『大運動会と女神の聖杯』遊び方」を
ご確認ください。

シナリオクエストでオリジナルストーリーを楽しもう！

全14節からなるシナリオクエストを全てクリアすると
指定の日時以降に全2節の追加シナリオを読むことができます。

※「大運動会と女神の聖杯」の各シナリオを読むことで
　星彩石×5個を獲得できます。
※シナリオクエストの14節のみ、初回クリア報酬は星彩石×5個ではなく
　イベント素材「由緒ある杯」×1個となります。

シナリオクエストをクリアして、討伐クエストを開放しよう！

シナリオクエストの開放は3段階に分かれており、4節/9節/14節をクリアすると
それぞれに対応する討伐クエストが開放されます。
また、追加シナリオ2節をクリアすると大討伐クエストが開放され、
よりたくさんのイベント素材やイベントポイントを集めることができます。

討伐クエスト、大討伐クエストでイベント素材を集めよう！

クエストで集めたイベント素材は
イベント限定キャラクターや様々なアイテムと交換できます。
また、★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材と交換が可能です。
専用ぶきを進化させると、強力なスキルが使用できるようになるほか、
全編フルボイスの「メモリアルクエスト」の続きが読めるようになります。

※メモリアルクエストは、
　MENU＞クエスト＞メモリアルクエスト から閲覧できます。

イベントをプレイして称号を取得しよう！

今回のイベントでは、最大2つの称号が取得できます。

取得条件
・シナリオクエストを全てクリアする
・大討伐クエストをクリアする

★4 エンギ【運動会】（せんし　炎属性）がすぐに手に入る！！

購入可能期間
5月28日 17:00 ～ 6月9日 15:59

きららパスポートはイベント期間に合わせて販売される、
お得なアイテムセットです。
きららパスポートを購入することで、
イベント報酬の★4 エンギ【運動会】（せんし　炎属性）を
交換するために必要な量の素材「ごせんぞうサンドバッグ」を
入手することができます。

販売価格
490円

購入場所
・ホーム画面上部＞星彩石購入ウィンドウ
・ホーム画面＞商店街＞星彩石購入

※星彩石（無償）は購入日を含めた14日間、
　毎日初回ログイン時（1日目は購入直後）に入手できます。
　ただし、期間中ログインを行わなかった日は星彩石（無償）を入手できません。
※2日目以降の星彩石（無償）はプレゼントリストへ送られます。
※購入後のきららパスポートの有効期限は星彩石購入ウィンドウの
　「イベント」タブもしくは「購入履歴」から確認できます。
※「大運動会と女神の聖杯」きららパスポートは1度しか購入できません。
※イベント「大運動会と女神の聖杯」終了後も有効期限までの間は
　毎日初回ログイン時に星彩石（無償）が入手できます。
