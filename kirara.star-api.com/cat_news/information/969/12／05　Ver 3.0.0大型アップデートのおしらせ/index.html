大型アップデート その3

ポイント召喚の対象となる召喚では、召喚を実行することでボーナスPtが貯まります。

ボーナスPtは1回召喚で1pt、10回召喚で10pt貯まり、
一定pt貯まると、対象のキャラクター1人を指定して仲間にすることができます。
ポイント召喚の対象キャラクターや必要なpt数は、
召喚開催に際して公開されるおしらせおよび、
召喚画面のポイント召喚のボタンよりご確認ください。

ボーナスPtについて

・ボーナスPtは召喚毎に付与されます。
・ボーナスPtは入手元の召喚のポイント召喚のみで使用でき、
　他の召喚のポイント召喚では使用できません。
・ボーナスPtの消費期限は、入手元の召喚の開催終了日時までとなります。
 ・未消費のボーナスPtがある状態で入手元の召喚が開催終了すると、
　ボーナスPt1ptにつき交換メダル1個に変換され、
　プレゼントリストへ自動で送られます。
・ボーナスPtから交換メダルへの変換は、入手元の召喚の開催終了後に
　召喚の館へ遷移したタイミングで実行されます。
・消費期限が切れたボーナスPtの変換は、複数の召喚において一括で行われます。

大型アップデート その4

新アイテム「リトライキーホルダー」は、召喚結果が出た後に使用することで、
星彩石や召喚チケットを消費せずに召喚をやり直せるアイテムです。
召喚をやり直すには、
1回召喚で1個、10回召喚で10個の「リトライキーホルダー」が必要です。

「チャンスアップキーホルダー」を使用して召喚を行った後、
さらに「リトライキーホルダー」を使用すると、
★4・★5キャラクターの出現確率アップ効果を維持した状態で
召喚をやり直すことができます。
2種類のキーホルダーを組み合わせて使ってみましょう！

「リトライキーホルダー」の使用について

・一部の召喚では「リトライキーホルダー」を使用することができません。
・召喚の途中で「リトライキーホルダー」を購入することはできません。
・召喚時に入手できる「星屑のかけら」やボーナスPtは、
　召喚結果を確定した際に付与されます。
　また、「10回召喚」を行うごとにステップアップする召喚で
　「リトライキーホルダー」を使用した場合、
　召喚結果を確定した際にステップ数が加算されます。
　どちらも召喚をやり直す過程では付与や加算が行われませんのでご注意ください。
・召喚結果が確定する前に何らかの影響でタイトル画面に戻ってしまった場合は
　召喚をやり直すことはできません。
　メンテナンス直前や不安定な通信環境下では特にご注意ください。
　なお、召喚のやり直しが不可能な状態となってしまった場合には、
　直前の召喚内容で結果が確定されます。
