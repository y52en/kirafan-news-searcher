12月10日 17:00

自動発動スキル強化対象の専用ぶきを持つキャラクター

今回自動発動スキル内容が強化されるのは、以下キャラクターの専用ぶきです。

★5 勇【お正月】（アルケミスト　炎属性）

■自動発動スキル
　更新前
　「自身」の通常攻撃が状態異常に特効を持つ +
　「自身」の通常攻撃を受けた敵の炎属性耐性を
　一定ターン小ダウン

　更新後
　「自身」の通常攻撃が状態異常に大きな特効を持つ +
　「自身」の通常攻撃・クラススキルを受けた敵の
　炎属性耐性を一定ターン中ダウン

★5 紅葉【バレンタイン】（まほうつかい　水属性）

■自動発動スキル
　更新前
　通常攻撃が全体攻撃になる +
　通常攻撃・クラススキルを使用するたび
　「自身」のリキャストをかなり減らす

　更新後
　通常攻撃が全体攻撃になる +
　通常攻撃・クラススキルを使用するたび
　「味方全体」のリキャストをかなり減らす

★5 万年 大会（ナイト　水属性）

■自動発動スキル
　更新前
　スタン状態にならない +
　「自身」が攻撃を受けるたび
　「敵全体」の魔法攻撃を一定ターン中ダウン

　更新後
　スタン状態にならない +
　「自身」の魔法防御が1.5倍になる +
　「自身」が攻撃を受けるたび
　「敵全体」の魔法攻撃を一定ターン中ダウン

★5 【荒野の旅人】リン（せんし　炎属性）

■自動発動スキル
　更新前
　通常攻撃のダメージが小アップ +
　通常攻撃・クラススキルを使用するたび
　「自身」のリキャストをかなり減らす

　更新後
　通常攻撃・クラススキルのダメージが小アップ +
　通常攻撃・クラススキルを使用するたび
　「自身」のリキャストをかなり減らす
