イベント「ブレンドコールアクシデント」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「コウモリの羽」や「スライムの液体」が入手できます。
イベントで手に入る素材は★5キャラクター専用ぶきの進化に必要な
エトワリウムなどと交換できます。

交換可能なアイテムについては、
MENU＞トレード＞限定交換＞「ブレンドコールアクシデント」よりご確認ください。

限定交換期間
5月17日 17:00 ～ 6月7日 13:59

「イベント復刻コールチケット」をGETしてキャラクターを覚醒させよう！

「イベント復刻コールチケット」は、イベント素材「スライムの液体」×100個で
1度だけ交換が可能です。
また、イベントミッションの報酬と合わせて
最大9枚の「イベント復刻コールチケット」が手に入ります。

「イベント復刻コールチケット」1枚で、
過去のイベント報酬で登場した★4キャラクターと交換できます。
既に入手している同一キャラクターとの交換も可能で、
最大覚醒を目指すことも可能です。
最大覚醒後に、同一キャラクターを入手した場合は、
輝星のかけらを入手できます。

※「イベント復刻コールチケット」の対象のキャラクター詳細は
　MENU＞トレード＞限定交換＞「ブレンドコールアクシデント」
　よりご確認ください。

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

「メニュー表」は、★5キャラクター用の限界突破素材と交換できます。
今回のイベントでは、10作品のコンテンツ限定限界突破素材と交換可能です。

豪華素材がドロップする【超強敵】クエスト開催！

開放期間
5月19日 17:00 ～ 5月31日 15:59

【超強敵】クエストは、シナリオクエストを全てクリアしていると
指定の日時以降に開放されます。
また、挑戦するためには専用の挑戦券が必要です。
専用の挑戦券は、収集クエスト（超級）やイベントミッションの報酬で手に入ります。

※使用しない専用の挑戦券は、
　MENU＞トレード＞限定交換＞「ブレンドコールアクシデント」
　にて「スライムの液体」に交換も可能です。
