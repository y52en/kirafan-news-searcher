内容

出現する★5＆★4キャラクターが
2018年に登場した水着キャラクターのみの召喚です。
さらに、「10回召喚」を行うごとにステップアップする召喚となっており、
各ステップで以下のおまけを入手できます。
また、7月29日 ～ 8月1日の4日間は1日1回のみ無料で「10回召喚」が行えます。

1～5ステップ目：交換メダル200枚
   6ステップ目：エトワリウムのかけら1個
   7ステップ目：2018水着復刻★5確定召喚チケット1枚

「2018水着復刻★5確定召喚チケット」を入手すると
召喚画面に「2018水着復刻★5キャラクター確定召喚」が出現し、
対象の★5キャラクター1人をランダムで仲間にすることができます。
対象のキャラクターは下記をご確認ください。

2018水着復刻★5確定召喚チケット対象キャラクター
