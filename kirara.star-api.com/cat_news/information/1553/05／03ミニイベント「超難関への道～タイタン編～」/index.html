ミニイベント「超難関への道～タイタン編～」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

5月5日より
ミニイベント「超難関への道～タイタン編～」を開催いたします。

開催期間

5月5日 0:00 〜 5月17日 15:59

イベント概要

本イベントは、収集クエストととっておき育成クエスト、
【超難関】クエストで構成されています。
アビリティツリーや育成クエストでキャラクターを強化して、
【超難関】クエストに挑戦してみましょう！

アビリティツリーに関する詳細は、
別途掲載のおしらせ「アビリティツリーの遊び方」をご確認ください。

育成が捗る収集クエストが登場！

開放期間
5月5日 0:00 〜 5月17日 15:59

収集クエストではキャラクターの強化に必要な新芽・大種、
ぶきの強化に必要なクレスト・シンボルがドロップします。
この機会にたくさんの強化素材を集めましょう！

キャラクターのとっておきレベルを上げよう！

開放期間
5月5日 0:00 ～ 5月17日 15:59

とっておき育成クエストは、曜日クエストの修練場よりも簡単に
キャラクターのとっておきレベルを上げることができるクエストです。

とっておき育成クエストに出撃するために必要な「とっておき育成場入場券」は、
イベントミッションのクリア報酬で最大40枚、
トレードショップの通常交換で最大40枚入手できます。

■とっておき育成場入場券
　交換に必要なアイテム：星彩石 × 20
　交換期間：5月5日 0:00 〜 5月17日 15:59
　※40回まで交換できます。

大激闘！【超難関】クエスト！

開放期間
5月8日 0:00 〜 5月17日 15:59

【超高難易度】クエストを超える難しさの【超難関】クエストが登場します。
また、本クエストをクリアすることで
強敵として登場するタイタンの称号を取得できます！

イベントミッション実施！

実施期間
5月5日 0:00 ～ 5月17日 23:59

限定交換期間
5月5日 0:00 〜 5月24日 13:59

「イベント復刻コールチケット」最大9枚GET！

実施期間のうち5月5日 0:00 〜 5月13日 23:59は、
1日につきクエストを5回クリアすることで
過去のイベント報酬で登場した★4キャラクターと交換できる
「イベント復刻コールチケット」が1枚入手できます。

「イベント復刻コールチケット」の交換対象キャラクターは、
MENU＞トレード＞限定交換＞「超難関への道～タイタン編～」よりご確認ください。

※「イベント復刻コールチケット」が報酬のミッション「クエストを5回クリア」は
　毎日リセットされます。報酬は当日中にお受け取りください。

「交換メダル」最大800個GET！

実施期間のうち5月14日 0:00 〜 5月17日 23:59は、
1日につきクエストを5回クリアすることで「交換メダル」が200個入手でき、
集めた「交換メダル」は、トレードショップの限定交換にて
各クラスの星結晶などのアイテムに交換することができます。

交換可能なアイテムについては、
MENU＞トレード＞限定交換＞「超難関への道～タイタン編～」よりご確認ください。

※「交換メダル」が報酬のミッション「クエストを5回クリア」は
　毎日リセットされます。報酬は当日中にお受け取りください。

「とっておき育成場入場券」最大40枚GET！

実施期間のうち5月5日 0:00 〜 5月14日 23:59は、
1日につきクエストを10回クリアすることで「とっておき育成場入場券」が4枚
入手できます。

※「とっておき育成場入場券」が報酬のミッション「クエストを10回クリア」は
　毎日リセットされます。報酬は挑戦期限までにお受け取りください。

さらに、実施期間中、「超難関への道～タイタン編～」のクエストの
累計クリア回数に応じて「スタミナ回復アイテム　大」と「クエストキー　金」が
それぞれ最大15個入手できます。
