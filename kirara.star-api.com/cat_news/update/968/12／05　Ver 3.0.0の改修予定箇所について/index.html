　Ver 3.0.0の改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

12月10日実施予定のVer 3.0.0アップデートにおいて
実装予定の機能改善内容をご案内いたします。

機能アップデート

・メインクエスト第2部の追加
　メインクエスト第2部「断ち切られし絆」を追加します。

・クエスト選択画面TOPのUIを一新
　メインクエスト第2部の追加に伴い、
　クエスト選択画面TOPのデザインをリニューアルします。

・「ポイント召喚」機能の追加
　召喚毎にボーナスPtが獲得でき、
　ボーナスPtを貯めることで特定のキャラクターを仲間にできるようになります。

・「リトライキーホルダー」の追加
　召喚をやり直せるアイテムが登場します。

　※詳細は、別途掲載のおしらせ「Ver 3.0.0大型アップデートのおしらせ」を
　　ご確認ください。

・タイトル画面の変更
・アプリアイコンの変更

不具合対応

・軽微な不具合修正
