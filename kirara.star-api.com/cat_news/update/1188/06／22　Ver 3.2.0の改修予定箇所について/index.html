　Ver 3.2.0の改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

6月24日実施予定のVer 3.2.0アップデートにおいて
実装予定の機能改善内容をご案内いたします。

機能アップデート

・キャラクター強化機能「アビリティツリー」の実装
　任意の★4・★5キャラクターに対してステータスの強化や
　自動発動スキルの付与ができるようになります。

　※詳細は、別途掲載のおしらせ「アビリティツリーの遊び方」をご確認ください。

・「試練クエスト」、「探掘クエスト」の追加
　「アビリティツリー」の解放や強化に使用するアイテムが入手可能な
　新クエスト「試練クエスト」、「探掘クエスト」に挑戦できるようになります。

・アプリアイコンの変更

機能改修

・メインクエスト第2部におけるクエストのドロップアイテム調整
　メインクエスト第2部1章・2章の一部クエストのドロップアイテムを調整

・ウィークリーミッションの内容を追加、調整
　※6月28日始まりの週より適用

・その他の細かな機能改修
