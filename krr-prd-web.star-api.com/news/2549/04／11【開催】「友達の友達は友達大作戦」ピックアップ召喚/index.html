【開催】「友達の友達は友達大作戦」ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

4月11日 メンテナンス後より、「友達の友達は友達大作戦」ピックアップ召喚を開催いたします。

「スロウスタート」より★5 一之瀬 花名（CV.近藤玲奈）、★4 千石 冠（CV.長縄まりあ）、★4 十倉 栄依子（CV.嶺内ともみ）、★4 百地 たまて（CV.伊藤彩沙）が『きららファンタジア』に登場！

 

更に、「Aチャンネル」より★5 るん（CV.福原香織）、「NEW GAME!」★4 阿波根 うみこ（CV. 森永千才）も出現率がアップしています。

全体開催期間：4月11日 メンテナンス後 ～ 4月25日 11:59

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

1日1回有償限定召喚

「友達の友達は友達大作戦」ピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

イベントボーナス効果持ちのキャラクター登場！

今回のピックアップ対象キャラクターは、4月11日 メンテナンス後から開催されるイベント「友達の友達は友達大作戦」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「友達の友達は友達大作戦」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「友達の友達は友達大作戦」のおしらせをご確認下さい。

 


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

※今回のピックアップ対象の★3キャラクターはイベントボーナスの効果を持ちません。ご注意ください。
