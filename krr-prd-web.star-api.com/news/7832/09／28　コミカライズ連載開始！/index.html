『COMIC FUZ』とは

『COMIC FUZ』は、芳文社が発行しているマンガ雑誌や
連載されている人気マンガを楽しめるアプリです。
基本無料で、読める作品は約1,700話以上！

『COMIC FUZ』内で30分に１回配布されるメダルを使用することで、
毎日更新される連載コーナーなどにて、好きなマンガを無料で読むことができます。

本作品も無料でお楽しみいただけますので、ぜひお読みください。
