「きんいろダークネス」あらすじ

全25節からなるシナリオクエストは、

従来の形のシナリオに加え、原悠衣先生による描き下ろしマンガも楽しめる

新しい形式のクエストとなっております。

また、シナリオクエストを全てクリアすると
特別なルームアイテムを入手することができます。

 

マンガが読める新機能登場！

Ver1.19.0アップデートにおいて
シナリオクエストでマンガが読める新機能が追加されます。

 

一度読んだマンガは、MENU＞図書館＞物語ふりかえりで読み返すことができます。

 

シナリオで活躍するキャラクターが召喚に登場！

シナリオクエストで活躍する
★5 アリス・カータレット、★5 松原 穂乃花が召喚に登場します。

詳しくは別途掲載のおしらせ
「きんいろダークネス ピックアップ召喚開催！」をご確認ください。

 

収集クエストでアイテムを集めよう！

シナリオクエストを全てクリアした後は、
ぶき素材の収集がしやすい収集クエストや、
高難易度の【超強敵】クエスト、【極】クエストにも挑戦することができます。

クエストの詳細は、別途掲載のおしらせ
作家クエスト「きんいろダークネス」遊び方をご確認ください。
