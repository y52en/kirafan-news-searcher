クロスシナリオとは？

クロスシナリオとは作品の垣根を越えたキャラクター同士の物語です。
クロスシナリオが存在している2キャラクターそれぞれの
なかよし度が4以上になると開放されます。
条件達成後に【図書館＞物語ふりかえり＞キャラシナリオ＞各作品＞
対象キャラクター＞クロス】にてご確認ください。
