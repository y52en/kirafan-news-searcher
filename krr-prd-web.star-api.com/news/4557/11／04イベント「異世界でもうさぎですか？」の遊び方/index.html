イベントボーナス効果について

今回のイベントでは、一部のキャラクターがイベントボーナス効果を持っています。
イベントボーナス効果を持ったキャラクターをパーティーに編成することで、
イベント「異世界でもうさぎですか？」のクエストにおいて
イベント素材のドロップ数がアップします。

▼エスプレッソ＋5　ティッピーパン＋3

★5ココア（せんし　風属性）
★5チノ（まほうつかい　陽属性）


▼エスプレッソ＋3　ティッピーパン＋2

★4ココア（せんし　炎属性）
★4チノ（まほうつかい　水属性）


※イベントボーナスの効果は、サブメンバー・サポート枠に
　編成されたキャラクターからも受けることができます。

豪華素材がドロップする「【乱戦】クエスト」！

今回のイベントでは、クリアすることで
豪華報酬が手に入る「【乱戦】クエスト」が発生します。
【乱戦】クエストは、イベント開始後の2018年11月13日 17:00から開催されます。

【乱戦】クエストに挑戦するためには、
【乱戦】かつてない盛況挑戦券を消費する必要があります。
【乱戦】かつてない盛況挑戦券は「【超級】新メニュー考案の旅」、
「【超級】オリジナルブレンドを求めて」で入手することができます。

乱戦クエストをクリアすると、イベント共通の素材と、
まれに★4限界突破素材を入手することができます。
その他、クエスト初クリア時や、一人も撤退せずにクリアした時などに
特別な報酬を入手することができます。

※使用しない【乱戦】かつてない盛況挑戦券は、
　トレードショップ＞限定交換＞「異世界でもうさぎですか？」から
　「ティッピーパン」に交換することができます。
