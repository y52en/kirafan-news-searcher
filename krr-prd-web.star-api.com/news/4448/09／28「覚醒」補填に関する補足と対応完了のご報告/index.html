「覚醒」補填に関する補足と対応完了のご報告
『きららファンタジア』をご利用いただき、誠にありがとうございます。

9月27日(木)12:00からのメンテナンス時より開始した
新機能「覚醒」の実装に伴う補填対応が完了いたしましたのでご報告いたします。

また、補填内容につきまして補填作業及び補填事例を補足説明いたします。


補填内容

今までお客様が重複して入手された同一キャラクター（進化前後含む）について、
チュートリアル召喚及びトレードショップで
入手されたキャラクターを除くすべての召喚で入手されたキャラクターを対象に
過去にさかのぼって「覚醒」機能実装後に重複入手されたものと同じ状態に
補填いたしました。


補填対応例

パターン①
　・チュートリアル召喚にてキャラクターA（★5）を2キャラクター入手
　・任意の星彩石召喚にてキャラクターA（★5）を3キャラクター入手

　・パターン①の補填内容
　　・キャラクターA（★5）を覚醒レベル2に変更
　　　内訳：チュートリアル召喚は補填対象ではありません（カウントされません）。
　　　　　　星彩石召喚にて入手したキャラクターは補填対象となります。
　　　　　　3キャラクターが補填対象となるため、
　　　　　　重複している2キャラクター分が補填されます。

・パターン②
　・任意の星彩石召喚にてキャラクターB（★5）を1キャラクター入手
　・スペシャルコールチケットの召喚にてキャラクターB（★5）を2キャラクター入手
　・★5確定召喚チケットの召喚にてキャラクターB（★5）を1キャラクター入手
　
　・パターン②の補填内容
　　・キャラクターB（★5）を覚醒レベル3に変更
　　　内訳：星彩石召喚及び召喚チケットによるキャラクター召喚にて
　　　　　　入手したキャラクターは補填対象となります。
　　　　　　4キャラクターが補填対象となるため、
　　　　　　重複している3キャラクター分が補填されます。

・パターン③
　・任意の星彩石召喚にてキャラクターC（★4）を5キャラクター入手
　・★4確定召喚チケットの召喚にてキャラクターC（★4）を1キャラクター入手
　
　・パターン③の補填内容
　　・キャラクターC（★4）を覚醒レベル2に変更
　　　内訳：星彩石召喚及び召喚チケットによるキャラクター召喚にて
　　　　　　入手したキャラクターは補填対象となります。
　　　　　　6キャラクターが補填対象となるため、
　　　　　　重複している5キャラクター分が補填されます。
　　　　　　また、★4レアリティでは5キャラクター分が重複している場合、
　　　　　　覚醒レベル2まで上昇します。

※チュートリアル召喚及びトレードショップ以外の
　すべての召喚で入手したキャラクターが補填対象になるため、
　スペシャルコールチケット/★5確定召喚チケット/★4確定召喚チケットなどで
　召喚したキャラクターも補填対象です。
※補填対応時の例であり、覚醒機能実装後にチュートリアル召喚や
　トレードショップにて同一キャラクターを
　重複して入手した場合は覚醒が発生します。
※レアリティによって覚醒レベルが上昇するキャラクター数が異なります。
　詳細はヘルプ内 成長 ＞ 覚醒 ページをご参照下さい。
