　メインクエスト ハードモード7章追加！

『きららファンタジア』をご利用いただき、誠にありがとうございます。

8月27日 17:00より、メインクエストにハードモード7章を追加実装いたします。

 

追加日時

8月27日 17:00より

 

ハードモード7章は、メインクエスト7章（ノーマル） 最終節をクリアすると

開放されます。

※ハードモードのシナリオ内容は、ノーマルモードと同じ内容となります。

 

また、このハードモードでは特定条件をクリアすると、

各章で最大220個の星彩石を手に入れることができます。

 

■ハードモード各クエストをクリア

　→　クエストクリア報酬：星彩石×5個

■ハードモード各章内の全クエストをクリア（32節までクリア）

　→　オールクリア報酬：星彩石×20個

■ハードモード各章内の全クエストをクリアランクコンプリート達成
　（コンプリートはパーティーメンバーが一人も戦闘不能にならずに

　クリアすることで獲得）

　→　オールコンプリートクリア報酬：星彩石×40個

詳細はMENU＞クエスト＞メインクエストにてご確認ください。
※メインクエスト画面左下の章選択ボタンより7章を選択後、

　画面左上の「ハード」ボタンをタップすることで移動できます。

 

注意事項

・仕様や日程は、予告なく変更する可能性がございます。
