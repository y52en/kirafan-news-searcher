Ver1.16.0での改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.16.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・MENUの改修
　※表示ボタンの調整（「フレンド」ボタンを「設定・その他」へ移動）


・トレーニング画面の改修
　※トレーニング内容の更新
　※「おまかせ編成」の抽出条件改修
　※トレーニング選択画面、ポップアップ文言等改修
　※完了済みのトレーニングが複数あった場合、一括で完了できる機能を実装
　※トレーニング完了時画面で、同じトレーニングに再挑戦できる機能を実装


・ルーム倉庫拡張機能の改善
　※倉庫上限に達していた場合、ルームアイテムを倉庫に送った際に
　　「倉庫拡張」を行えるよう改修

 

・ルームアイテム購入・設置機能の改善
　※ルームアイテム購入時に同アイテムを複数個購入、連続設置できる機能を実装
　　（倉庫から設置をする際も、同機能の利用が可能です）
　　

・ステータス画面の改修
　※各キャラクタースキル、ぶき情報表示用のタブを追加
　※その他各情報タブ内のレイアウト調整

 

・アプリアイコン変更


・細かな機能改修
　・サポート選択画面の改修
　・図書館での未読表示の追加
　・プレゼントリスト内での
　　アイテム受取時に表示される所持上限文言の調整
　・その他、機能改修や今後実装予定のスキル効果実装など
　・一部キャラクター専用ぶきのスキル説明文言修正
　　※スキル効果内容には変更ありません。

 

不具合対応

　・軽微な不具合の修正
