Ver1.14.0での改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.14.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・ダウンロード容量表示の改修　※iOS版のみ
　・DL画面の改修
　・シナリオ読み込み時のDLサイズ表示


・アプリアイコン変更

・細かな機能改修
　・里のキャラクターアイコンをタップしたとき、
　　スタミナが上限値を超えて回復できるよう変更
　・ルームアイテムに「売却不可」の設定を追加
　・スタミナ回復アイテム使用時に、複数個使用できるよう変更
　・里画面で「トレーニング所」「きらら商店街」「クエストゲート」
　　「召喚の館」「マイルーム」をタップした際の確認表示を行うよう変更
　・その他、機能改修や今後実装予定のスキル効果実装など

 

不具合対応

　・軽微な不具合の修正
