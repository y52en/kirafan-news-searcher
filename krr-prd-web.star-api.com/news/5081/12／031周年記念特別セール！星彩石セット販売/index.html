1周年記念スペシャルコールチケット召喚とは

★5キャラクター1人を指定して仲間にすることができるチケットです。

 

指定することのできる★5キャラクターについては、
ゲーム内の召喚画面＞スペシャルコールチケット召喚にて確認できます。
必ずご確認のうえ、スペシャルコールチケットをご使用下さい。

既に加入済みのキャラクターが存在する場合に、その加入済みのキャラクターを
「1周年記念スペシャルコールチケット召喚」で仲間にすると、
そのキャラクターのクラスの限界突破素材の入手に加えて、
最大覚醒していない場合はキャラクターが「覚醒」し、
最大覚醒している場合は「輝星のかけら　金」を入手できます。
