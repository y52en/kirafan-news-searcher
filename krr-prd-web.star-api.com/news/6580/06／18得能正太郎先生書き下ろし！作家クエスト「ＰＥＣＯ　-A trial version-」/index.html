作家クエスト新登場！

作家クエストは、各作品の原作者が物語の舞台や設定、
登場するキャラクターのデザインなどをセットで考案し、リリースされる
これまでにないクエストです。

また、シナリオクエストのバトル難易度は低く設定されており、
ゲームを始めたばかりの方でも挑戦することができます。

シナリオクエストを全てクリアした後は、
ぶき素材の収集がしやすい収集クエストや、
高難易度の【超強敵】クエスト、【極】クエストにも挑戦することができます。

クエストの詳細は、別途掲載のおしらせ
作家クエスト「ＰＥＣＯ　-A trial version-」遊び方をご確認ください。
