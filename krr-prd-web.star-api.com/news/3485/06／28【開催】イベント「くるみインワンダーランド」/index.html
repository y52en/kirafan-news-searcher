『きららファンタジア』をご利用いただき、誠にありがとうございます。

開催期間

6月28日 17:00 ～ 7月9日 13:59

 

 

イベント内容

 
本イベントは、シナリオクエストと素材収集クエストで構成されています。

マップ内の「エリア」で発生するシナリオクエストクエストをクリアすることで、イベントを進めていくことができます。

シナリオクエストを最後までクリアすることで、外伝クエストが開放可能となります。

※外伝クエストは時限開放となっております。開放日時前にシナリオクエストを最後までクリアしていても、外伝クエストは開放されません。

 

「外伝クエスト？？？」をクリア時に入手できるアイテムは、トレードショップにて「せんし専用イベント限定ぶき」と交換できます。

GETした素材をトレードショップで交換しよう！

今回は、「★5キャラクター専用ぶき」を進化させるための新アイテム「エトワリウム」も収集アイテムと交換可能です。

「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。

※「エトワリウム」はイベント期間中、最大3つまで交換できます。

※「十字のチョーカー」はイベント期間中、最大5つまで交換、入手することができます。

 

「コンテンツ限定限界突破素材」は、対応する作品の★5キャラクターならクラスを問わずに使用することのできるアイテムです！

イベントボーナス効果について

今回のイベントでは、イベントボーナス効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターをパーティーに編成することで、イベント「くるみインワンダーランド」のクエストにおいて、イベント素材のドロップ数がアップします。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

豪華素材がドロップする「【乱戦】クエスト」も！

今回のイベントでは、強いモンスターが登場する「【乱戦】クエスト」が発生します。「【乱戦】クエスト」に参加するためにはシナリオクエストを全てクリアしている必要があります。

【乱戦】クエストに挑戦するためには、【乱戦】激闘！黒黒団挑戦券が必要となります。

※【乱戦】激闘！黒黒団挑戦券は、トレードショップ＞限定交換＞「くるみインワンダーランド」から、イベント限定アイテム「サイコロ」と交換することができます。

 

【乱戦】クエスト発生期間

7月2日 17:00 ～ 7月9日 13:59

 

クリア報酬

・【乱戦】クエストを初めてクリアした時：星彩石×40個
・【乱戦】クエストにおいて、一人も撤退しないで初めてクリアした時：まほうつかいの果実×2個
