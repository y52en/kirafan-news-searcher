【開催】イベント「復刻・友達の友達は友達大作戦」

『きららファンタジア』をご利用いただき、誠にありがとうございます。


3月13日 17:00より、「復刻・友達の友達は友達大作戦」を開催いたします。
本イベントは2018年4月に開催したイベントの復刻開催となります。


なお、本イベントと連動した特別なWEBキャンペーン
「友達の友達は友達大作戦 みんなで応援キャンペーン」も
実施いたします。

 

キャンペーンにつきましては別途掲載のおしらせをご確認ください。

 

開催期間

3月13日 17:00 ～ 3月27日 13:59


スロウスタートイベント復刻！


本イベントは、シナリオクエスト・復刻記念追加シナリオと収集クエスト、
【乱戦】クエストで構成されています。
※本イベントのシナリオクエストは過去に開催した
　「友達の友達は友達大作戦」と同じ内容ですが、
　交換可能な報酬は一新されています。

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。


※過去に開催した「友達の友達は友達大作戦」で入手した
　名称/デザインが同一のイベント素材は使用できませんのでご注意ください。

より多くのイベント素材や限界突破素材が入手できる【乱戦】クエストは、
「【乱戦】友達大作戦挑戦券 」を消費することで挑戦できます。
 
イベントのあらすじや遊び方は、イベント画面＞イベント詳細をご確認ください。

追加シナリオにはあのキャラクターが登場！


復刻イベント開催を記念して、新規シナリオを3月20日から毎日追加！
以前イベントを遊んだことがある人も、追加シナリオを楽しもう！

1節追加：3月20日 0:00 ～
2節追加：3月21日 0:00 ～
3節追加：3月22日 0:00 ～
4節追加：3月23日 0:00 ～

※復刻記念追加シナリオはシナリオクエストを全てクリアすると開放されます。
