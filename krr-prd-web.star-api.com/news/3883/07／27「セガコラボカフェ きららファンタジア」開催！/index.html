「セガコラボカフェ きららファンタジア」開催！

7月28日（土）よりセガコラボカフェ秋葉原 2号館（セガ秋葉原2号館6F）にて

『きららファンタジア』とコラボレーションした「セガコラボカフェ きららファンタジア」がオープン！


期間中は『きららファンタジア』のキャラクターや世界観をイメージしたオリジナルのフード、デザート、ドリンクや、セガコラボカフェ限定のオリジナルグッズを販売中！ ぜひお越し下さい！
