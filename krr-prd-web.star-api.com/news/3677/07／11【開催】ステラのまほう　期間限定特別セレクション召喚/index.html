【開催】ステラのまほう　期間限定特別セレクション召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

7月11日 15:00より、「ステラのまほう」のキャラクターがピックアップされた「ステラのまほう　期間限定特別セレクション召喚」を開催いたします。


今回の期間限定特別セレクション召喚では、過去に期間限定として出現したキャラクターもピックアップ対象になっておりますので、ぜひご利用下さい。

 

開催期間

7月11日 15:00 ～ 7月23日 11:59 9:59

※7月23日 10:00～のメンテナンス実施に伴い、終了時刻を7月23日 9:59に変更いたします。 ご迷惑をおかけいたしますが、何卒よろしくお願い申し上げます。（7月23日 19:50追記）

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

1日1回有償限定召喚

「ステラのまほう　期間限定特別セレクション召喚」開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。
