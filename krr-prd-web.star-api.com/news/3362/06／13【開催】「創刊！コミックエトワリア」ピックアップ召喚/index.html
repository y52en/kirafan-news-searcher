【開催】「創刊！コミックエトワリア」ピックアップ召喚

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

6月13日 17:00より、「創刊！コミックエトワリア」ピックアップ召喚を開催いたします。

 

召喚前半期間には、「こみっくがーるず」より★5 萌田 薫子（CV.赤尾ひかる）、★4 恋塚 小夢（CV.本渡楓）、★4 色川 琉姫（CV.大西沙織）、★4 勝木 翼（CV.高橋李依）が『きららファンタジア』に初登場し、出現率がアップ！

 

また召喚後半期間には、「うらら迷路帖」より★5 雪見 小梅（CV.久保ユリカ）が登場！さらに「うらら迷路帖」より★4 二条 臣（CV.市ノ瀬加那）、「ブレンド・S」より★4＆★3 天野 美雨（CV.種﨑敦美）が『きららファンタジア』に初登場し、出現率がアップ！

 

 

開催期間

全体開催期間：6月13日 17:00 ～ 6月28日 11:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意ください。

 

・ピックアップ召喚前半：6月13日 17:00 ～ 6月28日 11:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能になります

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「創刊！コミックエトワリア」ピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターはイベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、6月13日 17:00から開催されるイベント「創刊！コミックエトワリア」 にてイベントボーナスの効果を持っています。

 イベントボーナスを持ったキャラクターは、イベント 「創刊！コミックエトワリア」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「創刊！コミックエトワリア」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

※★4 萌田 薫子（そうりょ　風属性）は、イベント「創刊！コミックエトワリア」の期間中のみ仲間にできるキャラクターです。イベント中に「羽根ペン」を集めることで、交換可能になります。

※★3　天野 美雨（まほうつかい　土属性）は、イベントボーナス効果はありません。
