「水も滴る渚のきららキャンペーン」期間中、通常のログインボーナスに加え、
「夏の育成応援ログインボーナス」を開催いたします。
開催期間中、毎日ログインすることで
合計700個の交換メダルを入手することができます。
入手した「交換メダル」は後述のお得な進化セットと交換して、
キャラクターの育成に役立てましょう！

■ログインボーナス内容
1日目：交換メダル×400
2日目～7日目：交換メダル×50

期間：7月18日 0:00 ～ 7月24日 23:59

期間中、トレードショップの【通常交換】にお得な進化セットを追加いたします。
クラス用進化セットと属性用進化セットを1つずつ交換することで、
★5キャラ、あるいは★4キャラをひとり
進化させることができる量の素材を入手できます。
交換に必要な「交換メダル」は
「夏の育成応援ログインボーナス」で期間中毎日プレゼント！
毎日ログインして、パーティーを一気に強化しましょう！

▼お得なクラス進化セット 例 ★5あるいは★4キャラクターの進化に必要な量の
各クラスのスタチューと限界突破素材が詰まったセットです。
交換に必要なアイテム：交換メダル×300
限度数：1　

▼お得な属性進化セット 例 ★5あるいは★4キャラクターの進化に必要な
各属性の進化珠と強化素材が詰まったセットです。
交換に必要なアイテム：交換メダル×100
限度数：1　

交換可能期間：7月18日 0:00 ～ 7月24日 23:59
※クラス用進化セットは
　全5種（せんし・まほうつかい・そうりょ・ナイト・アルケミスト）が登場します
※属性用進化セットは、全6種（炎・風・土・水・月・陽）が登場します
※期間中、各セットにつき1回限り交換可能です
