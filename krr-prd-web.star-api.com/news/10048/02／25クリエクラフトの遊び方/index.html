クリエクラフトの遊び方

①クリエを貯める
コンテンツルームが開放されている作品のキャラクターを編成して
クエストをクリアすると、キャラクターごとに獲得できる経験値やなかよし度のほかに
作品ごとにクリエを獲得できます。
なお、★5キャラクターや覚醒しているキャラクターをパーティーに編成していると、
クリエの獲得量が多くなります。

②キャラクターから依頼を受注する
作品ごとにクリエが一定量貯まることで、
キャラクターからの依頼を受注できるようになります。
各作品のコンテンツルーム＞クリエクラフト＞受注可能な依頼＞受注する より
依頼を受注しましょう。

③クリエクラフト開始
依頼を受注するとクリエクラフトが開始されます。

④依頼をこなす
依頼の達成条件となっているクラフトクエストに挑戦しましょう。
該当のクラフトクエストは、
各作品のコンテンツルーム＞クリエクラフト＞進行中の依頼＞クラフトクエストへ
もしくは、
MENU＞クエスト＞クラフトクエスト＞該当作品
より挑戦できます。

⑤依頼の達成報告をする

クラフトクエストをクリアしたら、
各作品のコンテンツルーム＞クリエクラフト＞報告可能な依頼＞報告する より
達成報告をして依頼を完了させましょう。

⑥クリエクラフト完了

達成報告をすると、達成報酬となっているアイテムを入手した後、
カンナが訪れてコンテンツルームを改装します。
達成報酬では、交換アイテム「クラフトメダル」や、
その作品のきらら装備「クリエオーブ」などを入手できます。
また、クリエクラフトで開放したルームアイテムは
ルームショップに追加され、マイルームに設置できるようになります。

⑦次の依頼に向けてクリエを貯める

依頼を達成したら、次の依頼に向けて再びクリエを貯めましょう。
すべての依頼を完了させるまで、①～⑥を繰り返します。

⑧ボーナスクラフトが発生

すべての依頼を達成した後、クリエを貯めることで
ボーナスクラフトが発生します。

⑨ボーナスクラフトでクラフトメダルを入手

ボーナスクラフトでは、交換アイテム「クラフトメダル」が入手できます。
なお、ボーナスクラフトは、クリエが一定量貯まるごとに繰り返し発生します。

⑩クラフトメダルを交換

クリエクラフトやボーナスクラフトで集めた「クラフトメダル」は
トレードショップの通常交換にて
各クラスの星結晶やエトワリウムのかけらなどと交換できます。

作品ごとのクリエクラフトの進行状況は、
MENU＞作品リスト よりいつでも確認できます。
