04/28
ジンジャー参戦記念ミニイベント「黄金の記憶」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

七賢者・ジンジャーの参戦を記念して、
4月29日 0:00より、七賢者・ジンジャー参戦記念ミニイベント「黄金の記憶」を
開催いたします。

 

開催期間

4月29日 0:00 ～ 5月10日 23:59

 

内容

期間中、ジンジャーや「きんいろモザイク」のキャラクターが登場する
特別シナリオ「黄金の記憶」を読むことができます。

「黄金の記憶」は全4節のシナリオで構成されています。

 

【開放スケジュール】

・特別シナリオ1節

     4月29日0:00 ～ 5月10日23:59

・特別シナリオ2節　

     4月30日0:00 ～ 5月10日23:59

・特別シナリオ3節

     5月1日0:00 ～ 5月10日23:59

・特別シナリオ4節

     5月2日0:00 ～ 5月10日23:59

 

※「黄金の記憶」の各シナリオを読むことで、星彩石×5個を獲得できます。

 

 七賢者チャレンジクエスト開催！

「七賢者チャレンジクエスト」では、強敵として
七賢者・ジンジャーが登場します！

 

チャレンジクエストは、
「ホーム＞黄金の記憶＞七賢者チャレンジクエスト」から挑戦可能です。
挑戦する際には、クエストキー　銀×3個を消費する必要があります。

 

アイテム「チャレンジメダル」を集めよう！

七賢者チャレンジクエストをクリアすると、報酬を入手できます。

初めてクリアした時：チャレンジメダル×1個

一人も撤退しないでクリアした時：チャレンジメダル×1個

 

※クリア報酬のほかにミッション達成でもチャレンジメダルを入手できます。
　獲得可能なチャレンジメダルは、クリア報酬として2個、

　ミッション報酬として6個となり、本ミニイベント開催期間中で

　合計最大8個のチャレンジメダルを入手できます。

七賢者チャレンジクエストで入手することのできるチャレンジメダルは、
「トレードショップ＞限定交換＞黄金の記憶」にて、
限界突破素材や「エトワリウムのかけら」と交換できます。

 

また、「エトワリウムのかけら」を4つ集めると、
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

 

 

■交換可能期間

　4月29日 0:00 ～ 5月17日 23:59

 

注意事項

・ミニイベント「黄金の記憶」開催期間中は、
　「5月強敵チャレンジクエスト」は開催されません。
　ミニイベント開催終了後の「5月強敵チャレンジクエスト」では、引き続き
　七賢者・ジンジャーが登場します。
　5月強敵チャレンジクエストにつきましては、

　別途公開予定のお知らせをご確認ください。

・クエストキー　銀はミッション報酬やイベント報酬で入手することができます。

・「七賢者チャレンジクエスト」をクリアした際には、
　アイテムのドロップはございませんのでご注意ください。

・「七賢者チャレンジクエスト」での達成報酬は、
　通常の強敵チャレンジクエストには持ち越されず、リセットされます。

・チャレンジメダルやエトワリウムのかけらは、
　「七賢者チャレンジクエスト」の期間終了後も所有することができ、
　通常の強敵チャレンジクエストの
　「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて
　再度交換を行うことができます。

・イベントの仕様や日程は、予告なく変更する可能性がございます。
