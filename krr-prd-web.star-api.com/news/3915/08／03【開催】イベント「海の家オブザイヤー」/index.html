【開催】イベント「海の家オブザイヤー」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

8月3日 17:00より、イベント「海の家オブザイヤー」を開催いたします。

 

 

開催期間

8月3日 17:00 ～ 8月30日 11:59

 

 

イベント内容

本イベントは、シナリオクエストと素材収集クエスト、【乱戦】クエストで構成されています。

シナリオクエストでは、『きららファンタジア』オリジナルストーリーが楽しめ、
イベント期間中様々なアイテムと交換可能な素材を入手できます。

高難易度な【乱戦】クエストでは、初回クリアで星彩石、
限界突破素材などの豪華な報酬を入手できます。

 

 

遊び方

シナリオクエストをクリアするとストーリーを読むことができます。


また、クエストで手に入るイベント専用アイテム「水鉄砲」「かき氷」を集めると
トレードショップにてイベント限定キャラクターや限界突破素材など、様々な報酬と交換できます。

 

 

イベントで入手できる専用アイテムは
トレードショップ＞限定交換＞「海の家オブザイヤー」から交換できます。

 

交換可能期間：

8月3日 17:00 ～ 9月6日 13:59

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持つキャラクターが存在します。

 

イベントボーナスを持つキャラクターを編成することで、イベント「海の家オブザイヤー」のクエストにおいて
イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

※★5 ゆずこ【水着】は今後開催予定のイベントでもイベントボーナス効果を持つため、他の「★5【水着】キャラクター」よりもボーナス値を低く設定しております。

 

 



以下キャラクターにつきましては、8月後半開催のイベント・召喚にて入手可能となります。

 

 

豪華素材がドロップする「【乱戦】クエスト」！

今回のイベントでは
クリアすることで豪華報酬が手に入る「【乱戦】クエスト」が発生します。
【乱戦】クエストは、イベント開始後の2018年8月6日 17:00から開催されます。

 

【乱戦】クエストに挑戦するためには、
【乱戦】ビーチ防衛戦挑戦券を消費する必要があります。
【乱戦】ビーチ防衛戦挑戦券は、「素材収集クエスト上級」で入手することができます。

 

乱戦クエストをクリアすると、イベント共通の素材と、
まれに★4限界突破素材を入手することができます。
その他、クエスト初クリア時や、一人も撤退せずにクリアした時などに特別な報酬を入手することができます。

 

※使用しない【乱戦】ビーチ防衛戦挑戦券は、
　トレードショップ＞限定交換＞「海の家オブザイヤー」から
　「かき氷」に交換することができます。
