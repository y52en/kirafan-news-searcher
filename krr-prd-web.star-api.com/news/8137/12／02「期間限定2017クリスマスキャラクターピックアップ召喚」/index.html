2017年の期間限定クリスマスキャラクターが再登場する

開催期間

12月2日 17:00 ～ 12月10日 23:59

ピックアップ対象キャラクター

★5 ユー子【クリスマス】（ナイト　月属性）
★5 アリス【クリスマス】（まほうつかい　土属性）
★5 はじめ【クリスマス】（せんし　水属性）

★4 るん【クリスマス】（アルケミスト　水属性）
★4 たまき【クリスマス】（そうりょ　風属性）

内容

開催期間中、ピックアップ対象キャラクターの出現率がアップしています。

「1回召喚」では星彩石40個、「10回召喚」では星彩石400個を消費しますが、
初回のみ星彩石300個で「10回召喚」が行えます。
さらに、1日1回のみ有償星彩石10個で「1日1回有償限定召喚」が行えます。

注意事項

・本召喚で登場した期間限定キャラクターは、今後の更新において
　再登場する可能性がございます。
・ピックアップ対象のキャラクターは同じレアリティの他のキャラクターよりも
　提供割合が高く設定されています。
・本召喚の提供割合は、
　MENU＞召喚＞本召喚のバナー＞提供割合 よりご確認ください。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・召喚を行った際に「星屑のかけら」を「1回召喚」で1個、
　「10回召喚」で10個入手できます。
・通常の「1回召喚」「10回召喚」では無償星彩石から使用されます。
・所持している星彩石の内訳は、召喚画面下部の表示または、
　MENU＞設定・その他＞購入履歴 よりご確認ください。
・「1日1回有償限定召喚」は、毎日0時に回数がリセットされます。
　「1日1回有償限定召喚」に対応している召喚が同時に複数開催されている場合は、
　各召喚につき1日1回までご利用いただけます。
・内容は予告なく変更になる場合がございます。ご了承ください。
