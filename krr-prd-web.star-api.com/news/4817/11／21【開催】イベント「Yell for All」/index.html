【開催】イベント「Yell for All」

『きららファンタジア』をご利用いただき、誠にありがとうございます。
11月21日 17:00より、イベント「Yell for All」を開催いたします。



開催期間

11月21日 17:00 ～ 12月4日 13:59


「アニマエール！」参戦！


『きららファンタジア』に「アニマエール！」が参戦！
「アニマエール！」より鳩谷 こはね、有馬 ひづめ、猿渡 宇希が登場します。
※舘島 虎徹、牛久 花和も今冬に登場予定です。

また、「アニマエール！」のコンテンツ施設
「チアアップハイスクール」が追加されます。
「チアアップハイスクール」は、イベント「Yell for All」開催と
同時に里で建設可能になります。


イベント概要


本イベントは、シナリオクエストと収集クエストで構成されています。
イベントの報酬や遊び方の詳細は、イベント画面＞イベント詳細をご確認下さい。

シナリオクエストでシナリオを楽しもう！

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめます。

シナリオクエストは3シーズンに分かれています。
2シーズン目は3つのルートに分かれており好きなルートからシナリオを読み始められます。
シナリオをすべてクリアすると次のシーズンのシナリオを読むことができます。

収集クエストで素材を集めてアイテムと交換しよう！

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換可能です。
また、より多くのイベント素材が入手できる、
難易度の高い収集クエスト「Cheer Up!」を11月26日 17:00 に開放いたします。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになります。


★4 鳩谷 こはね（まほうつかい　陽属性）がすぐに手に入る！！


購入可能期間
11月21日 17:00 ～ 12月3日 23:59

・ホーム画面上部＞星彩石購入ウィンドウ
・商店街画面＞星彩石購入ボタン

・販売価格：480円
・星彩石（有償）×40個
・購入日から14日間、ログインごとに星彩石（無償）×10個（最大合計140個）
・「Yell for All」イベント素材「ポンポン」×100個

※星彩石（無償）は購入日を含めた14日間、
　毎日初回ログイン時（1日目は購入直後）に入手できます。
　ただし、期間中ログインを行わなかった日は星彩石（無償）を入手できません。
※2日目以降の星彩石（無償）はプレゼントリストへ送られます。
※購入後のきららパスポートの有効期限は星彩石購入ウインドウの
　「おすすめ」タブもしくは「購入履歴」から確認できます。
※「Yell for All」きららパスポートは1度しか購入できません。
※イベント「Yell for All」終了後も有効期限までの間は
　毎日初回ログイン時に星彩石（無償）が入手できます。
