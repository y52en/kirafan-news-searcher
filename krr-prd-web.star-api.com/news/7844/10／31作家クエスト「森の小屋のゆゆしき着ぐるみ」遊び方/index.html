作家クエスト「森の小屋のゆゆしき着ぐるみ」遊び方

三上小又先生の原案シナリオを楽しもう！

作家クエスト「森の小屋のゆゆしき着ぐるみ」では、
「ゆゆ式」の原作者、三上小又先生原案のシナリオが楽しめます。

 

11月5日より三上小又先生による開発裏話を公開予定！

ぜひご覧ください！

 

 

収集クエストでぶき素材や【超強敵】チケットを集めよう！

シナリオクエストをクリアすると、収集クエスト・【超強敵】クエスト【極】クエストが開放されます。

収集クエストでは、まれに【超強敵】クエストに挑戦するために必要な専用の挑戦券がドロップします。

収集クエストをクリアして、【超強敵】クエストに挑戦しましょう！

 

■ぶき素材ドロップ内容一覧

・怪しい爪
・銅の繊維
・電気を帯びた鉱石
・神樹の枝
・竜の大牙
・ブロンズ塊
・ブラッドファング
・千里眼
etc.

 

豪華素材がドロップする「【超強敵】クエスト」開催！

【超強敵】クエストではぶき強化素材がドロップする他、
まれに「エトワリウムのかけら」がドロップします。

「エトワリウムのかけら」を4つ集めると、トレードショップの通常交換で
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

【超強敵】クエストをクリアし、ぶきを強化しましょう！

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたら
さらなる高難易度【極】クエストに挑戦してみましょう！
【極】クエストに挑戦するためにはスタミナを消費します。
