イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「応援旗」や
「はちまき（紅白）」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「開催！エトワリア大運動会」からご確認ください。

また、イベント素材の交換期限は6月5日13:59までとなっておりますので
期限までに交換をお願いいたします。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。


イベント限定コールチケットは、トレードショップで
★4 ランプ【運動会】（ナイト　陽属性）に交換が可能です。

たくさん交換して、★4 ランプ【運動会】（ナイト　陽属性）を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。

※★4 ランプ（ナイト　陽属性）の覚醒は、最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。

※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。
