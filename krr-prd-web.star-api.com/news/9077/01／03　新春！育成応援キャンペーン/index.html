01/03
　新春！育成応援キャンペーン

『きららファンタジア』をご利用いただき、誠にありがとうございます。

1月5日より、
「新春！育成応援キャンペーン」を開催いたします。

育成が捗る収集クエスト&【乱戦】クエストが登場！

開放期間
1月5日 0:00 ～ 1月15日 13:59

新芽や大種がドロップする収集クエストが登場します。
また、【乱戦】クエストの挑戦に必要な「【乱戦】エトワリア蔵開き挑戦券」も
まれにドロップします。

開放期間
1月5日 0:00 ～ 1月15日 13:59

【乱戦】クエストでは、収集クエストよりも多くの
新芽や大種を集めることができます。

※使用しない「【乱戦】エトワリア蔵開き挑戦券」は、
　MENU＞トレード＞限定交換＞「新春！育成応援キャンペーン」から
　「クエストキー　金」に交換することができます。

交換メダルが入手できる特別ミッション実施！

実施期間
1月5日 0:00 ～ 1月15日 13:59

「交換メダル」が入手できる特別ミッションを実施します。
実施期間中に毎日ミッションをクリアすることで、
「交換メダル」が最大1,100個入手できます。

また、集めた交換メダルは、
MENU＞トレード＞限定交換＞「新春！育成応援キャンペーン」から
各クラスの星結晶に交換することができます。

そのほかにも、
「新春！育成応援キャンペーン」のクエストの通算クリア回数に応じた
ミッションもございます。
内容の詳細は、MENU＞ミッション＞イベント よりご確認ください。

※「交換メダル」100個が報酬のミッション
　「新春！育成応援キャンペーンのクエストを5回クリア」は毎日リセットされます。
　報酬は当日中にお受け取りください。
※限定交換「新春！育成応援キャンペーン」は1月30日 13:59までです。
　交換忘れがないようご注意ください。

作家クエストスタミナ半減キャンペーン開催！

開催期間
1月5日 0:00 ～ 1月15日 13:59

開催期間中、作家クエストの消費スタミナが1/2になります。
この機会にまだ遊んだことのないクエストをプレイしてみましょう！

注意事項

キャンペーンの内容は、予告なく変更になる場合があります。ご了承ください。

 

昨年は皆さまから多くのご愛顧を賜り誠にありがとうございました。
本年も『きららファンタジア』をどうぞよろしくお願い申し上げます。
