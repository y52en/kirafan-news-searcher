【告知】強敵チャレンジクエスト実施について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

7月7日 0:00より、「強敵チャレンジクエスト」を実施いたします。「強敵チャレンジクエスト」は、毎週末（土曜・日曜）に発生する高難易度のクエストです。

 

開催期間

7月7日 0:00 ～ 7月8日 23:59

7月14日 0:00 ～ 7月15日 23:59

7月21日 0:00 ～ 7月22日 23:59

7月28日 0:00 ～ 7月29日 23:59

 

 

内容

今月の強敵は、「ペラペラのドラゴン」「ペラペラのイノシシ」「ペラペラの兵士」の3体です！

 

※強敵チャレンジクエストで登場する敵は、過去のイベント「ねねっちクエスト」に登場した際の強さとは異なります。

 



強敵チャレンジクエストは「ホーム＞クエスト＞強敵チャレンジクエスト」から挑戦可能です。 強敵チャレンジクエストに挑戦する際には、クエストキー　銀×3個を消費する必要があります。

 

※クエストキー　銀はミッション報酬やイベント報酬で入手することができます。

 

 

強敵チャレンジクエストをクリアすると報酬を入手することができます。

 

強敵チャレンジクエストを初めてクリアした時：チャレンジメダル×1個

強敵チャレンジクエストで、一人も撤退しないでクリアした時：チャレンジメダル×1個

 

※強敵チャレンジクエストをクリアした際には、アイテムのドロップはございませんのでご注意下さい。

 

※強敵チャレンジクエストでの達成報酬は翌週には持ち越されず、毎週リセットされます。（前週に報酬を入手していない場合、翌週クリア時に、前週と翌週の達成報酬は二重で獲得はできません）

 

 

アイテム「チャレンジメダル」！


 強敵チャレンジクエストで入手することのできるチャレンジメダルは、「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて、様々なアイテムと交換することができます。

 

「チャレンジメダル」で交換できるアイテム

・せんしの星結晶

・ナイトの星結晶

・まほうつかいの星結晶

・アルケミストの星結晶

・そうりょの星結晶

 

交換可能期間：

7月7日 0:00 ～ 8月3日 23:59

 

※チャレンジメダルは、7月の強敵チャレンジクエストの期間終了後も所有することができ、次回開催時の「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて再度交換を行うことができます。

 

 

注意事項

・イベントの仕様や日程は、予告なく変更する可能性がございます。
