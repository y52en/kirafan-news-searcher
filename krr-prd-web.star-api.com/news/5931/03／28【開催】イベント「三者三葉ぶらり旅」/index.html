03/28
【開催】イベント「三者三葉ぶらり旅」


『きららファンタジア』をご利用いただき、誠にありがとうございます。
3月28日 17:00より、イベント「三者三葉ぶらり旅」を
開催いたします。

 

開催期間

3月28日 17:00 ～ 4月10日 13:59

 

「三者三葉」参戦！


『きららファンタジア』に「三者三葉」が参戦！
「三者三葉」より西川 葉子、小田切 双葉、葉山 照が登場します。

また、「三者三葉」のコンテンツ施設
「張りぼて三つ葉館」が追加されます。
「張りぼて三つ葉館」は、イベント「三者三葉ぶらり旅」開催と
同時に里で建設可能になります。

 

イベント概要


本イベントは、シナリオクエストと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。
イベントの報酬や遊び方の詳細は、イベント画面＞イベント詳細をご確認ください。

 

シナリオクエストでストーリーを楽しもう！

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめます。

 

収集クエスト、【超強敵】クエストで素材を集めてアイテムと交換しよう！

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになります。

専用の挑戦券を消費することで挑戦できる【超強敵】クエストでは
より多くのイベント素材や限界突破素材が入手できます。

 

さらなる挑戦！【極】クエスト！

【超強敵】クエストもクリアした方は難易度の高いクエスト、
【極】クエストに挑戦してみましょう！
初回クリアで星彩石×40個、クリアコンプリート報酬で、
イベント素材「抽選マシーン」が入手できます。

 

★4 西川 葉子（まほうつかい　月属性）がすぐに手に入る！！
