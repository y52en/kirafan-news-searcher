『きららファンタジア』サービス開始いたしました！

 

12/11(月)より、『きららファンタジア』のサービスを開始いたしました！

 


サービス開始にともない、事前登録特典のプレゼント配布や、ゲーム内ログインボーナスも開始いたします。


以後、皆様にお楽しみいただけますよう尽力してまいりますので、『きららファンタジア』を何卒よろしくお願いいたします。

 

事前登録特典について

 

 

リリース後30日以内にゲームを開始されたお客様を対象に、事前登録特典をプレゼントいたします。

アイテムによって配布方法が異なるのでご注意ください。アイテムの配布方法は以下の通りになります。

 

※原作者の描き下ろし色紙（『ひだまりスケッチ』『Aチャンネル』『ゆゆ式』『うらら迷路帖』『きんいろモザイク』『ステラのまほう』『NEW GAME!』『がっこうぐらし！』）の抽選結果は別途公式ページにて告知いたします。

ログインボーナスについて

 

連続してゲームにログインすることで、ログイン回数に応じたアイテムをもらうことができます。ボーナスは7日目まで内容が連続し、7日目のログインボーナスを受け取ると、再び1日目の報酬へと戻ります。


また、途中でログインしない日があると、連続ログインボーナスはリセットされてしまいますのでご注意ください。
