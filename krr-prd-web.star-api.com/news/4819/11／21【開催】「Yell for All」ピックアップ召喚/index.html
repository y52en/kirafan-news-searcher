【開催】「Yell for All」ピックアップ召喚


『きららファンタジア』をご利用いただき、誠にありがとうございます。

11月21日 17:00より、 「Yell for Allピックアップ召喚」を開催いたします。

 

開催期間

11月21日 17:00 ～ 12月4日 13:59

 

 

ピックアップ対象キャラクター

ピックアップ召喚は、前半と後半に分かれています。
前半と後半でピックアップ対象が異なりますのでご注意下さい。

■ピックアップ召喚前半：11月21日 17:00 ～ 12月4日 13:59
＜ピックアップ対象＞
★5 鳩谷 こはね（まほうつかい　炎属性）
★4 有馬 ひづめ（ナイト　炎属性）
★4 猿渡 宇希（せんし　風属性）

■ピックアップ召喚後半：11月25日 0:00 ～ 12月4日 13:59
＜ピックアップ対象＞
★5 百地 たまて（せんし　水属性）
★4 折部 やすな（ナイト　水属性）
★4 ソーニャ（せんし　月属性）

 

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「Yell for Allピックアップ召喚」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、11月21日 17:00から開催されるイベント
「Yell for All」にてイベントボーナス効果を持っています。

イベントボーナス効果を持つキャラクターを編成すると、
イベント「Yell for All」のクエストにおいて、
イベント素材のドロップ数がアップします。

イベントボーナスに関する詳細は、
イベント「Yell for All」のイベント詳細をご確認下さい。
