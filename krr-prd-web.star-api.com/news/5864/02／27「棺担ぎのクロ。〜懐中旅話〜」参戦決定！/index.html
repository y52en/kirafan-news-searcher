「棺担ぎのクロ。〜懐中旅話〜」参戦決定！

『きららファンタジア』に、「棺担ぎのクロ。～懐中旅話～」の新規参戦が
決定しました！


「棺担ぎのクロ。～懐中旅話～」とは

「棺担ぎのクロ。～懐中旅話～」は、
芳文社「まんがタイムきらら」にて連載されていた
きゆづきさとこ先生による4コマ漫画作品です。

棺を背負った旅人クロと人の言葉を話す蝙蝠のセン。
放置された屋敷で出会った白い双子・ニジュクと
サンジュを道行きに加えた一行の目的は、
魔女にかけられた黒い呪いの秘密を探すこと。
自分を取り戻すための、ちょっと奇妙な旅物語。

外伝コミックスの「棺担ぎのクロ。～追憶旅話～」は、2月27日発売です！
また、ゲームへの登場は2019年4月を予定しております。

きゆづきさとこ先生が描かれたクロモンやアルシーヴも登場する
特別な新規参戦イベントを開催予定ですのでお楽しみに！

参戦決定ムービーも公開中！
