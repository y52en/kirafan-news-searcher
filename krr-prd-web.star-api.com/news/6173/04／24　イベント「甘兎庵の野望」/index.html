　イベント「甘兎庵の野望」


『きららファンタジア』をご利用いただき、誠にありがとうございます。
4月25日 17:00より、イベント「甘兎庵の野望」を
開催いたします。

 

開催期間

4月25日 17:00 ～ 5月14日 13:59

 

イベント概要


本イベントは、シナリオクエストと収集クエスト、
【乱戦】クエストで構成されています。
イベントの報酬や遊び方の詳細は、イベント画面＞イベント詳細をご確認ください。

 

シナリオクエストでストーリーを楽しもう！

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめます。


シナリオクエストはシーズンで区切られており、
前シーズンのすべてのシナリオクエストをクリアしていると、
指定の日時以降に次シーズンのシナリオクエストがプレイできるようになります。

 

【シーズン開放スケジュール】

・シーズン1
4月25日17:00 ～ 5月14日13:59

・シーズン2

4月27日17:00 ～ 5月14日13:59

・シーズン3

4月29日17:00 ～ 5月14日13:59

※「甘兎庵の野望」の各シナリオを読むことで
　初回クリア報酬として星彩石×5個を獲得できます。

※シーズン3の14節のみ、初回クリア報酬は星彩石×5個ではなく

　「お手製のテーブルセット」 ×1個となります。

 

収集クエスト、【乱戦】クエストで素材を集めてアイテムと交換しよう！

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになるほか、

全編フルボイスの「メモリアルクエスト」の続きが読めるようになります。

※各キャラクターの「メモリアルクエスト」は、
　クエスト＞メモリアルクエストから閲覧できます。

 

さらなる挑戦！【乱戦】クエスト！

シナリオクエストをクリアした方は難易度の高いクエスト、
【乱戦】クエストに挑戦してみましょう！
初回クリアで星彩石×40個、クリアコンプリート報酬で、
イベント素材「お手製のテーブルセット」×1個が入手できます。

 

★4千夜（そうりょ　炎属性）がすぐに手に入る！！


購入可能期間
4月25日 17:00 ～ 5月13日 23:59
