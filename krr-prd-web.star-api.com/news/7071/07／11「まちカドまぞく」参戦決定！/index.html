「まちカドまぞく」参戦決定！

『きららファンタジア』に、「まちカドまぞく」の新規参戦が決定しました！

「まちカドまぞく」とは

芳文社「まんがタイムきららキャラット」にて連載中の
伊藤いづも先生による4コマ漫画作品です。

15歳のある朝、封印されし魔族の力に覚醒した
吉田優子の任務は、町のどこかに住む「魔法少女」を
倒すこと!?　ツノと尻尾は生えたけど、力はフツーの
女の子以下な優子が、一族の「月４万円生活の呪い」
解除を目指して奮闘する、日常系マジカル４コマ！

ゲームへの登場は、2019年9月を予定しております。
また、7月11日よりTVアニメの放送がスタートします。
放送情報等は、TVアニメ「まちカドまぞく」公式ホームページをご確認ください。

「まちカドまぞく参戦決定記念ログインボーナス」開催！

TVアニメの放送期間中、毎週木～金曜日の間にログインすると
通常のログインボーナスに加えて、星彩石（無償）20個が貰えます！

アニメをチェックしながら、
『きららファンタジア』への参戦を楽しみにお待ちください♪
