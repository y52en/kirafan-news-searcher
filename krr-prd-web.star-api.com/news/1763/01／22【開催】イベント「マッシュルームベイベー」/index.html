【開催】イベント「マッシュルームベイベー」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。
1月22日 17:00より、「キルミーベイベー」正式参戦に合わせた新イベント「マッシュルームベイベー」を開催いたします。

 

 

開催期間

2018年 1月22日 17:00 ～ 2018年 1月31日 13:59

 

イベント内容

 

本イベントでは、ストーリーが楽しめ、イベント期間中様々なアイテムと交換可能な素材が手に入るシナリオクエストと、高難易度である代わりに、豪華な報酬が手に入る強敵クエスト・超強敵クエストをプレイできます。

※これまでのイベントと異なり、素材収集クエストはございません。シナリオクエストにて、各種イベント素材を入手することができます。

 

遊び方

 

シーズン開放スケジュール
シーズン1：2018年 1月22日 17:00 ～ 2018年1月31日 13:59
シーズン2：2018年 1月24日 17:00 ～ 2018年1月31日 13:59
シーズン3：2018年 1月26日 17:00 ～ 2018年1月31日 13:59


イベントは全部で3つのシーズンで構成されています。
シーズンは日ごとに開放され、そのシーズンのシナリオクエストを進めることで新たなストーリーを読むことができます。
また、クエストクリアで手に入る素材を集めることで、トレードショップでさまざまなアイテムと交換できます！

 

全シーズン共通の「きのコイン」と、各シーズンのクエストでしか入手できない「やすなす」「ソーナッツ」「変わり身のキノコ」を集めて豪華アイテムを手に入れましょう。

 

イベントで入手できる各素材はトレードショップ＞限定交換から以下のアイテムと交換可能です。
各素材を手に入れることで、その素材で交換できるアイテムがラインナップに追加されます。

 

 

交換可能期間：
2018年 1月 22日 17:00 ～ 2018年 2月7日 13:59 まで

※2月6日のメンテナンス実施に伴い、アイテム交換可能期間を

2月7日 13:59までに延長しております。

「きのコイン」で交換できるアイテム
・クエストキー　金
・限界突破きのこ

 

 

 

きのコインを集めて交換できる「限界突破きのこ」は、「コンテンツ限定限界突破素材」と交換できます。
「コンテンツ限定限界突破素材」は、対応する作品の★5キャラクターならクラスを問わずに使用することのできるアイテムです！
期間中、以下のものと各1個ずつ交換をすることができます。

※各作品につき、最大3個まで交換可能です。

 

・キルミーベイベーの星結晶
・ゆゆ式の星結晶
・ひだまりスケッチの星結晶

 

 

豪華素材がドロップする「強敵クエスト」も！

 

 

倒すことが困難な強敵モンスターの登場する「強敵クエスト」「超強敵クエスト」が各シーズンごとに開催されます。
「強敵クエスト」「超強敵クエスト」は、各シーズンのシナリオクエストを全てクリアすると開放されます。

今回の「強敵クエスト」は、シーズン１、シーズン２、シーズン３で異なる強敵が出現します。これらの強敵クエストに挑戦するためには、スタミナを消費する必要があります（「超強敵クエスト」とは異なり、「強敵クエスト」には挑戦券は必要ありません）。

 

強敵クエストをクリアすると、イベント共通の素材と各シーズンの素材、まれに「【超強敵】挑戦券」を入手することができます。

 

 

この「【超強敵】挑戦券」を消費することで「超強敵クエスト」に挑戦することができ、見事超強敵クエストをクリアすると、イベント共通の素材と各シーズンの素材、まれに☆4限界突破素材を入手できます。

 

パーティーを整えて、強敵クエスト・超強敵クエストに挑戦してみましょう。

 

※強敵クエストの発生期間は2018年1月22日 17:00 ～ 1月31日 13:59です。
