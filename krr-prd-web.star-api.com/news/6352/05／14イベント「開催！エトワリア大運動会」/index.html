イベント「開催！エトワリア大運動会」


『きららファンタジア』をご利用いただき、誠にありがとうございます。
5月15日 17:00より、イベント「開催！エトワリア大運動会」を
開催いたします。

本イベントは、全ユーザー協力型のイベントとなっております。

 

開催期間

5月15日 17:00 ～ 5月29日 13:59

 

イベント概要

本イベントは、シナリオクエストと討伐クエストで構成されています。
討伐クエストとはクエストに応じたイベントポイントを集められるクエストで、
ユーザー全員で獲得したイベントポイントの合計に応じて、
豪華報酬を受け取ることができます。

報酬内容や遊び方の詳細は、
別途掲載のおしらせ「イベント『開催！エトワリア大運動会』遊び方」または イベント画面＞イベント詳細をご確認ください。

 

シナリオクエストでオリジナルストーリーを楽しもう！

 

シナリオクエストをクリアして、討伐クエストを開放しよう！

シナリオクエストはシーズンで区切られており、
前シーズンのすべてのシナリオクエストをクリアしていると、
指定の日時以降に次シーズンのシナリオクエストがプレイできるようになります。

また、シナリオクエストをシーズン3まで全てクリアした後には
全2節からなるシナリオ「特別種目」が開放されます。

シナリオクエストを全てクリアすると、より難易度の高い

「全体競技」がプレイできるようになります。

※「開催！エトワリア大運動会」の各シナリオを読むことで
　初回クリア報酬として星彩石×5個を獲得できます。

※シーズン3の17節のみ、初回クリア報酬は星彩石×5個ではなく

　「トロフィー」×1個となります。

 

討伐クエストで素材やポイントを集めよう！

本イベントのクエストではイベント素材「応援旗」「はちまき（紅白）」がドロップし

討伐クエストでは、イベント素材とイベントポイントの両方を手に入れることができます。

 

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。
交換できるアイテムには、「召喚チケット」や

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」があります。

 

また、ユーザー全員で獲得したポイントの合計に応じて、

豪華報酬を受け取ることができます。

 

報酬一覧や遊び方の詳細は、

別途掲載のおしらせ「イベント『開催！エトワリア大運動会』遊び方」または

イベント画面>イベント詳細をご確認ください。
