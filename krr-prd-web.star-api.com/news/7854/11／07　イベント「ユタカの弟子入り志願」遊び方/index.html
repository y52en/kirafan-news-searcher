　イベント「ユタカの弟子入り志願」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「エアガン」や
「ランニングシューズ」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
MENU＞トレード＞限定交換＞「ユタカの弟子入り志願」から
ご確認ください。
また、イベント素材の交換期限は11月27日 13:59までとなっておりますので
期限までに交換をお願いいたします。

キャラクターのレベルが一定以上になると行えます。
限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。


イベント限定コールチケットは、トレードショップで
★4 ユタカ（ナイト　水属性）に交換が可能です。

たくさん交換して、

★4 ユタカ（ナイト　水属性）を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。

※★4 ユタカ（ナイト　水属性）の覚醒は、

　最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。

※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

「特訓かかし」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

 

イベントボーナス効果について

今回のイベントでは、一部のキャラクターがイベントボーナス効果を持っています。
イベントボーナス効果を持つキャラクターをパーティーに編成することで、
イベント「ユタカの弟子入り志願」のクエストにおいて
イベント素材のドロップ数がアップします。

★5 エンギ・スリーピース（ナイト　月属性）

 

▼エアガン+2個　ランニングシューズ+1個

★4 ユタカ（ナイト　水属性）
★5 ねね【水着】（せんし　土属性）
★5 メリー・ナイトメア（せんし　月属性）
★5 メリー【クリスマス】（ナイト　炎属性）
★5 ポルカ（せんし　炎属性）

※イベントボーナスの効果は、サブメンバー・サポート枠に編成された
　キャラクターからも受けることができます。

【超強敵】クエストに挑戦するためには、専用の挑戦券が必要です。

 

開放期間：11月10日 17:00 ～ 11月20日 13:59


※使用しない【超強敵】挑戦券は、
　MENU＞トレード＞限定交換＞「ユタカの弟子入り志願」から
　「ランニングシューズ」に交換することができます。

 

さらなる挑戦！「【極】クエスト」！
