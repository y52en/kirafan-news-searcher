　8月のトレードショップ更新について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

トレードショップの通常交換につきまして、
以下日程にてラインナップ更新や交換限度数のリセットを実施いたします。

 

トレードショップ通常交換　更新日時

8月1日 0:00

※今後も、毎月月初に通常交換の交換限度数のリセットを予定しております。

 

更新内容その1： 通常交換のラインナップにキャラクターを追加！

過去に登場した以下キャラクターを
通常交換の特殊召喚タブ内に追加いたします。

 

追加キャラクター：★4ランプ【運動会】、★4花名【荒野に咲く花】

追加日時：8月1日 0:00

交換に必要なアイテム数：星屑のかけら×20個　

限度数：1　

交換期間：期限なし

 

更新内容その2： 交換限度数のリセットを行います

通常交換内の「お得な召喚チケットセット」と、
「お得なクエスト挑戦セット」につきましては、
7月31日23:59に交換限度数をリセットいたします。

 

その他交換限度数に達していたアイテムにつきましても、
7月31日23:59に交換限度数をリセットし、
8月1日0:00より再度交換ができるようになります。

※特殊召喚タブ内のキャラクターにつきましては、
　交換限度数はリセットされません。ご注意ください。

 

今後とも、『きららファンタジア』をよろしくお願いいたします。
