注意事項

・販売期間中、各セットにつき1回限り購入出来ます。
・所持している星彩石の内訳は、召喚画面下部の表示または、
　MENU＞設定・その他＞購入履歴 よりご確認ください。
・「2周年記念スペシャルコールチケット」の使用期限は2020年1月2日 23:59です。
　期限を過ぎると「所持アイテム一覧」から自動的に削除され、
　使用不可となりますので、必ず期限内にご使用ください。
・「2周年記念スペシャルコールチケット」は、
　「★5選べるチケット付き！有償限定10回召喚・前半」と、
　「★5選べるチケット付き！有償限定10回召喚・後半」でも入手でき
　星彩石2,000個（2周年記念おまけつき）のご購入分と合わせて
　最大3枚まで入手できます。
・「2周年記念スペシャルコールチケット召喚」では
　「チャンスアップキーホルダー」を使用することはできません。
・「2周年記念スペシャルコールチケット召喚」で
　既に加入済みのキャラクターを仲間にすると、
　そのキャラクターのクラスの限界突破素材の入手に加えて
　最大覚醒していない場合はキャラクターが覚醒し、
　最大覚醒している場合は「輝星のかけら　金」を入手できます。
・キャンペーン内容は予告なく変更になる場合がございます。ご了承ください。
