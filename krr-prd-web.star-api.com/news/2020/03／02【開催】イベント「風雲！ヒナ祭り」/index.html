【開催】イベント「風雲！ヒナ祭り」

『きららファンタジア』をご利用いただき、誠にありがとうございます。
3月2日 メンテナンス後より、新イベント「風雲！ヒナ祭り」を開催いたします。

 

開催期間

3月2日 メンテナンス後 ～ 3月12日 13:59

 

イベント内容

本イベントでは、ストーリーが楽しめるシナリオクエストと、イベント期間中様々なアイテムと交換可能な素材が手に入る素材収集クエスト、高難易度である代わりに、豪華な報酬が手に入る【超強敵】クエストをプレイできます。

 

大沖先生描き下ろしのオリジナルキャラクターが登場！

 

大沖先生描き下ろしのオリジナルキャラクター「左大臣・右大臣」が登場します。

彼らの活躍にもご注目ください。

 

 

遊び方

シーズン開放スケジュール
シーズン1：3月2日 メンテナンス後 ～ 3月12日 13:59
シーズン2：3月4日 17:00 ～ 3月12日 13:59
シーズン3：3月6日 17:00 ～ 3月12日 13:59


イベントは全部で3つのシーズンで構成されています。
シーズンは日ごとに開放され、そのシーズンのシナリオクエストを進めることで新たなストーリーを読むことができます。
また、クエストクリアで手に入る素材を集めることで、トレードショップで様々なアイテムと交換できます！

 

全シーズン共通の「ヒナ団子」と、各シーズンのクエストでしか入手できない「ヒナあられ」「ヒナ餅」「ヒナ酒」を集めて豪華アイテムを手に入れましょう。

豪華素材がドロップする「【超強敵】クエスト」も！


倒すことの困難な強敵モンスターの登場する「【超強敵】クエスト」が開催されます。
「【超強敵】クエスト」はイベント開始後の3月６日 17:00から開催されますが、参加するためには各シーズンのシナリオクエストを全てクリアしている必要があります。



【超強敵】クエストに挑戦するためには、「【超強敵】風雲！ヒナ祭り挑戦券」を消費する必要があります。
【超強敵】風雲！ヒナ祭り挑戦券は、収集クエスト超級やイベントミッションから入手することができます。

 

 


今回の【超強敵】クエストでは、特定の条件でクリアすると報酬を入手することができます。

【超強敵】クエストを初めてクリアした時：星彩石×40個
【超強敵】クエストにおいて、一人も撤退しないで初めてクリアした時：せんしの果実×1個

また、見事【超強敵】クエストをクリアすると、イベント共通の素材と、まれに★4限界突破素材を入手できます。



パーティーを整えて、【超強敵】クエストに挑戦してみましょう。

 

※【超強敵】クエストの発生期間は3月6日 17:00 ～ 3月12日 13:59です。
※使用しない【超強敵】風雲！ヒナ祭り挑戦券は、トレードショップ＞限定交換＞「風雲！ヒナ祭り」から、イベント素材「ヒナ団子」に交換することができます。
