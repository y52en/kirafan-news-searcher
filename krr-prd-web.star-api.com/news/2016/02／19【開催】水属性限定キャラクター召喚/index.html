【開催】水属性限定キャラクター召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

2月19日 メンテナンス後より、「水属性限定キャラクター召喚」を開催いたします。水属性限定キャラクター召喚は、水属性のキャラクターのみが出現する特殊な召喚です。

 


 

開催期間

2月19日 メンテナンス後 ～ 3月2日 13:59

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・本召喚から出現するキャラクターにつきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。
