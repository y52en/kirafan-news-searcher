【特報２】メインクエストコミカライズ決定（3/25 13:00更新）

『きららファンタジア』をご利用いただき、誠にありがとうございます。

2019年春、創刊予定の芳文社公式漫画アプリ「COMIC FUZ」で
『きららファンタジア」メインクエストのコミカライズが連載決定いたしました！

連載開始は2019年夏を予定しております。(3/25 13:00追記）

「COMIC FUZ」とは

「COMIC FUZ」は、芳文社が発行しているマンガ雑誌や連載されている
人気マンガを楽しめるアプリです。
アプリ内でメダルを使用することで、約1000以上のマンガを楽しむことができ、
毎日更新される連載コーナーでは、
「COMIC FUZ」でしか読めない描き下ろしマンガが多数ラインナップされます。

創刊の際には、『きららファンタジア』にも参戦中の人気マンガ「ゆるキャン△」が、
現在の「まんがタイムきららフォワード」から「COMIC FUZ」に
連載の場を移します。
それ以外にも、多数の『きららファンタジア』参戦作品が掲載予定です。

「COMIC FUZ」事前登録受付中！

「COMIC FUZ」では、現在事前登録キャンペーンを実施中です。
事前登録キャンペーンでもらえるメダルでは、マンガを無料で読むことができます！
