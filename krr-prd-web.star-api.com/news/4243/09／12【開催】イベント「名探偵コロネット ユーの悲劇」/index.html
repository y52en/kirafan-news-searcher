名探偵コロネットは事件の真相を解き明かせるか！？


本イベントは、シナリオクエストと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換可能です。

より多くのイベント素材や限界突破素材が入手できる【超強敵】クエストは、
【超強敵】挑戦券を消費することで挑戦できます。
さらに高難易度な【極】クエストでは初回クリアで星彩石×40個が手に入ります。
 
イベントのあらすじや遊び方は、イベント画面＞イベント詳細をご確認下さい。
