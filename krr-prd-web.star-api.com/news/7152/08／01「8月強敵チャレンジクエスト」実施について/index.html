08/01
「8月強敵チャレンジクエスト」実施について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

8月3日 0:00より、「強敵チャレンジクエスト」を実施いたします。
「強敵チャレンジクエスト」は、毎週末（土曜・日曜）に発生する
高難易度のクエストです。

 

開催期間

8月3日 0:00 ～ 8月4日 23:59 

8月10日 0:00 ～ 8月11日 23:59 

8月17日 0:00 ～ 8月18日 23:59

8月24日 0:00 ～ 8月25日 23:59

8月31日 0:00 ～ 9月1日 23:59

 

内容

今月の強敵は、イベント「リュウグウ・アドベンチャー」

でも活躍中のオトヒメが登場します！

オトヒメは、芳文社「まんがタイムきららキャラット」にて
「まちカドまぞく」を連載中の伊藤いづも先生による

描き下ろしキャラクターとなっております。

 

強敵チャレンジクエストは、
「ホーム＞クエスト＞強敵チャレンジクエスト」から挑戦可能です。
強敵チャレンジクエストに挑戦する際には、
クエストキー　銀×3個を消費する必要があります。

 

強敵チャレンジクエストをクリアすると報酬を入手することができます。

初めてクリアした時：チャレンジメダル×1個

一人も撤退しないでクリアした時：チャレンジメダル×1個

 

アイテム「チャレンジメダル」を集めよう！

強敵チャレンジクエストで入手することのできるチャレンジメダルは、
「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて、
様々なアイテムと交換することができます。

「エトワリウムのかけら」は、
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

 

「チャレンジメダル」で交換できるアイテム

・エトワリウムのかけら
・せんしの星結晶 
・まほうつかいの星結晶 
・そうりょの星結晶 
・ナイトの星結晶 
・アルケミストの星結晶

 

「エトワリウムのかけら」で交換できるアイテム

・エトワリウム

 

交換可能期間：

8月3日 0:00 ～ 9月6日 23:59

 

注意事項

・クエストキー　銀はミッション報酬やイベント報酬で入手することができます。

・強敵チャレンジクエストをクリアした際には、
　アイテムのドロップはございませんのでご注意ください。

・強敵チャレンジクエストでの達成報酬は翌週には持ち越されず、
　毎週リセットされます。（前週に報酬を入手していない場合、
　翌週クリア時に、前週と翌週の達成報酬は二重で獲得はできません）

・チャレンジメダルやエトワリウムのかけらは、
　今月の強敵チャレンジクエストの期間終了後も所有することができ、
　次回開催時の「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて
　再度交換を行うことができます。

・イベントの仕様や日程は、予告なく変更する可能性がございます。
