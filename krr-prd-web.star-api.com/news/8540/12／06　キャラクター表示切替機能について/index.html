　キャラクター表示切替機能について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

12月10日実施予定のVer1.20.0アップデートにて
キャラクター表示切替機能を実装いたします。

キャラクター表示切替機能について

進化済みのキャラクターに限り、イラストおよびデフォルメキャラクターの
表⽰を進化前・進化後のどちらにするか⾃由に選択できるようになります。

手順

MENU＞編成＞キャラクター表示切替＞キャラクター一覧＞表示切替
にて、進化前・進化後の好きな方を選択し、
「設定する」ボタンをタップするとイラストとデフォルメキャラクターが
切り替わります。

なお、進化前に設定した場合は、
とっておきの演出も進化前のものになりますが
キャラクターのスキルレベルやステータスなどは変化しません。

設定した状態がホームやクエスト、ルームなどのキャラクター表示に反映されます。
お気に入りのイラストをお楽しみください♪

注意事項

・キャラクター一覧のアイコンなど、設定が反映されない画面もございます。
・表示切り替え設定はご自身のゲーム画面のみ反映され、
　他のユーザーから見える部分には影響いたしません。
　切り替え設定を行ったキャラクターを「おきにいりメンバー」や
　「サポートキャラクター」に設定している場合、
　他ユーザーのゲーム画面には「進化後」が表示されます。
・画像は開発段階のものです。
