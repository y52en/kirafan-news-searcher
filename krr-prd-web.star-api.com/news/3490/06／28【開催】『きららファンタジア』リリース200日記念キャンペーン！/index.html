【開催】『きららファンタジア』リリース200日記念キャンペーン！

『きららファンタジア』をご利用いただき、誠にありがとうございます。

6月28日に『きららファンタジア』はリリースから200日を迎えます。

 

6月28日 17:00より、『きららファンタジア』リリース200日を記念し、さまざまなキャンペーンやアップデートを実施いたします。

 

 

★5キャラクター専用ぶき＆ぶき進化機能実装！

 

6月28日 17:00より、「★5キャラクター専用ぶき」機能をリリースいたします。

「★5キャラクター専用ぶき」機能では、対象の★5キャラクターが特定の条件を満たすと「キャラクター専用のぶき」が入手できます。


また、「★5キャラクター専用ぶき」を強化し、特定の条件を満たすと「ぶき進化」ができるようになります。

「ぶき進化」には、イベントなどで入手できる新アイテム「エトワリウム」が必要になります。

さらに、「★5キャラクター専用ぶき」実装に伴い、専用ぶき対象キャラクターのフルボイスシナリオが実装された「メモリアルクエスト」が開放されます！

 

メモリアルクエスト開放には、特定の条件を達成する必要があります。

 

「★5キャラクター専用ぶき」についての詳細は、「新機能『★5キャラクター専用ぶき実装』のおしらせ」をご覧下さい。

 

 

キャラクタークロスシナリオ追加！

6月28日 17:00より、「クロスシナリオ」が追加されます。

 

今回追加されるキャラクターのクロスシナリオについては、

【図書館＞物語ふりかえり＞キャラシナリオ＞各作品内の対象キャラクター＞クロス】よりご確認下さい。

 

 

★5選べるチケット付き！有償限定10回召喚

『きららファンタジア』リリース200日を記念して、★5選べるチケット付き！有償限定10回召喚を開催いたします。

 

開催期間：6月28日 17:00 ～ 7月11日 11:59

※お一人様一回限り、有償星彩石限定の召喚となります。

 

本召喚を実行することで、★5キャラクター1人を指定して仲間にすることができる「リリース200日記念スペシャルコールチケット」が入手できます。

指定できる★5キャラクターにつきましては、ゲーム内の【召喚画面＞スペシャルコールチケット召喚】にてご確認下さい。

 

既に召喚済みの★5キャラクターを「スペシャルコールチケット」で仲間にすると、

そのキャラクターのクラスの限界突破素材を入手できます。

 

※「★5選べるチケット付き！有償限定10回召喚」は、有償星彩石400個で召喚可能です。無償星彩石では召喚できません。


※星彩石購入の際に付与されるオマケ分の星彩石は、無償星彩石となります。
所持している星彩石の内訳は、召喚画面下部の表示、もしくはホーム画面右上のMENU＞設定・その他ボタン＞購入履歴ボタンよりご確認いただけます。

 

※「リリース200日記念スペシャルコールチケット」の使用期限は7月18日 23:59です。期限が過ぎますと使用不可となりますので、必ず期限内での使用をお願いいたします。

 

※本キャンペーンの対象以外の★5キャラクターは、「リリース200日記念スペシャルコールチケット」で召喚することはできません。ご注意下さい。

 

※10回召喚を行った際には、特別アイテム「星屑のかけら」は入手できますが、「リリース200日記念スペシャルコールチケット」使用時の召喚では、特別アイテム「星屑のかけら」は入手できません。

 

※100日記念キャンペーンの「★5選べるチケット付き！有償限定10回召喚」にて入手したスペシャルコールチケットは使用できません。
