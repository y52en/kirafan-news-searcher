【開催】イベント「友達の友達は友達大作戦」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

4月11日 メンテナンス後より、「スロウスタート」参戦に合わせた新イベント「友達の友達は友達大作戦」を開催いたします。

 

 

「スロウスタート」が参戦！

『きららファンタジア』に「スロウスタート」が新規参戦！
一之瀬 花名・千石 冠・十倉 栄依子・百地 たまてが登場します。

さらに、「スロウスタート」のコンテンツ施設「スロウライフリゾート」が追加されます。
「スロウライフリゾート」は、イベント「友達の友達は友達大作戦」開催と同時に里で建設可能になります。

 

 

開催期間

4月11日 メンテナンス後 ～ 4月23日 13:59

 

 

イベント内容

本イベントでは、ストーリーが楽しめるシナリオクエストと、イベント期間中様々なアイテムと交換可能な素材が手に入る素材収集クエスト、高難易度である代わりに、豪華な報酬が手に入る乱戦クエストをプレイできます。

 

 

遊び方

シーズン開放スケジュール

シーズン1：4月11日 メンテナンス後 ～ 4月23日 13:59

シーズン2：4月13日 17:00 ～ 4月23日 13:59

シーズン3：4月15日 17:00 ～ 4月23日 13:59

 

イベントは全部で3つのシーズンで構成されています。

シーズンは日ごとに開放され、そのシーズンのシナリオクエストを進めることで新たなストーリーを読むことができます。

また、クエストクリアで手に入る素材を集めることで、トレードショップで様々なアイテムと交換できます！

 

 

全シーズン共通の「絵馬の根付」と、各シーズンのクエストでしか入手できない「湖の魚」「霊山の種火」「黄色いブーケ」を集めて豪華アイテムを手に入れましょう。

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターは、イベント「友達の友達は友達大作戦」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

 

 

豪華素材がドロップする「乱戦クエスト」！

倒すことの困難なモンスターが複数登場する「乱戦クエスト」が開催されます。
「乱戦クエスト」はイベント開始後の4月17日 17:00から開催されますが、参加するためには各シーズンのシナリオクエストを全てクリアしている必要があります。

 

 

「乱戦クエスト」に挑戦するためには、「【乱戦】友達大作戦挑戦券」を消費する必要があります。「【乱戦】友達大作戦挑戦券」は、各シーズンにて開放される「素材収集クエスト超級」から入手することができます。

 

乱戦クエストをクリアすると、イベント共通の素材と、まれに★4限界突破素材を入手することができます。

その他、特定の条件でクリアすると報酬を入手することができます。

 

【乱戦】クエストを初めてクリアした時：星彩石×40個

【乱戦】クエストにおいて、一人も撤退しないで初めてクリアした時：そうりょの果実×2個

 

※【乱戦】クエストの発生期間は4月17日 17:00 ～ 4月23日 13:59です。
※使用しない【乱戦】友達大作戦挑戦券は、トレードショップ＞限定交換＞「友達の友達は友達大作戦」から、イベント素材「絵馬の根付」に交換することができます。
