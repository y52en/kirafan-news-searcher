内容

　MENU＞設定・その他＞購入履歴 よりご確認ください。
・本召喚には「1回召喚」や「10回召喚」が存在せず、「3回召喚」のみとなります。
・本召喚を行った際に「星屑のかけら」を3個入手できます。
・本召喚で出現するキャラクターの詳細は、
　MENU＞召喚＞本召喚のバナー＞出現一覧 よりご確認ください。
・本召喚では、ピックアップ対象キャラクターは存在しません。
・本召喚は、1日1回限りとなっております。
　召喚を行うと召喚一覧から本召喚が消え、翌日0時に再度出現します。
