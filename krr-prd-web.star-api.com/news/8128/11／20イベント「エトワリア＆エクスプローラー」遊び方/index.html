イベント「エトワリア＆エクスプローラー」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「6面ダイス」や
「キャラクターシート」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
MENU＞トレード＞限定交換＞「エトワリア＆エクスプローラー」から
ご確認ください。
また、イベント素材の交換期限は12月11日 13:59までとなっておりますので
期限までに交換をお願いいたします。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。


イベント限定コールチケットは、トレードショップで
★4 櫟井 唯（そうりょ　炎属性）に交換が可能です。

たくさん交換して、

★4 櫟井 唯（そうりょ　炎属性）を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。

※★4 櫟井 唯（そうりょ　炎属性）の覚醒は、

　最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。

※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

「Ｅ＆Ｅ」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。
