【開催】「風雲！ヒナ祭り」限定ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

2月27日 メンテナンス後より、新イベント「風雲！ヒナ祭り」で活躍するキャラクターが登場する「期間限定ピックアップ召喚」を開催いたします。

 

ひな祭り衣装の★5「NEW GAME!」ゆん【ひな祭り】（CV.竹尾歩美）、★5「夢喰いメリー」勇魚【ひな祭り】（CV.茅野愛衣）、★4「ブレンド・S」苺香【ひな祭り】（CV.和氣あず未）が期間限定で登場します。

 


 

開催期間

2月27日 メンテナンス後 ～ 3月12日 13:59

※期間限定キャラクターはこの期間中のみ召喚から出現いたします。
※期間によって、ピックアップされる対象が異なりますのでご注意ください。

 

2月27日 メンテナンス後 ～ 3月4日 23:59　

ピックアップ対象：★5 ゆん【ひな祭り】　★4 苺香【ひな祭り】　
通常確率での出現：★5 勇魚【ひな祭り】


3月5日 0:00   ～ 3月12日 13:59　

ピックアップ対象：★5 勇魚【ひな祭り】　★4 苺香【ひな祭り】
通常確率での出現：★5 ゆん【ひな祭り】

 

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。
※ピックアップ変更時に、初回のみ星彩石300個で『10回召喚』が可能になります

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、3月2日 メンテナンス後から開催されるイベント「風雲！ヒナ祭り」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「風雲！ヒナ祭り」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「風雲！ヒナ祭り」のおしらせをご確認下さい。


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
