　Ver 1.17.0　改修内容の詳細

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.17.0アップデートにおいて実装予定の

改善内容の詳細についてご案内いたします。

 

とっておきゲージの改善

これまで、とっておきゲージが一定値以上の状態で
「とっておき」を使用し敵をスタン(気絶)状態にした際
ゲージ増加量が反映されておりませんでしたが

上記の場合でもゲージが増加するよう改修いたします。
これにより、さらに「とっておき」が発動しやすくなります。

 

WAVE開始時タイムライン順の改善

WAVE開始時またはリトライ時表示される行動順に、
行動速度(SPD)アップ・行動速度(SPD)ダウンがかかった場合の
ステータス増減分が反映されるようになります。
これにより、キャラクターの行動速度が変化した後の行動順が把握しやすくなり

戦闘をより有利に進めることができるようになります。

 

サポートキャラクターの残ターン数に関するバトル画面改修

サポートキャラクターがバトルに参加できる
残りターン数を表示するよう改修いたします。

また、NPCについてはバトルに参加できるターン数に制限が無くなります。

これにより戦闘不能にならない限り、戦闘終了まで

パーティーに参加することが可能です。

 

ルーム機能の改修

ルームでキャラクターを長押しすることにより、

任意の場所へと移動させられるようになります。

好きなキャラクターをキャラアクションがあるルームアイテムの近くへ

連れて行くことが可能です。

※フレンドルームでキャラクターを長押しした場合、

　移動はせず従来通りステータス画面を表示します。

 

また、ルームショップ＆倉庫にフィルター機能が実装されます。

「キャラアクションあり」「キャラアクションなし」等、目的別にルームアイテムを探しやすくなります。

 

オート解除の利便性向上

オートバトル状態の解除を「AUTOボタン」をもう一度タップ、または

「画面全体」をタップのいずれかの方法を選択することができるようになります。

また、上記に伴いAUTOボタンの位置を調整いたします。


皆様により一層『きららファンタジア』をお楽しみいただけますよう、
スタッフ一同鋭意努めてまいります。

今後とも、『きららファンタジア』をよろしくお願い申し上げます。
