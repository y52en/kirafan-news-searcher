　Ver 1.17.0の改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver 1.17.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・とっておきゲージの改善
　※とっておきゲージの仕様改修


・WAVE開始時タイムライン順の改善
　※戦闘中タイムラインにおいて表示される行動順に
　　SPDアップ系スキル・SPDダウン系スキルの効果が反映されるよう改修


・サポートキャラクターの残ターン数に関するバトル画面改修
　※サポートキャラクターが帰還するまでの

　　残りターン数を表示するよう改修

 

・NPCキャラクター帰還の改修
　※NPCキャラクターが帰還するまでの残りターン数を撤廃し、

　　戦闘終了までバトルに参加できるよう改修

 

・ルーム機能の改修

　※ルームでキャラクターを任意の場所に移動させる機能を追加

　※ルームショップにフィルター機能を追加

 

・オート解除の利便性向上
　※オートバトル状態の解除方法を

　「MENU＞設定・その他＞オプション＞バトル設定

　＞オート解除（タップ解除範囲）」で

　「AUTOボタンのみ」「画面全体」から選択可能になるよう改修
　※AUTOボタンの位置調整


・細かな機能改修
　・チュートリアルバトル中に表示されるポップアップ画像を変更

　・サポート選択画面の改修　

　・施設のLv.が里画面時に可視化されるよう改修
　・その他、機能改修など

 

※とっておきゲージの改善～オート解除の利便性向上の詳細については

　別途おしらせ「Ver 1.17.0　改修内容の詳細」をご確認ください。
