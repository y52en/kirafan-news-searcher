【告知】5月のトレードショップ更新について

『きららファンタジア』をご利用いただき、誠にありがとうございます。


トレードショップの通常交換につきまして、5月1日 0:00にラインナップ更新や交換限度数のリセットを行わせていただきます。

 

 

トレードショップ通常交換　更新日時

5月1日　0:00
※今後も、毎月月初に通常交換の交換限度数のリセットを予定しております。

 

 

更新内容その1： 通常交換のラインナップに追加・更新を行います

・キャラクター ★4 千矢【バレンタイン】を、通常交換の特殊タブ内に追加いたします。

※交換必要数：星屑のかけら×20個　限度数：1　期間：なし

・通常交換の特殊タブ内アイテム「クエストキー　金」の交換限度数を9→20に更新いたします。

 

更新内容その２： 交換限度数のリセットを行います

通常交換内の4月30日23:59が交換期限となっているアイテムにつきましても、4月30日23:59に交換限度数をリセットいたします。
交換限度数に達していたアイテムにつきましては、5月1日0:00より再度交換ができるようになります。

※特殊タブ内のキャラクターにつきましては、交換限度数はリセットされません。ご注意ください。
