★5選べるチケット付き！有償限定10回召喚

『きららファンタジア』リリース100日を記念して、★5選べるチケット付き！有償限定10回召喚を開催いたします。

 

 

開催期間：3月19日 メンテナンス後～ 3月28日 11:59
※お一人様一回限り、有償星彩石限定の召喚となります。

 

 

本召喚を実行することでスペシャルコールチケットを入手することができます。

 

「スペシャルコールチケット」は、★5キャラクター1人を指定して仲間にすることができるチケットです。指定することのできる★5キャラクターについては、必ずゲーム内の召喚画面＞スペシャルコールチケット召喚にて確認の上、「スペシャルコールチケット」をご使用ください。

 

既に加入済みのキャラクターが存在する場合に、その加入済みのキャラクターを「スペシャルコールチケット」で仲間にすると、そのキャラクターのクラスの限界突破素材を入手できます。

 

※「★5選べるチケット付き！有償限定10回召喚」は、有償星彩石400個で召喚可能です。無償星彩石では召喚できません。
※星彩石購入の際に付与されるオマケ分の星彩石は、無償星彩石となります。
所持している星彩石の内訳は、召喚画面下部の表示、もしくはホーム画面右上のMENUボタン＞設定・その他ボタン＞購入履歴ボタンよりご確認いただけます。

 

※「スペシャルコールチケット」の使用期限は2018年4月11日 23:59です。
※本キャンペーンの対象以外の★5キャラクターは、「スペシャルコールチケット」の召喚候補には追加されません。ご注意ください。

・特別アイテム「星屑のかけら」

召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。
（1回召喚で1個、10回召喚で10個入手できます）

スキルの効果調整

現在の戦闘進行や、お客様のプレイ状況等考慮し、主に以下の見直しと改修を行います。

 

 

・とっておきゲージ上昇の仕組みの見直し
これまで、とっておきゲージは、「敵にダメージを与える」スキル使用時のみ上昇していましたが、「敵にダメージを与える」スキル以外のクラススキル・ぶきスキルを使用した場合でも、とっておきゲージが上昇するようになります。

 

 

・スキルが再び使用可能になるまでの時間の見直し
全てのクラススキル・ぶきスキルについて、使用したスキルが再び使用可能になるまでの時間を全面的に見直し、戦闘中にスキルを複数回使用しやすいように調整を行います。

 

 

また、スキルLvが一定値に達した時、そのスキルが使用可能になるまでの時間がさらに短縮されます。これにより、スキルLvを上げる事でさらに戦闘中にスキルが複数回使用しやすくなります。

 

 

・スキルの効果量の調整
一部のキャラクターのクラススキルやとっておきについて、調整を行わせていただきます。

 

 

調整適用日時：3月19日 メンテナンス後～

 

 

詳細は、おしらせのアップデートタブ内「スキルの効果調整について」をご覧ください。

 

 

ログインボーナスリニューアル

通常開催されているログインボーナスの入手内容をリニューアルいたします。リニューアル後のボーナス内容は、100日記念キャンペーン終了後も恒常化します。

 

※リニューアルの際、ログインボーナスの進行状況を引き継いで、ログインボーナスの中身がリニューアルされます。例えば、3日目のログインボーナスを獲得した状態で、4日目時点でリニューアルが行われた場合、リニューアルされたログインボーナスを4日目のものから獲得することができます。

 

 

リニューアル日時：3月19日 メンテナンス後～
