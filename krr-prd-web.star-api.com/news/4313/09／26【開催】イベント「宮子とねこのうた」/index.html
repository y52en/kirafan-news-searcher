【開催】イベント「宮子とねこのうた」


『きららファンタジア』をご利用いただき、誠にありがとうございます。
9月27日 17:00より、イベント「宮子とねこのうた」を開催いたします。

開催期間

9月27日 17:00 ～ 10月9日 13:59


けいおん！×ひだまりスケッチ イベント開催！


本イベントは、シナリオクエストと収集クエスト、
乱戦クエストで構成されています。

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換可能です。

高難易度な【乱戦】クエストでは、初回クリアで星彩石、
限界突破素材などの豪華な報酬を入手できます。

イベントのあらすじや遊び方は、イベント画面＞イベント詳細をご確認下さい。

アイテムセット「宮子とねこのうた」きららパスポート販売開始！


購入可能期間
9月27日 17:00 ～ 10月8日 23:59

「宮子とねこのうた」きららパスポート　　
・販売価格：480円
・星彩石（有償）×40個
・購入日から14日間、ログインごとに星彩石（無償）×10個（最大合計140個）
・「宮子とねこのうた」イベント素材「ねこ村の釣り竿」×100個

きららパスポートはイベント期間に合わせて販売される、
お得なアイテムセットです。

きららパスポートを購入することで、
イベント報酬の★4 平沢 憂（ナイト　陽属性）を
すぐに入手するために必要な量の素材「ねこ村の釣り竿」も入手することができます。

購入場所
・ホーム画面上部＞星彩石購入ウィンドウ
・商店街画面＞星彩石購入ボタン

※星彩石（無償）は購入日を含めた14日間、
　毎日初回ログイン時（1日目は購入直後）に入手できます。
　ただし、期間中ログインを行わなかった日は星彩石（無償）を入手できません。
※2日目以降の星彩石（無償）はプレゼントリストへ送られます。
※購入後のきららパスポートの有効期限は星彩石購入ウインドウの
　「おすすめ」タブもしくは「購入履歴」から確認できます。
※「宮子とねこのうた」きららパスポートは1度しか購入できません。
※イベント「宮子とねこのうた」終了後も有効期限までの間は
　毎日初回ログイン時に星彩石（無償）が入手できます。
