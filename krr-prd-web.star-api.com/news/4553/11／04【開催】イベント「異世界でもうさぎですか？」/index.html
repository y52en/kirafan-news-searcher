イベント概要


本イベントは、シナリオクエストと収集クエスト、
乱戦クエストで構成されています。

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。
またシナリオクエストは前後編に分かれており、後編は
11月9日 17:00に開放されます。


シナリオクエストを進めていくと「ご注文はうさぎですか？」に登場する
「ラビットハウス」に関連したルームアイテム「木組みの街の背景」、
「ラビットハウスの壁」、「ラビットハウスの床」が入手できます。
ルームショップから購入できるルームアイテムと組み合わせて
あなたのルームを「ラビットハウス」にしましょう！

より多くのイベント素材や限界突破素材が入手できる乱戦クエストは、
【乱戦】挑戦券を消費することで挑戦できます。
初回クリア時のみ星彩石×40個を獲得できますので
【乱戦】挑戦券を入手したら挑戦してみましょう！

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになります。

イベントの遊び方詳細は、イベント画面＞イベント詳細をご確認下さい。
