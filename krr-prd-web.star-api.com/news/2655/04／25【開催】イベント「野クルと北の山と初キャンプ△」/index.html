【開催】イベント「野クルと北の山と初キャンプ△」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。
4月25日 メンテナンス後より、「ゆるキャン△」参戦に合わせた新形式イベント「野クルと北の山と初キャンプ△」を開催いたします。

 

今回のイベントはマップ探索イベント！
新形式イベントとして「マップ式クエスト」「ふしぎな宝箱」の、２つの新しい要素が追加されています。

 

 

「ゆるキャン△」が参戦！

『きららファンタジア』に「ゆるキャン△」が新規参戦！
志摩 リン・各務原 なでしこ・大垣 千明・犬山 あおいが登場します。

 

さらに、「ゆるキャン△」のコンテンツ施設「野クルツリーハウス」が追加されます。
「野クルツリーハウス」は、イベント「野クルと北の山と初キャンプ△」開催と同時に里で建設可能になります。

 

 

開催期間

4月25日 メンテナンス後 ～ 5月7日 13:59

 

 

イベント内容

本イベントでは、ストーリーが楽しめるシナリオクエストと、ふしぎな宝箱を開けることのできる素材が手に入る素材収集クエスト、高難易度である代わりに、豪華な報酬が手に入る【強敵】クエスト・【超強敵】クエスト・極クエストをプレイできます。

 

 

イベント新要素その１：マップ式クエスト！

マップ式クエストは、地図上に表示された「エリア」で発生するクエストをクリアしていくことで、ストーリーが進んでいくイベント形式です。

 

クエストマップ上に表示されている「エリアアイコン」をタップし、各エリアで発生するシナリオクエストを読み進めていきましょう。

 

エリアの中には一定期間経過後に開放されるエリアも存在します。

 

 

エリア開放スケジュール
・エリア「里」～「森」まで
4月25日 メンテナンス後 ～ 5月7日 13:59 

・エリア「渓流」以降
4月27日 17:00 ～ 5月7日 13:59

・エリア「苔むした洞窟」（【強敵】クエスト・【超強敵】クエスト）
4月29日 17:00 ～ 5月7日 13:59

・エリア「秘密の鍾乳洞」（【極】クエスト）
5月1日 17:00 ～ 5月7日 13:59

※エリア「苔むした洞窟」「秘密の鍾乳洞」が開放されるには、本イベントの全シナリオクエストをクリアしている必要があります。

 

 

各エリアの中にはシナリオクエストクリア時に、素材収集クエストが発生するものもあります。


素材収集クエストでは、ふしぎな宝箱を開けるために必要な宝箱開封用アイテムや、【強敵】クエストに挑戦するために必要な【強敵】北の山挑戦券が入手できます。

 

 

イベント新要素その２：ふしぎな宝箱！

素材収集クエストで手に入る宝箱開封用アイテムを集めると、ふしぎな宝箱を開封することができます！ふしぎな宝箱からは様々なアイテムが手に入ります。

ふしぎな宝箱画面へは、イベント時のクエスト画面の「宝箱アイコン」をタップするか、トレードショップの限定交換画面から移動することができます。

 


ふしぎな宝箱から手に入るアイテムの中には、更にトレードショップで別のアイテムに交換できる場合もあります。詳細は下記の「『ゆるキャン△』★4 志摩 リンを仲間にしよう！」を参照ください。

 

※ふしぎな宝箱は1～5箱目、10箱目に「あたりアイテム」が設定されており、「あたりアイテム」を手に入れると、宝箱のリセットを行うかどうか選択できるようになります。

 

※ふしぎな宝箱の6～9箱目、11箱目以降は「あたりアイテム」が存在せず、宝箱の全ての中身を空にするまでリセットを行うことができません。

 

※ふしぎな宝箱の各箱にて宝箱のリセットを行うと、次の宝箱に進んでしまうため、以前の宝箱で残っていた中身を引き直すことはできません。宝箱のリセットを行う際はご注意ください。

 

 

開封可能期間：
4月25日 メンテナンス後 ～ 5月14日 13:59

 

※ふしぎな宝箱の開封は、開封可能期間の間しか行うことができません。交換忘れにはご注意ください。

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターは、イベント「野クルと北の山と初キャンプ△」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

 

 

豪華素材がドロップする「【強敵】クエスト」「【超強敵】クエスト」も！

探索マップの中には、倒すことの困難なモンスターの登場する「【強敵】クエスト」「【超強敵】クエスト」「【極】クエスト」が発生します。

 

見事【超強敵】クエストをクリアすると、貴重なイベントアイテムと、まれに★4限界突破素材を入手できます。
パーティーを整えて、【強敵】クエスト・【超強敵】クエストに挑戦してみましょう。

 

※【強敵】クエスト・【超強敵】クエストの発生期間は4月29日 17:00 ～ 5月7日 13:59です。
※使用しない【強敵】北の山挑戦券・【超強敵】北の山挑戦券は、トレードショップ＞限定交換＞「野クルと北の山と初キャンプ△」から、宝箱開封用アイテム「松ぼっくり」に交換することができます。

 

【強敵】　北の山挑戦券：「松ぼっくり」×1個
【超強敵】北の山挑戦券：「松ぼっくり」×3個

 

【極】クエスト開催！

更に、【超強敵】クエストよりも難易度が高い「【極】クエスト」も発生！
「【極】クエスト」は、イベント開始後の5月1日 17:00から開催されますが、参加するためには全エリアのシナリオクエストを全てクリアしている必要があります。

 

【極】クエストに挑戦するためには、スタミナを消費する必要があります（「【強敵】クエスト」「【超強敵】クエスト」とは異なり、「【極】クエスト」には挑戦券は必要ありません）。

 

【極】クエストでは、特定の条件でクリアすると報酬を入手することができます。

 

【極】クエストを初めてクリアした時：星彩石×40個
【極】クエストにおいて、一人も撤退しないで初めてクリアした時：まほうつかいの果実×2個

 

※【極】クエストをクリアした際には、アイテムのドロップはございませんのでご注意ください。

※【極】クエストの発生期間は5月1日 17:00 ～ 5月7日 13:59です。
