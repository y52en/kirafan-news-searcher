【開催】イベント「エトワリアの歌姫」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

5月9日 メンテナンス後より、新イベント「エトワリアの歌姫」を開催いたします。

 

 

開催期間

5月9日 メンテナンス後 ～ 5月21日 13:59

 

 

イベント内容

本イベントでは、ストーリーが楽しめ、イベント期間中様々なアイテムと交換可能な素材が手に入るシナリオクエスト、高難易度である代わりに、豪華な報酬が手に入る【強敵】クエスト・【超強敵】クエスト・【極】クエストをプレイできます。

 

 

キキ先生描き下ろしのオリジナルキャラクターが登場！！

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターは、イベント「エトワリアの歌姫」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

 

 

 

豪華素材がドロップする「【強敵】クエスト」「【超強敵】クエスト」も！

倒すことの困難な強敵モンスターの登場する「【強敵】クエスト」「【超強敵】クエスト」がシナリオクエスト全クリアで開放されます。

 

【強敵】クエストに挑戦するためには、スタミナを消費する必要があります（「【超強敵】クエスト」とは異なり、「【強敵】クエスト」には挑戦券は必要ありません）。

 

【強敵】クエストをクリアすると、イベント素材と、まれに「【超強敵】エトワリアの歌姫挑戦券」を入手することができます。

 

 

この「【超強敵】エトワリアの歌姫挑戦券」を消費することで「【超強敵】クエスト」に挑戦することができ、見事【超強敵】クエストをクリアすると、イベント素材と、まれに★4限界突破素材を入手できます。

 

更に今回のイベントでは、【強敵】クエストと【超強敵】クエストの両クエストを、一人も撤退しないでクリアした場合、初回のみ「まほうの果実×1個」を入手することができます。

 

パーティーを整えて、【強敵】クエスト・【超強敵】クエストに挑戦してみましょう。

 

※【強敵】クエスト・【超強敵】クエストの発生期間は5月9日 メンテナンス後 ～ 5月21日 13:59です。
※使用しない【超強敵】エトワリアの歌姫挑戦券は、トレードショップ＞限定交換＞「エトワリアの歌姫」から、イベント素材「♯な旋律」に交換することができます。

 

【極】クエスト開催！

更に、【超強敵】クエストよりも難易度が高い「【極】クエスト」が登場！

「【極】クエスト」は、イベント開始後の5月15日 17:00から開催されますが、参加するためには各シーズンのシナリオクエストを全てクリアしている必要があります。

 

【極】クエストに挑戦するためには、スタミナを消費する必要があります（「【超強敵】クエスト」とは異なり、「【極】クエスト」には挑戦券は必要ありません）。

 

【極】クエストでは、特定の条件でクリアすると報酬を入手することができます。

 

【極】クエストを初めてクリアした時：星彩石×40個

【極】クエストにおいて、一人も撤退しないで初めてクリアした時：まほうつかいの果実×2個

 

※【極】クエストをクリアした際には、アイテムのドロップはございませんのでご注意ください。

※【極】クエストの発生期間は5月15日 17:00 ～ 5月21日 13:59です。
