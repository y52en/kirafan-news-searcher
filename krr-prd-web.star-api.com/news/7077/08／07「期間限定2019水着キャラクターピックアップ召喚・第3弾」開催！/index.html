「期間限定2019水着キャラクターピックアップ召喚・第3弾」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。


8月8日 17:00より、
「期間限定2019水着キャラクターピックアップ召喚・第3弾」

を開催いたします。

※「期間限定2019水着キャラクターピックアップ召喚・第3弾」では、
　期間限定水着キャラクターのうち、★5 唯【水着】、
　★5 チノ【水着】、★5 ランプ【水着】のみが出現、

　ピックアップ対象となります。

　第1弾・第2弾で出現する水着キャラクターは

　出現しませんのでご注意ください。

 

開催期間

8月8日 17:00 ～ 8月27日 11:59

 

ピックアップ対象キャラクター

★5 唯【水着】（そうりょ　水属性）

★5 チノ【水着】（ナイト　土属性）

★5 ランプ【水着】（まほうつかい　陽属性）

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認ください。

 

1日1回有償限定召喚

「期間限定2019水着キャラクターピックアップ召喚・第3弾」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。
※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、8月8日 17:00から開催のイベント
「パイレーツオブエトワリア」にてイベントボーナス効果を持っています。

イベントボーナスに関する詳細は、
イベント「パイレーツオブエトワリア」のイベント詳細をご確認ください。
