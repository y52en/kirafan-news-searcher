★5キャラクター専用ぶき更新！(2019年8月①)

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

8月8日 17:00より新たなキャラクターの
「★5キャラクター専用ぶき」を追加いたします。

 

その他の★5キャラクター専用ぶき追加済みキャラクターは、
【クエスト＞メモリアルクエスト＞各作品＞キャラクター】にて確認できます。

 

更新日時

8月8日 17:00より

 

追加キャラクター

今回「★5キャラクター専用ぶき」が追加されるキャラクターは、
以下2キャラクターです。

今回専用ぶきが追加された2キャラクターはぶき進化3回まで、
フルボイスのシナリオ「メモリアルクエスト」のうち1～3節が開放されます。

 

「自動発動スキル」紹介

★5キャラクター専用ぶきを3回進化させると「自動発動スキル」が解放されます。
「自動発動スキル」はバトル開始時から自動的に効果が発動します。
各★5キャラクター専用ぶきにて解放されるスキルは以下をご参照ください。

 

★5山口 如月専用ロッド

　スタン状態にならない +
　状態異常にならない

 

★5西川 葉子専用クリスタル

　通常攻撃が全体攻撃になる +

　「自身」のクラススキルによる
　ステータスアップ効果の持続ターンが長くなる

 

★5キャラクター専用ぶき入手＆強化方法

対象のキャラクターをLv.100にすることで、
各★5キャラクター専用のぶきが入手できます。

「★5キャラクター専用ぶき」は他のぶきと同様に「強化」が可能です。
「★5キャラクター専用ぶき」を「強化」していき、以下の条件を満たすと
「ぶき進化」ができるようになります。

① 「★5キャラクター専用ぶき」のレベルを上限まで上げる
② 対象のキャラクターに「★5キャラクター専用ぶき」を装備させ、
　バトルでぶきスキルを使用してぶきスキルを上限まで上げる
③ 「ぶき進化」に必要な「エトワリウム」などのアイテム、コインを集める

「ぶき進化」に必要なアイテムの1つ「エトワリウム」は、
主にイベントにてイベント素材を集めることで入手できます。
