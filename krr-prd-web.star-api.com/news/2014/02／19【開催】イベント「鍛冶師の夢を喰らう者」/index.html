【開催】イベント「鍛冶師の夢を喰らう者」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

2月19日 メンテナンス後より、「夢喰いメリー」参戦に合わせた新イベント「鍛冶師の夢を喰らう者」を開催いたします。

 

開催期間

2月19日 メンテナンス後 ～ 2月26日 13:59

 

イベント内容

本イベントでは、ストーリーが楽しめ、イベント期間中様々なアイテムと交換可能な素材が手に入るシナリオクエスト、高難易度である代わりに、豪華な報酬が手に入る強敵クエスト・超強敵クエストをプレイできます。


また今回のイベントでは、超強敵クエストよりも更に手強い極クエストも登場します！

 

遊び方

シナリオクエストを進めることで、新たなストーリーを読むことができます。また、クエストクリアで手に入る素材を集めることで、トレードショップで様々なアイテムと交換できます！

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。
イベントボーナスを持ったキャラクターは、イベント「鍛冶師の夢を喰らう者」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

豪華素材がドロップする「強敵クエスト」も！

倒すことの困難な強敵モンスターの登場する「強敵クエスト」「超強敵クエスト」が、シナリオクエスト全クリアで開放されます。

 

強敵クエストに挑戦するためには、スタミナを消費する必要があります（「超強敵クエスト」とは異なり、「強敵クエスト」には挑戦券は必要ありません）。

強敵クエストをクリアすると、イベント素材と、まれに「【超強敵】挑戦券」を入手することができます。

 

また、この「【超強敵】挑戦券」を消費することで「超強敵クエスト」に挑戦することができ、見事超強敵クエストをクリアすると、イベント素材と、まれに★4限界突破素材を入手できます。

 

また、今回のイベントでは、強敵クエストと超強敵クエストの両クエストを、一人も撤退しないでクリアした場合、初回のみ「せんしの果実×1個」を入手することができます。

 

パーティーを整えて、強敵クエスト・超強敵クエストに挑戦してみましょう。

 

※強敵クエスト・超強敵クエストの発生期間は2月19日 メンテナンス後 ～ 2月26日 13:59です。
※使用しない【超強敵】挑戦券は、トレードショップ＞限定交換＞「鍛冶師の夢を喰らう者」から、イベント素材「立派な大鎚」に交換することができます。

 

新クエスト：「極クエスト」登場！

超強敵クエストよりも更に難易度が高い「極クエスト」が登場！
「極クエスト」は、イベント開始後の2月23日 17:00から開催されます。

 

極クエストに挑戦するためには、スタミナを消費する必要があります（「超強敵クエスト」とは異なり、「極クエスト」には挑戦券は必要ありません）。

極クエストでは、特定の条件でクリアすると報酬を入手することができます。

 

極クエストを初めてクリアした時：星彩石×40個
極クエストにおいて、一人も撤退しないで初めてクリアした時：せんしの果実×2個

 

※極クエストをクリアした際には、アイテムのドロップはございませんのでご注意ください。

 

※極クエストの発生期間は2月23日 17:00 ～ 2月26日 13:59です。
