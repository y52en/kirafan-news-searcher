【更新】★5キャラクター専用ぶき更新！（2019年2月②）

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

2月27日 17:00より
以下キャラクターの、専用ぶき最終進化、メモリアルクエスト4節が開放されました。

 

開放日時

2月27日 17:00～

 

対象キャラクター

・★5 折部 やすな（ナイト）
・★5 メリー・ナイトメア（せんし）
・★5 高山 春香（そうりょ）

 

開放条件

対象キャラクターの専用ぶきを4回進化させる

 

最終進化で「メモリアルクエスト4節」追加＆ぶきスキル変化！

専用ぶき最終（4回）進化では「メモリアルクエスト4節」の開放と同時に
「ぶきスキル」が変化！
より強力な効果になると共に発動時の演出も特別なものに変化します。

 

・★5折部 やすな専用ランス
　★変化前のぶきスキル
　メモリア・ジャマー
　　「味方単体」の物理攻撃が一定ターン中アップ +
　　「味方単体」の狙われやすさが一定ターン中ダウン
　　
　★変化後のぶきスキル
　対ソーニャちゃん無敵スピア
　　「味方単体」の物理攻撃とクリティカル率が一定ターン大アップ +
　　「味方単体」の狙われやすさが一定ターン中ダウン

 

・★5メリー・ナイトメア専用ハンマー
　★変化前のぶきスキル
　メモリア・ストライク
　　「敵単体」に自属性の中ダメージ +
　　「自身」の物理攻撃を一定ターン中アップ 

　★変化後のぶきスキル
　夢と現の冒険譚
　　「敵単体」にねむり状態に特効を持つ月属性の特大ダメージ

 

・★5高山 春香専用ロッド
　★変化前のぶきスキル
　メモリア・ライズ
　　「味方単体」を中回復 +
　　「味方単体」の物理攻撃と魔法攻撃が一定ターン中アップ
　　
　★変化後のぶきスキル
　不思議な桜の、癒しの魔法
　　「味方全体」を中回復 ＋
　　「味方全体」の物理攻撃と魔法攻撃が一定ターン中アップ

 

注意事項

・機能の仕様や日程は、予告なく変更する可能性がございます。
・★5キャラクター専用ぶきは、★4及び★3の同じキャラクターに
　装備させることはできません。
・各キャラクターのフルボイスシナリオについては、
　クエスト＞メモリアルクエストから閲覧できます。
・他キャラクターのぶき最終進化、「メモリアルクエスト」の4節は
　順次リリース予定となりますので、今しばらくお待ちください。
