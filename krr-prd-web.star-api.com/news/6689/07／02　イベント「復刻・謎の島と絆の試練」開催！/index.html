　イベント「復刻・謎の島と絆の試練」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。
7月3日 17:00より、イベント「復刻・謎の島と絆の試練」を
開催いたします。

本イベントは2018年8～9月に開催したイベントの復刻開催となります。

 

開催期間

7月3日17:00 ～ 7月17日13:59

 

はるかなレシーブイベント復刻！

本イベントは、シナリオクエスト、復刻記念追加シナリオと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。
※本イベントのシナリオクエストは過去に開催した
　「謎の島と絆の試練」と同じ内容ですが、
　交換可能な報酬は一新されています。

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。
※過去に開催した「謎の島と絆の試練」で入手した
　名称/デザインが同一のイベント素材は使用できませんのでご注意ください。
 

イベントの報酬や遊び方の詳細は、
別途掲載のおしらせ「イベント『復刻・謎の島と絆の試練』遊び方」または

イベント画面＞イベント詳細をご確認ください。

以前イベントを遊んだことがある人も、追加シナリオを楽しもう！

4節追加：7月15日0:00 ～ 7月17日13:59

収集クエスト、【超強敵】クエストで素材を集めよう！

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになるほか、

全編フルボイスの「メモリアルクエスト」の続きが読めるようになります。

※各キャラクターの「メモリアルクエスト」は、
　クエスト＞メモリアルクエストから閲覧できます。

 

さらなる挑戦！【極】クエスト！

シナリオクエスト、【超強敵】クエストをクリアした方は難易度の高いクエスト、
【極】クエストに挑戦してみましょう！
初回クリアで星彩石×40個、クリアコンプリート報酬で、
イベント素材「絆のペンダント」×1個が入手できます。
