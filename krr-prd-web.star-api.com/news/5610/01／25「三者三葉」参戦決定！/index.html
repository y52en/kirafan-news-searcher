「三者三葉」参戦決定！

『きららファンタジア』に、「三者三葉」の新規参戦が決定しました！

「三者三葉」とは

「三者三葉」は、芳文社「まんがタイムきらら」にて連載されていた
荒井チェリー先生による4コマ漫画作品です。

元お金持ちお嬢様ながらも落ちぶれて、
昼食がパンの耳とマヨネーズになってしまった葉子。
無尽蔵の胃袋を持ち、大食い店を食べ歩く
元気いっぱいおてんば娘の双葉。
普段は温厚だけどその笑顔の裏で何を
考えているか分からない腹黒委員長の照。
そんな女子高生3人を中心に繰り広げられる
三者三様の楽しい日常譚！

コミックス完結の14巻は、1月25日発売です！
また、ゲームへの登場は2019年春を予定しております。
葉子たちのエトワリアでの活躍にもご期待ください！

参戦決定CMも公開中！
