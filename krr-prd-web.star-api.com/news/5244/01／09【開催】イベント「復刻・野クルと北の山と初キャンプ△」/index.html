【開催】イベント「復刻・野クルと北の山と初キャンプ△」

『きららファンタジア』をご利用いただき、誠にありがとうございます。
1月9日 13:30より、「復刻・野クルと北の山と初キャンプ△」を開催いたします。
本イベントは2018年4月に開催したイベントの復刻開催となります。

開催期間

1月9日 13:30 ～ 1月23日 13:59


ゆるキャン△イベント復刻！


本イベントは、シナリオクエスト・復刻記念追加シナリオと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。
※本イベントのシナリオクエストは過去に開催した
　「野クルと北の山と初キャンプ△」と同じ内容ですが、
　「ふしぎな宝箱」から入手可能な報酬は一新しております。


シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめ、
クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換可能です。


※過去に開催した「野クルと北の山と初キャンプ△」で入手した名称/デザインが同一のイベント素材では交換できませんのでご注意ください。
※本イベント素材につきまして、
　「松ぼっくり」「ツンツーン」のレアリティが誤って表示されておりました。
　（正しくは、前回開催イベントでのレアリティと同じとなります。）
　イベントボーナスやドロップ率などには影響はございません。
　（1月9日 17:00追記）

より多くのイベント素材や限界突破素材が入手できる【超強敵】クエストは、
「【超強敵】北の山挑戦券」を消費することで挑戦できます。
さらに高難易度な【極】クエストでは初回クリアで星彩石×40個が手に入ります。
 
イベントのあらすじや遊び方は、イベント画面＞イベント詳細をご確認ください。
