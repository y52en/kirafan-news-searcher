【おしらせ】アプリ容量増加について
『きららファンタジア』をご利用いただき誠にありがとうございます。

Appleガイドライン変更に伴う対応を、
次回ver1.14.0アップデートにて実施いたします。

上記対応に伴い、
App Storeからダウンロードする際のアプリ容量が150MB以上となっております。

ver1.14.0以降につきまして、iOS版『きららファンタジア』をご利用のお客様は
App Storeからのアプリver更新の際、Wi-Fi環境でのダウンロードが必要となっておりますので、予めご了承ください。
※iOS版以外の『きららファンタジア』をご利用のお客様は対象ではございません。

ver1.14.0アップデートは、
1月17日（木）に行うメンテナンスにてリリースとなり、
その際、強制アップデートを実施予定です。

※1月17日（木）のメンテナンス内容につきましては、
　後日掲載のメンテナンスのおしらせをご確認ください。

ご利用のお客様にはご迷惑おかけし、大変申し訳ございませんが、
ご理解のほど、何卒宜しくお願い致します。
