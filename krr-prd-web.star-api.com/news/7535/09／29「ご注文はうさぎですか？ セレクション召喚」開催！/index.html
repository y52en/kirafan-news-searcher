注意事項

・ピックアップ対象のキャラクターは同じレアリティの他のキャラクターよりも
　提供割合が高く設定されています。
・本召喚の提供割合は、
　MENU＞召喚＞本召喚のバナー＞提供割合 よりご確認ください。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・召喚を行った際に「星屑のかけら」を「1回召喚」で1個、
　「10回召喚」で10個入手できます。
・通常の「1回召喚」「10回召喚」では無償星彩石から使用されます。
・所持している星彩石の内訳は、召喚画面下部の表示または、
　MENU＞設定・その他＞購入履歴 よりご確認ください。
・「1日1回有償限定召喚」は、毎日0時に回数がリセットされます。
　「1日1回有償限定召喚」に対応している召喚が同時に複数開催されている場合は、
　各召喚につき1日1回までご利用いただけます。
・イベントボーナス効果に関する詳細は、
　別途掲載のおしらせ「イベント『はたらくチマメ隊』遊び方」または、
　MENU＞クエスト＞はたらくチマメ隊＞イベント詳細 よりご確認ください。
・内容は予告なく変更になる場合がございます。ご了承ください。
