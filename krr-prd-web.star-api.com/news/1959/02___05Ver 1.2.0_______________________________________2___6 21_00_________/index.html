Ver 1.2.0での改修予定箇所について（2/6 21:00更新）

『きららファンタジア』をご利用いただき、誠にありがとうございます。

2/6 11:00より予定しておりますメンテナンスおよびVer 1.2.0のアップデートにおいて、実装予定の各機能改善内容についてご案内いたします。

 

 

バトル

・編成コストの上限が全プレイヤー20増加されます。

・倍速機能を一括で変更可能なショートカットボタンが追加になります。

※ショートカットボタン追加に伴い、設定、オートボタンのデザインが変更になります。

 

ステータス

・キャラクターが進化している場合、進化前キャラクターのイラストとデフォルメが閲覧可能になります。

 

里

・一部のスケジュール表示名が、作品別ではなくキャラクター別になります。

・ぶき素材生産所タップ時、獲得したアイテムの生産アイコンが表示されないようになります（2月6日　21:00更新)

※今後のアップデートにて、獲得したアイテムが確認できるように対応予定です

 

メニュー

・編成機能内の「強化/限界突破/進化」が強化機能に移動になります。

 

皆様により一層『きららファンタジア』をお楽しみいただけますよう、スタッフ一同鋭意努めてまいります。

今後とも、『きららファンタジア』をよろしくお願い申し上げます。
