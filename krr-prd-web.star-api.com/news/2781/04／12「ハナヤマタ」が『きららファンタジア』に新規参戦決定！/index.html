「ハナヤマタ」が『きららファンタジア』に新規参戦決定！

 

『きららファンタジア』に、「ハナヤマタ」の新規参戦が決定しました！


■「ハナヤマタ」とは

「ハナヤマタ」は、浜弓場 双先生によるストーリー漫画。


容姿も普通、頭も普通、美術や体育……何だってフツウのぱっとしない14歳、関谷なるが、月夜の晩に出会った妖精のような異国の少女ハナに導かれ『よさこい』という非日常の世界へと踏み出していく、よさこい×中学生×青春のガールズストーリーです。


芳文社「まんがタイムきららフォワード」で先日グランドフィナーレを迎え、完結のコミックス10巻が4/12より絶賛発売中 。
