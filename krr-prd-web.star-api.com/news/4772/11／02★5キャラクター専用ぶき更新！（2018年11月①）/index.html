★5キャラクター専用ぶきを3回進化させると「自動発動スキル」が解放されます。
「自動発動スキル」はバトル開始時から自動的に効果が発動します。
自動発動スキルは以下をご参照下さい。

★5キャラクター専用ぶき入手＆強化方法

対象のキャラクターをLv.100にすることで、
各★5キャラクター専用のぶきが入手できます。

 

 

「★5キャラクター専用ぶき」は他のぶきと同様に「強化」が可能です。
ぶきを「強化」していき、特定の条件を満たすと
「ぶき進化」ができるようになります。

「ぶき進化」には、イベントなどで入手できるアイテム
「エトワリウム」が必要になります。

「エトワリウム」は、トレードショップ＞限定交換にて入手可能です。

 

各キャラクターのフルボイスシナリオについては、
クエスト＞メモリアルクエストから選択できます。

 

※他キャラクターのぶき最終進化、「メモリアルクエスト」の4節は
　順次リリース予定となりますので、今しばらくお待ち下さい。

※メモリアルクエスト開放には、特定の条件を達成する必要があります。
　条件達成を目指して、キャラクターやぶきを強化させましょう。

 

注意事項

・★5キャラクター専用ぶきは、★4、★3の同じキャラクターに
　装備させることはできません。
