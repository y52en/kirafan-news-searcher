【開催】イベント「エトワリア冒険譚 後編 目覚める鋼鉄の巨人」

『きららファンタジア』をご利用いただき、誠にありがとうございます。
2月27日 17:00より、イベント「エトワリア冒険譚 後編 目覚める鋼鉄の巨人」を
開催いたします。

 

本イベントは「エトワリア冒険譚」の後編となり、

現在開催中のイベント「エトワリア冒険譚 前編 リゼ教官の秘密特訓」から続くシナリオをお楽しみいただけます。

 

開催期間

2月27日 17:00 ～ 3月12日 13:59

イベント概要

本イベントは、シナリオクエストと収集クエスト、【超強敵】クエスト、
【極】クエスト、さらに【極】クエストをクリアすると挑戦可能となる
「超高難易度クエスト」で構成されています。

イベントの報酬や遊び方の詳細は、イベント画面＞イベント詳細をご確認ください。

 

シナリオクエストでシナリオを楽しもう！

シナリオクエストでは『きららファンタジア』オリジナルストーリーが楽しめます。


シナリオクエストはシーズンで区切られています。

シーズン2は2つのルートに分かれており好きなルートからシナリオを選択できます。
シーズン2のシナリオクエストをすべてクリアすると、次のシーズンのシナリオを読むことができます。

 

収集クエスト、【超強敵】クエストで素材を集めてアイテムと交換しよう！

クエストで集めた素材は様々なアイテムと交換できます。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになります。

専用の挑戦券を消費することで挑戦できる【超強敵】クエストでは
より多くのイベント素材や限界突破素材が入手できます。

 

さらなる挑戦！【極】クエスト！

【超強敵】クエストもクリアした方は難易度の高いクエスト、
【極】クエストに挑戦してみましょう！

初回クリアで星彩石×40個、クリアコンプリート報酬で
イベント報酬「褒章バッヂ」を入手できます。

 

初登場！超高難易度クエスト！

今回は新たに、「超高難易度クエスト」が登場！

【極】クエストをクリアして挑戦してみましょう！


初回クリアで星彩石×40個、クリアコンプリート報酬で

エトワリウムのかけら×2個を入手できます。

 

イベント限定素材が手に入る！！
