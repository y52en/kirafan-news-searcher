イベント「エトワリアのブラック・クリスマス」の遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「クリスマスドーナツ」や
「雪色の絵の具」が入手できます。


イベントで手に入る素材は★4 琉姫【クリスマス】（ナイト　風属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。

その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「エトワリアのブラック・クリスマス」からご確認ください。

また、イベント素材の交換期限は2019年1月2日 13:59までとなっておりますので
期限までに交換をお願いいたします。

 

イベント限定キャラクターを仲間にしよう！


イベント素材「クリスマスドーナツ」を100個集めると
★4 琉姫【クリスマス】（ナイト　風属性）を仲間にできます。

また、キャラクターを進化させることで
ステータスが強化されるだけでなく、
キャラクターのイラストも変化します。ぜひ進化させてみましょう。

キャラクターの進化は★4、★5のキャラクターが4回「限界突破」した後、
キャラクターのレベルが一定以上になると行えます。
限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。
イベント限定コールチケットは、トレードショップで★4 琉姫【クリスマス】（ナイト　風属性）に交換が可能です。

イベント限定コールチケットでの交換上限数は「10」となっておりますので、
たくさん交換して、★4 琉姫【クリスマス】（ナイト　風属性）を覚醒させてみましょう！

※★4 琉姫【クリスマス】（ナイト　風属性）の覚醒は、最大4回まで可能です。
※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。
※「イベント限定コールチケット」は、12月25日開催予定の
　クリスマスログインボーナスでも入手可能です。

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

イベント素材「クリスマスドーナツ」を集めると、
「セクシー雪だるま」と交換できます。


 「セクシー雪だるま」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

 

イベントボーナス効果について

今回のイベントでは、一部のキャラクターがイベントボーナス効果を持っています。
イベントボーナス効果を持ったキャラクターをパーティーに編成することで、
イベント「エトワリアのブラック・クリスマス」のクエストにおいて
イベント素材のドロップ数がアップします。

豪華素材がドロップする「【超強敵】クエスト」！

各種豪華素材がドロップする【超強敵】クエストも開催！



【超強敵】クエストに挑戦するためには、
「【超強敵】挑戦券 」が必要です。

開放期間：
12月14日 17:00 ～ 12月26日 13:59

 

※使用しない【超強敵】挑戦券は、
　トレードショップ＞限定交換＞「エトワリアのブラック・クリスマス」から
　「クリスマスドーナツ」に交換することができます。

 

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたという方は、
さらなる高難易度【極】クエストに挑戦してみましょう！


開放時刻以降にすべてのシナリオクエストをクリアしていると
【極】クエストに挑戦ができます。挑戦するためにはスタミナを消費します。

 

開放期間：
12月17日 17:00 ～ 12月26日 13:59
