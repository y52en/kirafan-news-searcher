　★5キャラクター専用ぶき更新！(2019年5月②)

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

5月30日 17:00より
以下キャラクターの、専用ぶき最終進化、メモリアルクエスト4節が開放されました。

 

開放日時

5月30日 17:00～

 

対象キャラクター

・★5 花小泉 杏（ナイト）
・★5 千矢【クリスマス】（そうりょ）
・★5 はじめ【クリスマス】（せんし）

 

開放条件

対象キャラクターの専用ぶきを4回進化させる

 

最終進化で「メモリアルクエスト4節」追加＆ぶきスキル変化！

専用ぶき最終（4回）進化では
「メモリアルクエスト4節」の開放と同時に「ぶきスキル」が変化！
より強力な効果になると共に発動時の演出も特別なものに変化します。

 

・★5花小泉 杏専用シールド
　★変化前のぶきスキル

　メモリア・エッジ

　　「敵単体」に自属性の中ダメージ +
　　「自身」の狙われやすさが一定ターン中アップ

　★変化後のぶきスキル

　ハピネス・コンティニュー

　　「自身」の狙われやすさが一定ターン大アップ +
　　「敵全体」に陽属性の超特大ダメージ +
　　「自身」のHPが減少する

 

・★5千矢【クリスマス】専用ロッド
　★変化前のぶきスキル

 　メモリア・ヒール

　　「味方単体」を中回復 +
　　「味方単体」の状態異常を回復

 

　★変化後のぶきスキル

　雪に微睡む迷路町

　　「味方単体」の狙われやすさが一定ターン大アップ +
　　状態異常を回復 + 状態異常を一定ターン無効化 +
　　リカバリーを付与

 

・★5はじめ【クリスマス】専用ソード
　★変化前のぶきスキル

　メモリア・ストライク

　　「敵単体」に自属性の中ダメージ +
　　「自身」の物理攻撃が一定ターン中アップ

　　　
　★変化後のぶきスキル

　聖夜のおもちゃ屋戦争

　　「敵単体」に水属性の特大ダメージ ＋
　　「味方全体」の物理攻撃が一定ターン中アップ

 

注意事項

・機能の仕様や日程は、予告なく変更する可能性がございます。
・★5キャラクター専用ぶきは、★4及び★3の同じキャラクターに
　装備させることはできません。
・各キャラクターのフルボイスシナリオについては、
　クエスト＞メモリアルクエストから閲覧できます。
・他キャラクターのぶき最終進化、「メモリアルクエスト」の4節は
　順次リリース予定となりますので、今しばらくお待ちください。
