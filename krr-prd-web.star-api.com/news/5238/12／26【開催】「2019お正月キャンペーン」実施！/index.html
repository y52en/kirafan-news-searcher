『きららファンタジア』をご利用いただき、誠にありがとうございます。

12月27日 16:00より「2019お正月キャンペーン」を実施いたします。

 

毎日10回召喚無料キャンペーン

キャンペーン期間中、
雑誌別召喚の「10回召喚」が1日1回無料でできるキャンペーンを実施します！


該当の召喚は毎日日替わりになります。

実施期間：2019年1月1日 0:00 ～ 2019年1月4日 23:59

 

キャンペーン対象の召喚と日程につきましては、以下となります。

※無料となるのは、1日1回のみとなります。
※1日1回の無料召喚終了後は、星彩石400個で「10回召喚」が可能となります。
※召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（10回召喚で10個入手できます）
※各召喚で出現するキャラクターの詳細につきましては、ゲーム内の各召喚画面＞出現一覧からご確認ください。

年末年始ログインボーナスキャンペーン

ログイン11日目には「イベント限定コールチケット」を入手することができます。

●ログインボーナス内容

1日目～10日目：無償星彩石20個
11日目：イベント限定コールチケット×1枚

 

※「イベント限定コールチケット」は、
　開催期間中のうち11日間ログインで入手可能です。

 

また、2019年1月1日には「お年玉ログインボーナス」として、
無償星彩石100個を入手することができます。

 

メインクエスト　ハードモード6章 追加！

メインクエストにハードモード6章を追加いたします。

 

追加日時：2018年12月27日 16:00～


ハード6章は、メインクエスト6章（ノーマル）最終節をクリアすると開放されます。

※ハードモードのシナリオ内容は、ノーマルモードと同じ内容となります。


また、このハードモードでは特定条件をクリアすると、各章で最大220個の星彩石を手に入れることができます。

・クエストクリア報酬：星彩石×5個
→ハードモード各クエストをクリアする

・オールクリア報酬：星彩石×20個
→ハードモード各章内の全クエストをクリアする（32節までクリア）

・オールコンプリートクリア報酬：星彩石×40個
→ハードモード各章内の全クエストをクリアランクコンプリート達成する
（コンプリートはパーティーメンバーが一人も戦闘不能にならずにクリアすることで獲得）

詳細はホーム画面＞クエスト画面＞メインクエストにてご確認ください。
※メインクエスト画面左下の章選択ボタンより、6章を選択後、画面左上の「ハード」ボタンをタップすることで移動できます。
