GETした素材をトレードショップで交換しよう！

イベント中に入手した「ピック」や「ティーカップ」は、トレードショップで報酬と交換できます。

 

「★5キャラクター専用ぶき」を進化させるためのアイテム「エトワリウム」もイベント素材と交換可能です。

「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。

※「エトワリウム」はイベント期間中、最大3つまで交換できます。

 

また、イベントアイテム「ティーカップ」を集めて交換できる「ネコミミメイド服」は、「コンテンツ限定限界突破素材」と交換できます。

※「ネコミミメイド服」はイベント期間中、最大5つまで交換、入手することができます。

 

「コンテンツ限定限界突破素材」は、対応する作品の★5キャラクターならクラスを問わずに使用することのできるアイテムです。期間中、以下の星結晶1個ずつと交換することができます。

 

・けいおん！の星結晶

・ステラのまほうの星結晶

・こみっくがーるずの星結晶

・あんハピ♪の星結晶

・ハナヤマタの星結晶

※各作品につき、最大4個まで交換可能です。

 

イベントボーナス効果について

今回のイベントでは、一部のキャラクターがイベントボーナス効果を持っています。

イベントボーナスを持ったキャラクターをパーティーに編成することで、イベント「けいおん！　エトワリアライブ！！」のクエストにおいて、イベント素材のドロップ数がアップします。

イベントボーナスが発生するキャラクターはイベント詳細をご確認下さい。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
