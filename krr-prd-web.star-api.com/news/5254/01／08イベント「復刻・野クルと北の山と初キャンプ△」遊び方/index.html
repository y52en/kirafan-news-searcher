宝箱開封用アイテムで「ふしぎな宝箱」を開封しよう！

素材収集クエストで手に入る宝箱開封用アイテムを集めると、「ふしぎな宝箱」を開封することができます！
「ふしぎな宝箱」からは様々なアイテムが手に入ります。

ふしぎな宝箱画面へは、イベント時のクエスト画面の「宝箱アイコン」をタップするか、トレードショップの限定交換画面から移動することができます。


※過去に開催した「野クルと北の山と初キャンプ△」で入手した 名称/デザインが同一のイベント素材では交換できませんのでご注意ください。
 ※本イベント素材につきまして、
　「松ぼっくり」「ツンツーン」のレアリティが誤って表示されておりました。
　（正しくは、前回開催イベントでのレアリティと同じとなります。）
　イベントボーナスやドロップ率などには影響はございません。
　（1月9日 17:00追記）

「復刻・野クルと北の山と初キャンプ△」からご確認ください。
 
また、イベント素材の交換期限は1月30日 13:59までとなっておりますので

豪華素材がドロップする「【超強敵】クエスト」！

各種豪華素材がドロップする【超強敵】クエストも開催！


【超強敵】クエストに挑戦するためには、
「【超強敵】北の山挑戦券 」が必要です。
※過去に開催した「野クルと北の山と初キャンプ△」で入手した
名称/デザインが同一の「【超強敵】北の山挑戦券」は使用できませんのでご注意ください。

開放期間：1月10日 17:00 ～ 1月23日 13:59

 

※使用しない「【超強敵】北の山挑戦券」は、
　トレードショップ＞限定交換＞「復刻・野クルと北の山と初キャンプ△」から
　「松ぼっくり」に交換することができます。

 

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたという方は、
さらなる高難易度【極】クエストに挑戦してみましょう！


開放時刻以降にすべてのシナリオクエストをクリアしていると
【極】クエストに挑戦ができます。挑戦するためにはスタミナを消費します。

 

開放期間：1月11日 17:00 ～ 1月23日 13:59
