フェンネル参戦記念ミニイベント「このフェンネルにお任せを！ 」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

3月20日 0:00より、
七賢者・フェンネル参戦記念ミニイベント「このフェンネルにお任せを！」を
開催いたします。

開催期間

3月20日 0:00 ～ 3月31日 13:59

内容

本イベントは、全4節からなる特別シナリオと七賢者チャレンジクエストで
構成されており、各シナリオを読むことで星彩石×5個がもらえます。

【シナリオ開放スケジュール】
・特別シナリオ1節：3月20日 0:00 ～ 3月31日 13:59
・特別シナリオ2節：3月21日 0:00 ～ 3月31日 13:59
・特別シナリオ3節：3月22日 0:00 ～ 3月31日 13:59
・特別シナリオ4節：3月23日 0:00 ～ 3月31日 13:59

七賢者チャレンジクエストに挑んで「チャレンジメダル」を集めよう！

七賢者チャレンジクエストには、強敵として七賢者・フェンネルが登場します。
挑戦するには、クエストキー　銀×3個が必要です。

クエストの初回クリア報酬とクリアランクコンプリート報酬でそれぞれ1個ずつ、
イベントミッションの達成報酬と合わせて、最大8個の「チャレンジメダル」が
手に入ります。

七賢者チャレンジクエストで入手できるチャレンジメダルは、
MENU＞トレード＞限定交換＞「このフェンネルにお任せを！」にて、
限界突破素材や「エトワリウムのかけら」と交換できます。
「エトワリウムのかけら」を4個集めると、
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

交換可能期間
3月20日 0:00 ～ 4月7日 13:59

注意事項

・ミニイベント「このフェンネルにお任せを！」開催期間中は、
　「3月強敵チャレンジクエスト」は開催されません。
・挑戦に必要なクエストキー　銀は
　ミッション報酬やイベント報酬で入手することができます。
・七賢者チャレンジクエストをクリアした際には、
　アイテムのドロップはございませんのでご注意ください。
・七賢者チャレンジクエストでの達成報酬は、
　通常の強敵チャレンジクエストには持ち越されず、リセットされます。
・チャレンジメダルやエトワリウムのかけらに交換期限はありません。
　七賢者チャレンジクエストの開催期間終了後は、
　MENU＞トレード＞限定交換＞強敵チャレンジクエスト にて
　アイテムと交換することができます。
・イベントの仕様や日程は、予告なく変更する可能性がございます。
