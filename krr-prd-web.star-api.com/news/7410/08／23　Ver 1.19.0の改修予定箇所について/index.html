　Ver 1.19.0の改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

8月27日実施予定のVer1.19.0アップデートにおいて
実装予定の機能改善内容をご案内いたします。

機能アップデート

・マンガが読める機能を実装
　シナリオクエストのシナリオパートでマンガが読める機能を追加
　それに伴う図書館画面の改修

・購入画面の拡張
　きららパスポートを「イベント」のタブに移動
　支援パスポートを「おすすめ」のタブに追加

・購入履歴画面の拡張
　購入済み支援パスポートの有効期限、再購入可能日を確認可能に

・新アイテム実装
　チャンスアップキーホルダー、無名の星結晶、
　スキルアップパウダー（専用ぶき）の追加と
　それに伴う召喚画面、強化画面、鍛冶屋画面の改修

・その他の細かな機能改修

※新たに追加される支援パスポートやアイテムの詳細については
　別途掲載のおしらせ「支援パスポート＆新アイテム登場！」をご覧ください。
