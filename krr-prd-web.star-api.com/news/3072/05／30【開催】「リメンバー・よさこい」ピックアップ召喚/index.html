【開催】「リメンバー・よさこい」ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

5月30日 17:00より、「リメンバー・よさこい」ピックアップ召喚を開催いたします。

 

召喚前半期間には、「ハナヤマタ」より★5 関谷 なる（CV.上田麗奈）、★4 ハナ・Ｎ・フォンテーンスタンド（CV.田中美海）、★4 笹目 ヤヤ（CV.奥野香耶）が『きららファンタジア』に初登場し、出現率がアップ！

 

また召喚後半期間には、「ゆゆ式」より★5 松本 頼子（CV.堀江由衣）、「きんいろモザイク」より★4 久世橋 朱里（CV.大西沙織）の出現率がアップします！

 

 

 

開催期間

全体開催期間：5月30日 17:00 ～ 6月13日 13:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意ください。

 

・ピックアップ召喚前半：5月30日 17:00 ～ 6月13日 13:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

 

ピックアップ対象：
★5　関谷 なる（そうりょ　陽属性）
★4　ハナ・Ｎ・フォンテーンスタンド（まほうつかい　月属性）
★4　笹目 ヤヤ（ナイト　水属性）


通常確率での出現：
★5　松本 頼子　（アルケミスト　陽属性）
★4　久世橋 朱里（ナイト　風属性）

 

 

・ピックアップ召喚後半：6月6日 0:00 ～ 6月13日 13:59

 

ピックアップ対象：
★5　松本 頼子　（アルケミスト　陽属性）

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能になります

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「リメンバー・よさこい」ピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、5月30日 17:00から開催されるイベント「リメンバー・よさこい」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「リメンバー・よさこい」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「リメンバー・よさこい」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

※★4 関谷 なる（そうりょ　土属性）は、イベント「リメンバー・よさこい」の期間中のみ仲間にできるキャラクターです。イベント中の宝箱の中からイベント限定コールチケットを手に入れることで、交換可能になります。
