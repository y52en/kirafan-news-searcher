芳文社公式マンガアプリ『COMIC FUZ』創刊！

芳文社初の公式マンガアプリ『COMIC FUZ（コミックファズ）』が、
ついに配信開始しました！

インストールはこちらから！

『COMIC FUZ』とは

『COMIC FUZ』は、芳文社が発行しているマンガ雑誌や
連載されている人気マンガを楽しめるアプリです。
基本無料で読める作品は約1,700話以上！

『COMIC FUZ』内で30分に１回配布されるメダルを使用することで、
毎日更新される連載コーナーなどにて、好きなマンガを無料で読むことができます。
さらに、リリースにあわせて、事前登録5万件突破を記念し、
『COMIC FUZ』内で使えるボーナスメダル1,000枚をプレゼントいたします。

きらら作品を『COMIC FUZ』で楽しもう！

創刊にあわせて、『きららファンタジア』にも参戦中の人気マンガ
「ゆるキャン△」が、『まんがタイムきららフォワード』から
『COMIC FUZ』に連載の場を移します。

それ以外にも、「ご注文はうさぎですか？」「NEW GAME!」、
そして参戦したばかりの「三者三葉」など
多数の『きららファンタジア』参戦作品が連載コーナーで配信されています。

そして、2019年夏には『きららファンタジア』メインクエストのコミカライズが『COMIC FUZ』にて連載開始予定です。

ぜひ『きららファンタジア』と合わせて『COMIC FUZ』をお楽しみください！
