【開催】「エトワリアの歌姫」ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

5月9日 メンテナンス後より、「エトワリアの歌姫」ピックアップ召喚を開催いたします。

 

召喚前半期間には、「うらら迷路帖」より★5 巽 紺（CV.本渡楓）が登場！
更に、★4 千矢（CV.原田彩楓）、★4 雪見 小梅（CV.久保ユリカ）、★4 棗 ノノ（CV.佳村はるか）も出現率がアップします。


また召喚後半期間には、「ステラのまほう」より★5 藤川 歌夜（CV.悠木碧）が登場！
その他、★4 本田 珠輝（CV.長縄まりあ）、★4 村上 椎奈（CV.村川梨衣）、★4 関 あやめ（CV.小澤亜李）も出現率がアップします。

全体開催期間：5月9日 メンテナンス後 ～ 5月23日 13:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意ください。

 

・ピックアップ召喚前半：5月9日 メンテナンス後 ～ 5月23日 13:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。
※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「エトワリアの歌姫」ピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、5月9日 メンテナンス後から開催されるイベント「エトワリアの歌姫」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「エトワリアの歌姫」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「エトワリアの歌姫」のおしらせをご確認下さい。


 

※★4 百武 照（まほうつかい　月属性）は、イベント「エトワリアの歌姫」の期間中のみ仲間にできるキャラクターです。イベント中に手に入る素材を集めることで、交換可能になります。


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
