【開催】「名探偵コロネット ユーの悲劇ピックアップ召喚」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

9月12日 17:00より、「名探偵コロネット ユーの悲劇ピックアップ召喚」を開催いたします。

 

開催期間

開催期間：9月12日 17:00 ～ 9月27日 11:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意下さい。

 

■ピックアップ召喚前半：9月12日 17:00 ～ 9月27日 11:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

 

＜ピックアップ対象＞
★5　千石 冠　　　（アルケミスト 陽属性）
★4　千石 冠　　　（アルケミスト 水属性）
★4　十倉 栄依子　（まほうつかい 風属性）
★4　百地 たまて　（せんし 陽属性）

※上記以外のキャラクターは通常確率で出現いたします。

 

 

■ピックアップ召喚後半：9月17日 0:00 ～ 9月27日 11:59

＜ピックアップ対象＞
★5　ユー子　　　（まほうつかい 土属性）
★4　ユー子　　　（まほうつかい 土属性）
★4　星川 ほたる　（まほうつかい 土属性）

※上記以外のキャラクターは通常確率で出現いたします。

 

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を
　追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

1日1回有償限定召喚

「名探偵コロネット ユーの悲劇ピックアップ召喚」開催期間中、
　1日1回のみ、有償星彩石10個で1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、9月12日 17:00から開催されるイベント「名探偵コロネット ユーの悲劇」にてイベントボーナス効果を持っています。

イベントボーナス効果を持つキャラクターを編成すると、イベント「名探偵コロネット ユーの悲劇」のクエストにおいて、イベント素材のドロップ数がアップします。
イベントボーナスに関する詳細は、イベント「名探偵コロネット ユーの悲劇」の遊び方をご確認下さい。
