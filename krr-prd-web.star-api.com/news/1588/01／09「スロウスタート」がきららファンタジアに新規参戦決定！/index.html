「スロウスタート」がきららファンタジアに新規参戦決定！

『きららファンタジア』に、「スロウスタート」の新規参戦が決定！

 

「スロウスタート」とは

「スロウスタート」は、芳文社「まんがタイムきらら」にて連載中の篤見唯子氏による4コマ漫画。

 

中学3年生の一之瀬花名が高校入試直前に病気にかかり、中学浪人になってしまった1年後、自分を知る者が誰もいない高校に進学し、浪人していたことを秘密にしたまま高校生活をスタートさせたその先の物語が描かれています。


「スロウスタート」は1月6日深夜よりTVアニメの放送がスタートしています。是非ご覧ください！
