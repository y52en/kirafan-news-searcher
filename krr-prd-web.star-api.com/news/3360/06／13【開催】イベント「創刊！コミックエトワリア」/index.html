イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターは、イベント「創刊！コミックエトワリア」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

 

 

豪華素材がドロップする「【超強敵】クエスト」「【極】クエスト」も！

さらに、豪華報酬が手に入る【超強敵】クエストや【極】クエストをプレイできます。

【超強敵】クエストをクリアすると、貴重なイベントアイテムと、★4限界突破素材もドロップすることがあります。【超強敵】クエストに挑戦するためには、「【超強敵】連載会議挑戦券 」が必要です。

※【超強敵】クエストの発生期間は6月15日 17:00 ～ 6月25日 13:59です。

 

「【極】クエスト」は、イベント開始後の6月18日 17:00から開催されますが、挑戦するためにはシナリオクエストを全てクリアしている必要があります。

【極】クエストに挑戦するためには、スタミナを消費する必要があります（「【超強敵】クエスト」とは異なり、「【極】クエスト」には挑戦券は必要ありません）。

【極】クエストでは、特定の条件でクリアすると報酬を入手することができます。

 

【極】クエストを初めてクリアした時：星彩石×40個
【極】クエストにおいて、一人も撤退しないで初めてクリアした時：そうりょの果実×2個

※【極】クエストをクリアした際には、アイテムのドロップはございませんのでご注意ください。

※【極】クエストの発生期間は6月18日 17:00 ～ 6月25日 13:59です。


※使用しない【超強敵】連載会議挑戦券 は、トレードショップ＞限定交換＞「創刊！コミックエトワリア」から、アイテム交換素材「羽根ペン」に交換することができます。

 

 

【特別読切】クエスト登場！

「【特別読切】クエスト」がシナリオクエスト全クリアで開放されます。

【特別読切】クエストに挑戦するためには、スタミナを消費します（「【超強敵】クエスト」とは異なり、挑戦券は必要ありません）。

※【特別読切】クエストの発生期間は6月21日 17:00 ～ 6月25日 13:59です。
