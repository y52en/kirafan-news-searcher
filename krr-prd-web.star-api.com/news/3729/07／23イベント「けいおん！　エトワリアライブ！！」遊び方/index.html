シナリオクエストでストーリーを楽しもう！

 

シナリオクエストをクリアするとストーリーを読むことができます。

 

シナリオクエストはシーズンで区切られており、前シーズンのすべてのシナリオクエストをクリアしていると、指定の日時以降に次シーズンのシナリオクエストがプレイできるようになります。

また、クエストで手に入るイベント専用アイテム「ティーカップ」「ピック」を集めるとトレードショップにてイベント限定キャラクターや限界突破素材など様々な報酬と交換できます。

 

 

GETした素材をトレードショップで交換しよう！

イベントで入手した「ピック」や「ティーカップ」は、トレードショップで報酬と交換できます。

 

「★5キャラクター専用ぶき」を進化させるためのアイテム「エトワリウム」もイベント素材と交換できます。

「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。

※「エトワリウム」はイベント期間中、最大3つまで交換できます。

 

また、イベントアイテム「ティーカップ」を集めて交換できる「ネコミミメイド服」は、「コンテンツ限定限界突破素材」と交換できます。

※「ネコミミメイド服」はイベント期間中、最大5つまで交換することができます。

 

「コンテンツ限定限界突破素材」は、対応する作品の★5キャラクターならクラスを問わずに使用することのできるアイテムです！

期間中、以下の星結晶と各1個ずつ交換をすることができます。

・ハナヤマタの星結晶

※各作品につき、最大4個まで交換可能です。

 

イベントボーナス効果について

今回のイベントでは、イベントボーナス効果を持ったキャラクターが存在します。

 

イベントボーナスを持ったキャラクターをパーティーに編成することで、イベント「けいおん！　エトワリアライブ！！」のクエストにおいて、イベント素材のドロップ数がアップします。

 

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
