09/12
ソルト参戦記念ミニイベント「ソルト最大の変身」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

七賢者・ソルトの参戦を記念して、
9月14日 0:00より、

七賢者・ソルト参戦記念ミニイベント「ソルト最大の変身」を
開催いたします。

 

開催期間

9月14日 0:00 ～ 9月19日 23:59

 

内容

期間中、ソルトやシュガー、アルシーヴが登場する
全4節からなる特別シナリオ「ソルト最大の変身」を読むことができます。

 

【シナリオ開放スケジュール】

・特別シナリオ1節：9月14日 0:00 ～ 9月19日 23:59

・特別シナリオ2節：9月15日 0:00 ～ 9月19日 23:59

・特別シナリオ3節：9月16日 0:00 ～ 9月19日 23:59

・特別シナリオ4節：9月17日 0:00 ～ 9月19日 23:59

※「ソルト最大の変身」の各シナリオを読むことで、

　星彩石×5個を獲得できます。

 

 七賢者チャレンジクエスト開催！

七賢者チャレンジクエストでは、強敵として
七賢者・ソルトが登場します！

七賢者チャレンジクエストは、
MENU＞クエスト＞「ソルト最大の変身」＞七賢者チャレンジクエスト

から挑戦可能です。
挑戦する際には、クエストキー　銀×3個を消費する必要があります。

 

アイテム「チャレンジメダル」を集めよう！

七賢者チャレンジクエストをクリアすると、報酬を入手できます。

初回クリア報酬：チャレンジメダル×1個

クリアランクコンプリート報酬：チャレンジメダル×1個

 

※クリア報酬のほかにミッション達成でもチャレンジメダルを入手できます。
　獲得可能なチャレンジメダルは、クリア報酬として2個、

　ミッション報酬として6個となり、本ミニイベント開催期間中で

　合計最大8個のチャレンジメダルを入手できます。

七賢者チャレンジクエストで入手することのできるチャレンジメダルは、
MENU＞トレード＞限定交換＞「ソルト最大の変身」 にて、
限界突破素材や「エトワリウムのかけら」と交換できます。

 

また、「エトワリウムのかけら」を4つ集めると、
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

 

■交換可能期間

　9月14日 0:00 ～ 9月26日 23:59

 

注意事項

・ミニイベント「ソルト最大の変身」開催期間中は、
　「9月強敵チャレンジクエスト」は開催されません。
　ミニイベント開催終了後の「9月強敵チャレンジクエスト」では、引き続き
　七賢者・ソルトが登場します。
　9月強敵チャレンジクエストにつきましては、

　公開中のおしらせをご確認ください。

・クエストキー　銀はミッション報酬やイベント報酬で入手することができます。

・七賢者チャレンジクエストをクリアした際には、
　アイテムのドロップはございませんのでご注意ください。

・七賢者チャレンジクエストでの達成報酬は、
　通常の強敵チャレンジクエストには持ち越されず、リセットされます。

・チャレンジメダルやエトワリウムのかけらに交換期限はありません。

　七賢者チャレンジクエストの開催期間終了後は、
　MENU＞トレード＞限定交換＞強敵チャレンジクエスト にて
　アイテムと交換することができます。

・イベントの仕様や日程は、予告なく変更する可能性がございます。
