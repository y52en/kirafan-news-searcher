【開催】「ゆゆ式」セレクション召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。


3月2日 メンテナンス後より、「ゆゆ式」のキャラクターがピックアップされた「セレクション召喚」を開催いたします。

 

 

 

開催期間

3月2日 メンテナンス後 ～3月9日 13:59

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・本召喚のピックアップ対象キャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

ピックアップ対象キャラクターの中には、イベントボーナス効果持ちも！

今回のピックアップ対象キャラクターの★5　野々原 ゆずこ（せんし　炎属性）・★5　櫟井 唯（ナイト　風属性）は、3月2日 メンテナンス後から開催されるイベント「風雲！ヒナ祭り」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「風雲！ヒナ祭り」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「風雲！ヒナ祭り」のおしらせをご確認下さい。


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
