　9月のトレードショップ更新について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

トレードショップの通常交換につきまして、
以下日程にて交換限度数のリセットを実施いたします。

 

トレードショップ通常交換　更新日時

9月1日 0:00

※今後も、毎月月初に通常交換の交換限度数のリセットを予定しております。

 

更新内容：交換限度数のリセットを行います

通常交換内の「お得な召喚チケットセット」と、
「お得なクエスト挑戦セット」につきましては、
8月31日23:59に交換限度数をリセットいたします。

 

その他交換限度数に達していたアイテムにつきましても、
8月31日23:59に交換限度数をリセットし、
9月1日0:00より再度交換ができるようになります。

※特殊召喚タブ内のキャラクター、ぶき&素材タブ内のぶきにつきましては、
　交換限度数はリセットされません。ご注意ください。

 

また、期間限定で増加していた一部アイテムの交換限度数が

9月1日0:00より通常の交換限度数へ戻ります。

対象のアイテムにつきましては、以下のおしらせ内

「いつもの2倍！通常交換一部アイテムの交換限度数が増加！」をご確認ください。
夏を満喫！エトワリアの夏休みキャンペーン・前半 開催！

 

今後とも、『きららファンタジア』をよろしくお願いいたします。
