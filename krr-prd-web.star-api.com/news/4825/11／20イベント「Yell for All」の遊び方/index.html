イベント「Yell for All」の遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「ポンポン」や「髪留め」が入手できます。


イベントで手に入る素材は★4 鳩谷 こはね（まほうつかい　陽属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。

その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「Yell for All」からご確認下さい。

また、イベント素材の交換期限は12月11日 13:59までとなっておりますので
期限までに交換をお願いいたします。



イベント限定キャラクターを仲間にしよう！


イベント素材「ポンポン」を100個集めると
★4 鳩谷 こはね（まほうつかい　陽属性）を仲間にできます。

また、キャラクターを進化させることで
ステータスが強化されるだけでなく、
キャラクターのイラストも変化します。ぜひ進化させてみましょう。

キャラクターの進化は★4、★5のキャラクターが4回「限界突破」した後、
キャラクターのレベルが一定以上になると行えます。
限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。
