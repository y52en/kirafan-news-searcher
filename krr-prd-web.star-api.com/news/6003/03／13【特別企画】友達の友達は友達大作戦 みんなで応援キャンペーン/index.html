【特別企画】友達の友達は友達大作戦 みんなで応援キャンペーン


イベント「復刻・友達の友達は友達大作戦」の開催を記念して、
イベント連動の特別WEBキャンペーンを実施いたします。

友達作りを頑張る花名をクレアと一緒に応援しましょう！

キャンペーン概要

イベント「復刻・友達の友達は友達大作戦」における
全ユーザーの「絵馬の根付」ドロップ数と連動して、
キャンペーンページ上でポイントが貯まります。
ポイントを貯めると、花名や栄依子たちからお礼のプレゼントを貰うことができます。
みんなで協力して、たくさんのプレゼントをゲットしましょう！

キャンペーン期間：
3月13日 17:00 ～ 3月27日 13:59

参加方法：
キャンペーン期間中にイベント「復刻・友達の友達は友達大作戦」を
プレイすることで、自動的に本キャンペーンへ参加できます。
