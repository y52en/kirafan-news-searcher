【開催】メインクエスト日替わりピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

5月23日 17:00 17:30より、メインクエストに登場するキャラクターが日替わりでピックアップされた「メインクエスト日替わりピックアップ召喚」を開催いたします。

 

特定のキャラクターを入手する機会ですので、スケジュールをご確認の上、メインクエスト日替わりピックアップ召喚をご利用ください。

※メンテナンス延長のため、開始時刻を変更しております

 

開催期間

全体開催期間　　：5月23日 17:00  17:30 ~ 5月30日 11:59

各召喚の開催時間：各召喚開催日 0:00～ 23:59

 

※初日の開始時刻は5月23日 17:00～　5月23日 17:30～となります。

※最終日の終了時刻は5月30日 11:59となります。

※メンテナンス延長のため、開始時刻を変更しております

 

 

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・各日替わり召喚の初回は、星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

メインクエスト日替わりピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。
