【開催】「友達の友達は友達大作戦2019」ピックアップ召喚 開催！


『きららファンタジア』をご利用いただき、誠にありがとうございます。


3月13日 17:00より、「友達の友達は友達大作戦2019ピックアップ召喚」を

開催いたします。

 

開催期間

3月13日 17:00 ～ 3月28日 13:59

 

ピックアップ対象キャラクター

ピックアップ召喚は、前半と後半に分かれています。
前半と後半でピックアップ対象が異なりますのでご注意ください。

■ピックアップ召喚前半：3月13日 17:00 ～ 3月28日 13:59
＜ピックアップ対象＞
★5 万年 大会（ナイト　水属性）

★4 千石 冠（アルケミスト　水属性）

★4 十倉 栄依子（まほうつかい　風属性）

★4 百地 たまて（せんし　陽属性）

■ピックアップ召喚後半：3月18日 0:00 ～ 3月28日 13:59
＜ピックアップ対象＞
★5 久米川 牡丹（そうりょ　月属性）

★4 鬼頭 紀美子（そうりょ　炎属性）

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認ください。

 

 

1日1回有償限定召喚

「友達の友達は友達大作戦2019ピックアップ召喚」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、3月13日 17:00から開催のイベント
「復刻・友達の友達は友達大作戦」にてイベントボーナス効果を持っています。

イベントボーナスに関する詳細は、
イベント「復刻・友達の友達は友達大作戦」のイベント詳細をご確認ください。
