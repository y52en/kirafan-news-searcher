イベントで素材をGETしてトレードショップで交換しよう！


イベントでは、イベント素材「園芸スコップ」や「水やりジョウロ」が入手できます。
素材はトレードショップで報酬と交換できます。
 
「★5キャラクター専用ぶき」を進化させるためのアイテム
「エトワリウム」もイベント素材と交換可能です。
「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。
 

イベントで手に入る素材を集めて、★4 雲雀丘 瑠璃（まほうつかい　月属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムを入手しましょう。
 
その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「ヒバリとりーさんと幸福の花」からご確認下さい。
 
また、イベント素材の交換期限は11月12日 13:59までとなっておりますので
交換期限までに交換をお願いいたします。

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

イベント素材「水やりジョウロ」を集めると、
「幸福の種」と交換できます。

 「幸福の種」は、対応する作品の★5キャラクターなら
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

イベント「ヒバリとりーさんと幸福の花」のクエストにおいて
イベント素材のドロップ数がアップします。

豪華素材がドロップする「【超強敵】クエスト」！

各種豪華素材がドロップする【超強敵】クエストも開催！


【超強敵】クエストに挑戦するためには、
「【超強敵】挑戦券 」が必要です。

開放期間：
10月27日 17:00 ～ 11月5日 13:59

 

※使用しない【超強敵】挑戦券は、
　トレードショップ＞限定交換＞「ヒバリとりーさんと幸福の花」から
　「水やりジョウロ」に交換することができます。

 

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたという方は、
さらなる高難易度【極】クエストに挑戦してみましょう！


開放時刻以降にすべてのシナリオクエストをクリアしていると
【極】クエストに挑戦ができます。挑戦するためにはスタミナを消費します。

 

開放期間：
10月31日 17:00 ～ 11月5日 13:59
