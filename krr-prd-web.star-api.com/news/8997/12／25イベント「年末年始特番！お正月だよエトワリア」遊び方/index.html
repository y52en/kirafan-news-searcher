イベント「年末年始特番！お正月だよエトワリア」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「年越しそばセット」や
「カンペ」が入手できます。
イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。
また、2020年1月1日に交換できるアイテムが追加されます。
エトワリウムや召喚チケットの他、各クラスの星結晶との交換が可能です。

その他のイベント報酬については、
MENU＞トレード＞限定交換＞「年末年始特番！お正月だよエトワリア」から
ご確認ください。

限定交換期間
・2019年12月26日 17:00 ～ 2020年1月21日 13:59
・2020年1月1日 0:00 ～ 2020年1月21日 13:59　※第2弾追加

※交換には期限がございます。ご注意ください。

イベント素材「カンペ」を100個集めると
★4 大会【お正月】（ナイト　土属性）を仲間にできます。

また、キャラクターを進化させることで
キャラクターのイラストも変化します。
ぜひ進化させてみましょう。

キャラクターの進化は★4、★5のキャラクターが4回「限界突破」した後、
キャラクターのレベルが一定以上になると行えます。
限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。


イベント限定コールチケットは、トレードショップで
★4 大会【お正月】（ナイト　土属性）に交換が可能です。

たくさん交換して、

★4 大会【お正月】（ナイト　土属性）を覚醒させてみましょう！

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。
※★4 大会【お正月】（ナイト　土属性）の覚醒は、
　最大4回まで可能となっており、
　同一キャラクターを10回重複させることで最大覚醒となります。
※「イベント限定コールチケット」の交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

「水晶テレビ玉」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。
今回は『きららファンタジア』登場作品すべての「コンテンツ限定限界突破素材」
が交換対象になります。
