新たなる曜日クエスト「修練場」とは？

各曜日クエストに新たに出現する「修練場」では、
使用回数に応じてレベルが上がるスキルの育成に向いた特別なエネミーが出現！
また、ぶき強化時に使用するとぶき経験値がたくさんもらえる
「シンボル」や「クレスト」がドロップします。

「修練場」を活用してキャラクターやぶきを育成しましょう！


ワンポイントアドバイス

・「修練場」のクリアには一定以上の戦力が必要です。
　キャラクターレベル平均80以上を目安にパーティーを編成することをおすすめします。
・修練場は適正クラス以外でのクリアが難しい設定になっています。
　修練場へはクエスト名にあわせたクラスでパーティーを編成して挑戦してみましょう。
　例：せんしの修練場はクラス「せんし」で挑戦するとクリアしやすい
