Ver1.7.0での改修予定箇所について
『きららファンタジア』をご利用いただき、誠にありがとうございます。

5/30 12:00より予定しておりますメンテナンスおよびVer1.7.0アップデートにおいて、実装予定の各機能改善内容についてご案内いたします。

〇機能アップデート
・きららパスポートの追加
・新キャラクターがピックアップされている時、ホームの「召喚の館」の上にアイコンを表示
・クエスト選択時のパーティー編成画面で「イベントボーナス」所持キャラクターのみを絞り込める機能を追加
・MENUに「トレーニング」の項目を追加
・ホーム画面でイベントロゴをタップするとイベントクエスト画面へ遷移する機能を追加
・クエスト選択画面のUI調整
・「星彩石を購入する」メニューのUI調整
・「星彩石を購入する」メニューに新商品が追加されている時、通知表示
・トレードショップの「交換期限」に「期限なし」を追加
・トレードショップの「限度数」に「制限なし」を追加
・トレードショップの「通常交換」の期間限定アイテムに期間限定ラベルを表示
・メンテナンス中のポップアップに「アプリ終了」ボタンを追加

〇不具合対応
・スタン中のキャラクターにリカバリーが付与されると表示が崩れる不具合を修正
・おでかけ中のキャラクタータップ時に得られるなかよし度でキャラシナリオが開放された時、キャラシナリオへ遷移できない不具合を修正
・コンテンツ施設リストの並び順を正しい並びに修正
・キャラクター一覧の「登場作品」ソート使用時の並びを正しい並びに修正
