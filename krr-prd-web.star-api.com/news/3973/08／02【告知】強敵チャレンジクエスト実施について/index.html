【告知】強敵チャレンジクエスト実施について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

8月4日 0:00より、「強敵チャレンジクエスト」を実施いたします。「強敵チャレンジクエスト」は、毎週末（土曜・日曜）に発生する高難易度のクエストです。

 

開催期間

8月4日 0:00 ～ 8月5日 23:59

8月11日 0:00 ～ 8月12日 23:59

8月18日 0:00 ～ 8月19日 23:59

8月25日 0:00 ～ 8月26日 23:59

 

 

内容

今月の強敵は、「けいおん！　エトワリアライブ！！」で登場した「ファル・エア・ロール」の3人です！

 

※強敵チャレンジクエストで登場する敵は、イベント「けいおん！　エトワリアライブ！！」に登場した際の強さとは異なります。

 

強敵チャレンジクエストは「ホーム＞クエスト＞強敵チャレンジクエスト」から挑戦可能です。 強敵チャレンジクエストに挑戦する際には、クエストキー　銀×3個を消費する必要があります。

 

※クエストキー　銀はミッション報酬やイベント報酬で入手することができます。

 

 

強敵チャレンジクエストをクリアすると報酬を入手することができます。

 

強敵チャレンジクエストを初めてクリアした時：チャレンジメダル×1個

強敵チャレンジクエストで、一人も撤退しないでクリアした時：チャレンジメダル×1個

 

※強敵チャレンジクエストをクリアした際には、アイテムのドロップはございませんのでご注意下さい。

 

※強敵チャレンジクエストでの達成報酬は翌週には持ち越されず、毎週リセットされます。（前週に報酬を入手していない場合、翌週クリア時に、前週と翌週の達成報酬は二重で獲得はできません）

 

 

アイテム「チャレンジメダル」が「エトワリウムのかけら」と交換可能に！

強敵チャレンジクエストで入手することのできるチャレンジメダルは、「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて、様々なアイテムと交換することができます。

 

今回から「エトワリウムのかけら」が追加されました！

 

「エトワリウムのかけら」は、★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

 

「チャレンジメダル」で交換できるアイテム

・エトワリウムのかけら

・せんしの星結晶

・ナイトの星結晶

・まほうつかいの星結晶

・アルケミストの星結晶

・そうりょの星結晶

 

「エトワリウムのかけら」で交換できるアイテム

・エトワリウム

 

交換可能期間：

8月4日 0:00 ～ 8月31日 23:59

 

※チャレンジメダルやエトワリウムのかけらは、8月の強敵チャレンジクエストの期間終了後も所有することができ、次回開催時の「トレードショップ＞限定交換＞強敵チャレンジクエスト」にて再度交換を行うことができます。
