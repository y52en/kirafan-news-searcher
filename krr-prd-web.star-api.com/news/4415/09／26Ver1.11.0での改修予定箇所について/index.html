Ver1.11.0での改修予定箇所について
『きららファンタジア』をご利用いただき、誠にありがとうございます。

9月27日12:00より予定しておりますメンテナンスおよび
Ver1.11.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

機能アップデート

・新機能「覚醒」の追加
　詳細はおしらせ「【先行公開！】新機能「覚醒」をご紹介」をご確認下さい。

・「消費スタミナ1/2」機能の改修
　消費スタミナ1/2キャンペーン開催中はホーム画面上や
　クエスト一覧などにアイコンが表示されるよう改修いたしまます。

・バトル中のスキルアイコンの変更
　キャラクターのステータス上昇及び減少スキルのアイコンを調整いたします。


不具合対応

・毎日4時のセッション切れエラーが発生するタイミングで
　タイトルに戻った際にプレイ再開するとタイトル画面から
　進行しないことがある不具合の修正
・その他、軽微な不具合の修正
