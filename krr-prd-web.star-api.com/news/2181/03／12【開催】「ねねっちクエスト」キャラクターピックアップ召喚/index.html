【開催】「ねねっちクエスト」キャラクターピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。
3月12日 メンテナンス後より、「ねねっちクエスト」キャラクターピックアップ召喚を開催いたします。

「NEW GAME!」より★5 滝本 ひふみ（CV.山口愛）、「キルミーベイベー」より★5 ソーニャ（CV.田村睦心）、「きんいろモザイク」より★4 烏丸 さくら（CV.佐藤聡美）が『きららファンタジア』に登場！

 

更に「きんいろモザイク」★4 大宮 勇（CV.田村ゆかり）も出現率がアップしています。

 

 

 

開催期間

3月12日 メンテナンス後 ～ 3月19日 16:59　3月19日 15:59
※期間によって、ピックアップされる対象が異なりますのでご注意ください。

 

3月12日 メンテナンス後 ～ 3月15日 23:59　
ピックアップ対象：★5 滝本 ひふみ　★4 烏丸 さくら　★4 大宮 勇
通常確率での出現：★5 ソーニャ

 

 

3月16日 0:00 ～ 3月19日 16:59　3月19日 15:59
ピックアップ対象：★5 ソーニャ　★4 烏丸 さくら　★4 大宮 勇
通常確率での出現：★5 滝本 ひふみ

 

 

※3/19 16:00～のメンテナンス実施に伴い、終了時刻を3/19 15:59に変更いたします。
ご迷惑をおかけいたしますが、何卒よろしくお願い申し上げます。

 

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。
※ピックアップ変更時に、初回のみ星彩石300個で『10回召喚』が可能になります。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、3月12日 メンテナンス後から開催されるイベント「ねねっちクエスト」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「ねねっちクエスト」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「ねねっちクエスト」のおしらせをご確認下さい。

 

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
