　イベント「暗雲！ヒナ祭り」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

2月28日 17:00より、
イベント「暗雲！ヒナ祭り」を開催いたします。

開催期間

2月28日 17:00 ～ 3月13日 13:59

イベント概要

本イベントは、シナリオクエストと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。
イベントの報酬や遊び方の詳細は、
別途掲載のおしらせ「イベント『暗雲！ヒナ祭り』 遊び方」を
ご確認ください。

シナリオクエストでオリジナルストーリーを楽しもう！

シナリオクエストは全14節あり、各クエスト毎の初回クリア報酬で
星彩石×5個がもらえます。
シナリオクエストを全てクリアすると、収集クエストが開放され、
指定の日時以降に後日談シナリオを読むことができます。

※シナリオクエストの14節のみ、初回クリア報酬は星彩石×5個ではなく
　イベント素材「ヒナの平額」×1個となります。

収集クエスト、【超強敵】クエストでイベント素材を集めよう！

クエストで集めたイベント素材は、
イベント限定キャラクターや様々なアイテムと交換できます。
また、★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材と交換が可能です。
専用ぶきを進化させると、強力なスキルが使用できるようになるほか、
全編フルボイスの「メモリアルクエスト」の続きが読めるようになります。
さらに、専用の挑戦券でのみ挑戦できる【超強敵】クエストでは、
より多くのイベント素材や限界突破素材が入手できます。

※メモリアルクエストは、
　MENU＞クエスト＞メモリアルクエスト から閲覧できます。

さらなる挑戦！【極】クエスト！

シナリオクエストをクリアした方は難易度の高いクエスト、
【極】クエストに挑戦してみましょう！
初回クリアで星彩石×40個、クリアランクコンプリート報酬で、
限界突破素材と交換ができるイベント素材「ヒナの平額」を入手できます。

イベントをプレイして称号を取得しよう！

今回のイベントでは、最大3つの称号が取得できます。

取得条件
・シナリオクエストを全てクリアする
・【超強敵】クエストをクリアする
・【極】クエストをクリアする

※称号機能については、
　別途掲載の「Ver 2.0.0大型アップデートのおしらせ」をご確認ください。

購入可能期間
2月28日 17:00 ～ 3月13日 13:59

販売価格
490円

購入場所

　毎日初回ログイン時（1日目は購入直後）に入手できます。
　ただし、期間中ログインを行わなかった日は星彩石（無償）を入手できません。
※2日目以降の星彩石（無償）はプレゼントリストへ送られます。
※購入後のきららパスポートの有効期限は星彩石購入ウィンドウの
　「イベント」タブもしくは「購入履歴」から確認できます。
※「暗雲！ヒナ祭り」きららパスポートは1度しか購入できません。
※イベント「暗雲！ヒナ祭り」終了後も有効期限までの間は
　毎日初回ログイン時に星彩石（無償）が入手できます。
