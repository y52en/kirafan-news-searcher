「強敵チャレンジクエストラッシュ！」遊び方

1日1クエスト追加！過去に登場した強敵たちに挑戦しよう！

過去に開催された「強敵チャレンジクエスト」の強敵たちが登場する
「強敵チャレンジクエストラッシュ！」を開催いたします。
2周年当日までのカウントダウンクエストとして、1日1クエストずつ追加されます。

「強敵チャレンジクエストラッシュ！」は、
MENU＞クエスト＞2周年カウントダウンクエスト！
　　　　　　　　＞強敵チャレンジクエストラッシュ！ から挑戦可能です。
また、強敵チャレンジクエストに挑戦するには、
クエストキー　銀×3個を消費する必要があります。

開催期間

12月4日 0:00 ～ 12月10日23:59

1日1クエストずつ追加されますが、
クエスト終了日時はすべて12月10日23:59となります。
また、12月10日 12:00 ～ 17:00でメンテナンスの実施を予定しております。
プレイ可能な時間が短くなっておりますので、予めご了承ください。

「大×2感謝抽選券」を集めよう！

本クエストでは「大×2感謝リアルグッズ抽選会」の参加に必要な
「大×2感謝抽選券」を集めることができます。
なお、通常の「強敵チャレンジクエスト」で入手できる「チャレンジメダル」は、
本クエストでは入手できません。

「大×2感謝抽選券」は、 各クエストの初回クリア報酬として22枚（計154枚）、
全7クエストを誰も撤退せずにクリアすることで222枚入手でき、
本クエストを通して最大376枚入手できます。

「大×2感謝リアルグッズ抽選会」については、
別途掲載のおしらせ「『大×2感謝リアルグッズ抽選会』開催！」をご確認ください。
