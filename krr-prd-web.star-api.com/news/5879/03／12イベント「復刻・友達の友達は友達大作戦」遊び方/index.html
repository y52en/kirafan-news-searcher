イベント「復刻・友達の友達は友達大作戦」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「黄色いブーケ」や
「絵馬の根付」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「復刻・友達の友達は友達大作戦」からご確認ください。

また、イベント素材の交換期限は4月3日 16:59までとなっておりますので
期限までに交換をお願いいたします。

 

「イベント復刻コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント復刻コールチケット」が入手できます。
イベント復刻コールチケットは、トレードショップで
過去のイベント報酬で登場した★4キャラクターと交換が可能です。

 

「イベント復刻コールチケット」は、最大9枚入手できます。

たくさん交換して、過去のイベント報酬で登場した★4キャラクターを
覚醒させてみましょう！

 

※各キャラクターの交換上限数は「9」となっております。

※「イベント復刻コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。
※★4キャラクターの覚醒は、最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。
※キャラクターの最大覚醒後に、同一キャラクターを入手した場合は
　輝星のかけらを入手できます。
※「イベント復刻コールチケット」の対象のキャラクター詳細は
　トレードショップ＞限定交換よりご確認ください。
 
 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

 「ハートのゼムクリップ」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

豪華素材がドロップする「【乱戦】クエスト」！

各種豪華素材がドロップする【乱戦】クエストも開催！


【乱戦】クエストに挑戦するためには、
「【乱戦】友達大作戦挑戦券 」が必要です。

開放期間：3月15日 17:00 ～ 3月27日 13:59

 

※使用しない「【乱戦】友達大作戦挑戦券 」は、
　トレードショップ＞限定交換＞「復刻・友達の友達は友達大作戦」から
　「絵馬の根付」に交換することができます。

 

※過去に開催した「友達の友達は友達大作戦」で入手した
　名称/デザインが同一の「【乱戦】友達大作戦挑戦券 」は
　使用できませんのでご注意ください。
