【実施】新機能「トレーニング」追加のおしらせ

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。
3月19日 メンテナンス後から、新機能「トレーニング」が実装されます。

 

機能実装日時

3月19日  メンテナンス後～

 

機能使用条件

ゲームを開始された時点から使用可能となります。

 

新機能「トレーニング」！！

いざ戦闘に出かけても、敵が強くて倒せない……。

そんなクリエメイトをトレーニングして導いてくれるのは、みんなのお姉さん「ライネ」！

かつて勇者だったときの経験を活かして、様々なトレーニング課題を設定してくれるらしい……？

 

「トレーニング」機能では、所持しているキャラクターをパーティー編成し、トレーニングに挑戦することができます。
トレーニング画面へは、ホーム画面や里画面のトレーニング施設をタップすることで移動します。


トレーニングに挑戦し、一定の時間が経過することで、パーティーに編成したキャラクターに、キャラクター経験値やアイテムなどを獲得させることができます。


挑戦中のトレーニングは星彩石を使用することで、残り時間に関係なく完了させることができます。


※トレーニング挑戦中のキャラクターも、クエストに参加させることは可能です。

 

 

遊び方

トレーニング画面ではトレーニング候補が複数表示されます。
トレーニングの種類によっては、参加条件が設けられているものもあり、その条件を満たしている場合のみ、トレーニングに挑戦することができます。

 

 

また、各トレーニングの中には挑戦可能回数を持っているものもあり、挑戦可能回数を持っているトレーニングは、挑戦可能回数分そのトレーニングを行うと、トレーニング候補に表示されなくなります。


※各トレーニングの挑戦可能回数は、毎日0:00にリセットされます。

 

 

トレーニングに挑戦できるパーティーは最大5名で構成され、最大4パーティーまで同時にトレーニングに挑戦することができます。

 

 

トレーニングによっては、指定された人数のパーティーしか挑戦することのできない、パーティー人数の制限がかけられているものも存在します。

 

 

※同名キャラクターを複数のパーティーで編成することはできません。

 

 

トレーニングに挑戦できるパーティーの数は初めは2パーティーですが、ユーザーランクが上がることで追加され、最大4つになります。


トレーニングのリタイアについて

一度挑戦したトレーニングは、各トレーニングをタップし、トレーニング情報内の「リタイア」ボタンをタップすることで、トレーニングの挑戦をリタイアすることができます。

 

 

※挑戦したトレーニングをリタイアした場合、トレーニング開始時に消費されたものは戻ってきません。また、挑戦可能回数を持っているトレーニングをリタイアした場合、挑戦可能回数は回復しません。

 

 

※挑戦したトレーニングをリタイアした場合、トレーニングの報酬を獲得することはできません。
