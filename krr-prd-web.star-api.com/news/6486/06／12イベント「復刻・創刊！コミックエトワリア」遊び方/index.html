イベント「復刻・創刊！コミックエトワリア」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「真っ白な原稿」や「羽根ペン」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「復刻・創刊！コミックエトワリア」からご確認ください。

また、イベント素材の交換期限は7月3日11:59までとなっておりますので
期限までに交換をお願いいたします。

 

「イベント復刻コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント復刻コールチケット」が入手できます。
イベント復刻コールチケットは、トレードショップで
過去のイベント報酬で登場した★4キャラクターと交換が可能です。

 

「イベント復刻コールチケット」は、最大9枚入手できます。

たくさん交換して、過去のイベント報酬で登場した★4キャラクターを
覚醒させてみましょう！

 

※各キャラクターの交換上限数は「9」となっております。

※「イベント復刻コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。
※★4キャラクターの覚醒は、最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。
※キャラクターの最大覚醒後に、同一キャラクターを入手した場合は
　輝星のかけらを入手できます。
※「イベント復刻コールチケット」の対象のキャラクター詳細は
　トレードショップ＞限定交換＞「復刻・創刊！コミックエトワリア」よりご確認ください。

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

 「コミックエトワリア創刊号」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

 

イベントボーナス効果について

今回のイベントでは、一部のキャラクターがイベントボーナス効果を持っています。
イベントボーナス効果を持つキャラクターをパーティーに編成することで、
イベント「復刻・創刊！コミックエトワリア」のクエストにおいて
イベント素材のドロップ数がアップします。

★4 萌田 薫子（そうりょ　風属性）
★4 恋塚 小夢（まほうつかい　陽属性）

※イベントボーナスの効果は、サブメンバー・サポート枠に編成された

　キャラクターからも受けることができます。

 

豪華素材がドロップする「【超強敵】クエスト」開催！


【超強敵】クエストに挑戦するためには、専用の挑戦券が必要です。

開放期間：6月15日 17:00 ～ 6月26日 13:59

 

※使用しない【超強敵】挑戦券 は、
　トレードショップ＞限定交換＞「復刻・創刊！コミックエトワリア」から
　「羽根ペン」に交換することができます。

※過去に開催した「創刊！コミックエトワリア」で入手した
　名称/デザインが同一の「【超強敵】連載会議挑戦券 」は
　使用できませんのでご注意ください。

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたら、
さらなる高難易度【極】クエストに挑戦してみましょう！

【極】クエストに挑戦できます。挑戦するためにはスタミナを消費します。

 

開放期間：6月17日 17:00 ～ 6月26日 13:59
