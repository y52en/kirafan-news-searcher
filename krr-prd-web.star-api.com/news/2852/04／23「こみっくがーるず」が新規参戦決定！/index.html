「こみっくがーるず」が新規参戦決定！
『きららファンタジア』に、「こみっくがーるず」の新規参戦が決定しました！

 

■「こみっくがーるず」とは
「こみっくがーるず」は、芳文社「まんがタイムきららMAX」にて連載中のはんざわかおり先生による4コマ漫画。

へっぽこ4コマまんが家のかおす、ふわふわ少女まんが家の小夢、えっちなTLまんが家の琉姫、熱血少年まんが家の翼は、文芳社の女子寮に住む現役女子高生まんが家！

お互いの仕事を手伝ったり、締め切りに追われたり、まんがトークに花を咲かせたり……。
「こみっくがーるず」の日常は、普通とはちょっと違う！！

また、「こみっくがーるず」の参戦決定CMを公開中です！ 詳しくは公式サイトのニュースをご覧ください。
