　Ver 1.18.0 改修内容の詳細

『きららファンタジア』をご利用いただき、誠にありがとうございます。

7月3日に実施予定のVer1.18.0アップデートにおいて

実装予定の改善内容の詳細についてご案内いたします。

 

ステータス・強化画面の統合

キャラクター強化画面において、「ステータス」・「強化」・「限界突破」・「進化」が統合されるよう改修いたします。

また、上記に伴い各画面のUI・機能を改修いたします。

これにより、「強化」・「限界突破」・「進化」

各画面の行き来がスムーズになります。

※強化画面以外で「ステータス画面」を開いた場合の

　強化系画面への移動はできません。

 

限界突破画面の改善

限界突破画面において、限界突破アイテムが複数使用可能になりました。

これにより、キャラクターを一度に最大まで限界突破させることが可能になります。

 

公式Twitterで新しくなった強化画面の動画を公開中！

よりキャラクターを育成しやすくなった強化画面をご覧ください。

 

曜日クエストの導線改善

画面右下のバナーをタップすることで

常に曜日クエストの開催スケジュールを確認できるようになりました。

 

皆様により一層『きららファンタジア』をお楽しみいただけますよう、
スタッフ一同鋭意努めてまいります。

今後とも、『きららファンタジア』をよろしくお願い申し上げます。
