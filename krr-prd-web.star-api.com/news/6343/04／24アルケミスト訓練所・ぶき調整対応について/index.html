一部訓練所・ぶきについて以下の調整を行います。

 

調整内容

・強化施設：アルケミスト訓練所の見直し
　施設の性能を見直し、物理攻撃力(ATK)の代わりに
　行動速度(SPD)が上昇するようになります。

 

・アルケミストのぶきの見直し
　アルケミストのぶきを装備した時のステータス上昇量を見直します。
　対象となるぶきは以下の通りです。

 

　＜対象ぶき＞
　・古びた試験管
　・レプリカフラスコ
　・見習いの試験管
　・ノーマルフラスコ
　・中級者の試験管
　・グレートフラスコ
　・上級者の試験管
　・マスターフラスコ
　・達人の試験管
　・スターフラスコ
　・清夏のびんづめ
