【開催】★5選べるチケット付き！有償限定10回召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

『きららファンタジア』リリース200日を記念して、6月28日 17:00より、★5選べるチケット付き！有償限定10回召喚を開催いたします。

 

開催期間

6月28日 17:00 ～ 7月11日 11:59

 

内容

・お一人様一回限り、有償星彩石限定の召喚となります。

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

・「10回召喚」実行時に、「リリース200日記念スペシャルコールチケット」を入手することができます。

 

リリース200日記念スペシャルコールチケットとは

「リリース200日記念スペシャルコールチケット」とは、★5キャラクター1人を指定して仲間にすることができるチケットです。

指定することのできる★5キャラクターについては、ゲーム内の召喚画面＞スペシャルコールチケット召喚にて必ずご確認の上、スペシャルコールチケットをご使用下さい。

既に加入済みのキャラクターが存在する場合に、その加入済みのキャラクターを「リリース200日記念スペシャルコールチケット」で仲間にすると、そのキャラクターのクラスの限界突破素材を入手できます。

 

※「★5選べるチケット付き！有償限定10回召喚」は、有償星彩石400個で召喚可能です。無償星彩石では召喚できません。


※星彩石購入の際に付与されるオマケ分の星彩石は、無償星彩石となります。
所持している星彩石の内訳は、召喚画面下部の表示、もしくはホーム画面右上のMENUボタン＞設定・その他ボタン＞購入履歴ボタンよりご確認いただけます。

 

※「リリース200日記念スペシャルコールチケット」の使用期限は7月18日 23:59です。期限が過ぎますと使用不可となりますので、必ず期限内での使用をお願いいたします。

 

※本キャンペーンの対象以外の★5キャラクターは、「リリース200日記念スペシャルコールチケット」の召喚候補には追加されません。ご注意下さい。

 

※10回召喚を行った際には、特別アイテム「星屑のかけら」は入手できますが、「リリース200日記念スペシャルコールチケット」使用時の召喚では、特別アイテム「星屑のかけら」は入手できません。

 

※100日記念キャンペーンの「★5選べるチケット付き！有償限定10回召喚」にて入手したスペシャルコールチケットは使用できません。
