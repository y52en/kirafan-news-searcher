メインクエスト　ハードモード3章・4章 追加！

メインクエストにハードモード3章・4章を追加実装いたします。
ハードモード3章・4章は、メインクエスト5章（ノーマル） 最終節をクリアすると開放されます。

※ハードモードのシナリオ内容は、ノーマルモードと同じ内容となります。


また、このハードモードでは特定条件をクリアすると、各章で最大220個の星彩石を手に入れることができます。

・クエストクリア報酬：星彩石×5個
→ハードモード各クエストをクリアする

・オールクリア報酬：星彩石×20個
→ハードモード各章内の全クエストをクリアする（32節までクリア）

・オールコンプリートクリア報酬：星彩石×40個
→ハードモード各章内の全クエストをクリアランクコンプリート達成する
（コンプリートはパーティーメンバーが一人も戦闘不能にならずにクリアすることで獲得）

詳細はホーム画面＞クエスト画面＞メインクエストにてご確認下さい。
※メインクエスト画面左下の章選択ボタンより、3章・4章のどちらかを選択後、画面左上の「ハード」ボタンをタップすることで移動できます。
