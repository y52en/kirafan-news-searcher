【開催】「野クルと北の山と初キャンプ△」ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

4月25日 メンテナンス後より、「野クルと北の山と初キャンプ△」ピックアップ召喚を開催いたします。

 

「ゆるキャン△」より★5 志摩 リン（CV.東山奈央）、★4 各務原 なでしこ（CV.花守ゆみり）、★4 大垣 千明（CV.原紗友里）、★4 犬山 あおい（CV.豊崎愛生）が『きららファンタジア』に初登場！

 

 

 

開催期間

4月25日 メンテナンス後 ～ 5月9日 11:59

 

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「野クルと北の山と初キャンプ△」召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、4月25日 メンテナンス後から開催されるイベント「野クルと北の山と初キャンプ△」にてイベントボーナスの効果を持っています。

イベントボーナスを持ったキャラクターは、イベント「野クルと北の山と初キャンプ△」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「野クルと北の山と初キャンプ△」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
