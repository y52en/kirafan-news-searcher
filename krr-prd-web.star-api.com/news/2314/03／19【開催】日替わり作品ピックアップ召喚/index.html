【開催】日替わり作品ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

 

『きららファンタジア』リリース100日を記念して、3月19日メンテナンス後より、各作品のキャラクターが日替わりでピックアップされた「日替わり作品ピックアップ召喚」を開催いたします。

 


特定の作品キャラクターを入手する機会ですので、スケジュールをご確認の上、日替わり作品ピックアップ召喚をご利用ください。

 

 

開催期間

全体開催期間　　：3月19日 メンテナンス後 ~3月28日 11:59
各召喚の開催時間：各召喚開催日 0:00～ 23:59

 


※初日の開始時刻は3月19日メンテナンス後～となります。
※最終日の終了時刻は3月28日 11:59となります。

 

 

 

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・本召喚のピックアップ対象キャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。
