開催期間

12月4日 0:00 ～ 12月10日 23:59

内容

専用の「大×2感謝抽選券」を集めることで、
きらら役 楠木ともりさん＆ランプ役 高野麻里佳さんの直筆サイン入り

「大×2感謝抽選券」を持っていればいるほど当選確率が上がりますので
毎日ログイン＆プレイして、たくさん集めましょう！

12月4日 0:00 ～ 12月10日 23:59

内容

開催期間中、ログインすることで「大×2感謝リアルグッズ抽選会」の参加に必要な
「大×2感謝抽選券」が1日につき22枚、
毎日ログインすることで最大154枚もらえます。

開催期間

12月4日 0:00 ～ 12月10日 23:59

内容

過去に開催された「強敵チャレンジクエスト」の強敵たちが登場する
「強敵チャレンジクエストラッシュ！」を開催いたします。
2周年当日までのカウントダウンクエストとして、1日1クエストずつ追加されます。

「大×2感謝抽選券」を入手できます。

詳細は、別途掲載のおしらせ
「『強敵チャレンジクエストラッシュ！』遊び方」をご確認ください。

開催期間

12月4日 0:00 ～ 12月10日 23:59

内容

クリア回数に応じて「大×2感謝抽選券」や、「召喚チケット」、
「エトワリウムのかけら」などの報酬を入手できる特別ミッションを実施いたします。
「大×2感謝抽選券」は、クエストを5回クリアすることで1日につき22枚、
毎日ミッションをクリアすることで最大154枚入手できます。

※ミッション内容や報酬は、MENU＞ミッション＞イベント よりご確認ください。
※「大×2感謝！抽選券」22枚が報酬のミッション「クエストを5回クリア」は
　毎日リセットされます。報酬は当日中にお受け取りください。
※12月10日 12:00 ～ 17:00の間はメンテナンスのため、
　プレイ可能時間が短くなっております。ご注意ください。
