　Ver 1.18.0の改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

7月3日に実施予定のVer 1.18.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・ステータス・強化画面の統合
　※MENU項目「強化」内における

　　ステータス画面・強化画面・限界突破画面・進化画面が

　　統合されるよう改修

　※上記に伴うUI・機能の改修

 

・パーティー編成枠の増加
　※パーティー編成枠を5枠増加

 

・曜日クエストの導線改善

　※ホーム画面右下のバナーから曜日クエストスケジュールが

　　確認できるよう改修


・細かな機能改修
　・チュートリアル中に表示されるポップアップ画像を変更　

　・ルームアニメーションのロード改善
　・その他、機能改修など
