Ver1.12.0での改修予定箇所について
『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.12.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

機能アップデート

・チュートリアル召喚の改修
　チュートリアル召喚を好きな★5キャラクターを選べる召喚に変更いたします。
　また、ゲーム開始時にプレゼントリストに配布されるキャラクターと
　召喚チケット内容を調整いたします。
　※対応の詳細は別途掲載のおしらせ、
　　「【重要】Ver1.12.0 チュートリアル召喚改修に伴う各種対応について」を
　　ご確認下さい。

・任意のキャラクターを選べる召喚UIの改修
　任意のキャラクターを選べる召喚時に
　選べるキャラクターが見やすくなるよう改修いたします。

・トレーニング/里UIのレベル表記など一部UI表示改修
　　・トレーニング内各画面に現在所持しているきららポイントの表示を追加
　　・トレーニングパーティー選択画面のキャラクターアイコン上に
　　　キャラクターレベルの表示を追加
　　・コンテンツ施設一覧画面に施設レベルの表示を追加
　　・施設倉庫画面に施設レベルの表示を追加
　上記画面のUI表示を改修いたします。

・とっておきSKIP機能の追加
　キャラクターのとっておき演出中に画面をタップすると
　演出をSKIPする機能を追加いたします。
　※デフォルト設定では機能がOFFになっております。
　　オプションのバトル設定よりONに変更することで使用できます。

・バトル終了時にもう一度挑戦する機能の追加
　バトル終了時に「もう一度挑戦する」機能を追加いたします。
　初回クリア以降、クエストクリア時に同じ編成（サポート含む）で
　クエスト一覧などに戻らずそのままバトルを繰り返すことができるようになります。

・身長が小さいキャラクターのデフォルメスケール調整
　キャラクター設定上、身長が一定より低い一部キャラクターの
　デフォルメスケールを調整いたします。
　スケール調整はすべてのキャラクターを同時ではなく順次行う予定です。

・一部スキル演出改修
　ダメージ無効の「バリア」とダメージ軽減の「ダメージカット」の演出を、
　それぞれの効果にあわせて変化するように調整いたします。
　スキル効果に変更はございません。
