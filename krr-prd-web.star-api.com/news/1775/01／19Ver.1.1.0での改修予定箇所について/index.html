Ver.1.1.0での改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

1/22 0:00より予定しておりますメンテナンスおよびVer1.1.0のアップデートにおいて実装予定の各機能改善内容についてご案内いたします。

 

 

トレードショップ

・アイテムの複数交換に対応いたします。

 

バトル

・オートモード時のターゲット選定を強化いたします。
・ゲストキャラクターの滞在時間が1ターンから3ターンに変更されます。
・フレンドキャラクターの滞在時間が3ターンから5ターンに変更されます。

 

デイリー/ウィークリーミッション

・「里でキャラクターのおでかけアイコンを●回タップ」のミッションが、「生産施設の報酬アイコンをタップ」でも達成できるよう、条件が変更されます。

 

里

・里でキャラクターをタップした時のきららポイント入手量がアップします。
・里でキャラクターをタップした時の特別演出発生率がアップします。
・上記変更に伴い、入居者変更が1時間に1回までに調整されます。
