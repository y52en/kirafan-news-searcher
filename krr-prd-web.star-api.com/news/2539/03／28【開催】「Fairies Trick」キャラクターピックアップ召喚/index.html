【開催】「Fairies Trick」キャラクターピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

3月28日 メンテナンス後より、「Fairies Trick」キャラクターピックアップ召喚を開催いたします。

 

 

「桜Trick」より★5 園田 優（CV.井口裕香）、★4 野田 コトネ（CV.相坂優歌）、★4 南 しずく（CV.五十嵐裕美）、★4 池野 楓（CV.渕上舞）、★4 飯塚 ゆず（CV.戸田めぐみ）が『きららファンタジア』に登場！

 

更に、★5 高山 春香（CV.戸松遥）も出現率がアップしています。

 

 

開催期間

全体開催期間：3月28日 メンテナンス後 ～ 4月11日 13:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意ください。

 

・ピックアップ召喚前半：3月28日 メンテナンス後 ～ 4月11日 13:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。
※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「Fairies Trick」キャラクターピックアップ召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、3月28日 メンテナンス後から開催されるイベント「Fairies Trick」にてイベントボーナスの効果を持っています。

イベントボーナスを持ったキャラクターは、イベント「Fairies Trick」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「Fairies Trick」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。


 

※「★4　高山 春香」「★4　園田 優」「★3　高山 春香」「★3　園田 優」は「Fairies Trick」キャラクターピックアップ召喚のピックアップ対象キャラクターに含まれておりません。ご注意ください。
