芳文社の公式マンガアプリ「COMIC FUZ」事前登録受付中！



芳文社初の公式マンガアプリ「COMIC FUZ（コミックファズ）」が、
2019年春に創刊決定！
現在、こちらのサイトで事前登録キャンペーンを開始しております。
https://cp.comic-fuz.com

「COMIC FUZ」は、芳文社が発行しているマンガ雑誌や連載されている
人気マンガを楽しめるアプリです。
アプリ内でメダルを使用することで、約1000以上のマンガを楽しむことができ、
毎日更新される連載コーナーでは、
「COMIC FUZ」でしか読めない描き下ろしマンガが多数ラインナップされます。

創刊の際には、『きららファンタジア』にも参戦中の人気マンガ「ゆるキャン△」が、
現在の「まんがタイムきららフォワード」から「COMIC FUZ」に
連載の場を移します。
それ以外にも、多数の『きららファンタジア』参戦作品が掲載予定です。
この機会にぜひ、未読だった作品を、
事前登録キャンペーンでもらえるメダルで読破してください！

事前登録はこちらから！
