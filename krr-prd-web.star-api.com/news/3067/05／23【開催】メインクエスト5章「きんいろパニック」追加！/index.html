5月23日 17:00～  17:30～

クエスト開放条件

ランプが今まで隠してきた１つの事実……?

女神ソラを救うための旅を続けるランプ、マッチ、きらら。
筆頭神官アルシーヴとの戦いに敗れた後、ランプがゆっくりと口を開く。
今までランプが隠してきた、ある1つの事実とは……？

 

今回は「きんいろモザイク」のキャラクターたちがエトワリアに登場！


 

新キャラクター紹介

新しく追加されるメインクエスト5章「きんいろパニック」に登場する、カヅホ先生描き下ろしのオリジナルキャラクター・ジンジャーをご紹介！

 

七賢者の一人で、言の葉の樹の根本にある街の統治者でもあるジンジャー。
街を守り、そこに住まう民を守るため、きららたちの前に立ちふさがる……！

 

 

メインクエスト　ハードモード１章・2章 追加！

現在実装されているノーマルモードに加え、メインクエストにハードモード1章・2章を実装いたします。
ハードモード1章・2章の開放条件は、メインクエスト5章（ノーマル） 最終節クリアです。

※ハードモードのシナリオ内容は、ノーマルモードと同じ内容となります。


また、このハードモードでは特定条件をクリアすると、各章で最大220個の星彩石を手に入れることができます。

・クエストクリア報酬：星彩石×5個
→ハードモード各クエストをクリアする

・オールクリア報酬：星彩石×20個
→ハードモード各章内の全クエストをクリアする（32節までクリア）

・オールコンプリートクリア報酬：星彩石×40個
→ハードモード各章内の全クエストをクリアランクコンプリート達成する
（コンプリートはパーティーメンバーが一人も戦闘不能にならずにクリアすることで獲得）

詳細はホーム画面＞クエスト画面＞メインクエストにてご確認ください。
※メインクエスト画面左下の章選択ボタンより、1章・2章のどちらかを選択後、画面左上の「ハード」ボタンをタップすることで移動できます。

 

 

新章追加で新ぶき開放！

メインクエスト5章の追加にともない、鍛冶屋にて新しいぶきが作成できるようになりました。
作成に必要な素材は、メインクエスト5章やトレーニングで入手できます。

 

詳細はホーム画面＞商店街＞鍛冶屋＞ぶき作成にてご確認ください。

 

トレーニング内容追加！

・新ぶき作成に必要な素材が手に入るトレーニングを追加
メインクエスト5章追加に合わせて、トレーニングに新たなトレーニング「素材集め5～市場で警備の手伝い～」が追加されます。新ぶき作成に必要な素材が手に入りますので、ぜひお試しください。

 

・素材集めのトレーニングの挑戦回数を変更
また、素材集め1～5のトレーニングの挑戦可能回数を現状の3回から10回に変更いたします。
