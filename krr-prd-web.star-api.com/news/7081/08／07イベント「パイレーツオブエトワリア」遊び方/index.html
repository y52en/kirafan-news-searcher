イベント「パイレーツオブエトワリア」遊び方

マップをクリアしよう ！

マップ式クエストは、地図上に表示された「エリア」で発生するクエストを
クリアしていくことで、ストーリーが進んでいくイベント形式です。
クエストマップ上に表示されている「エリアアイコン」をタップし、
各エリアで発生するシナリオクエストを読み進めていきましょう。

シナリオクエストを全てクリアすると収集クエストが開放されます。

 

収集クエストでは、後述のふしぎな宝箱を開封するために

必要な宝箱開封用アイテム「海賊の金貨」や、

【超強敵】クエストに挑戦するために必要な【超強敵】挑戦券が入手できます。

 

ふしぎな宝箱を開封しよう！

ふしぎな宝箱から手に入るアイテムの中には、
トレードショップで別のアイテムに交換できるものもあります。
詳細は『イベント限定キャラクターを仲間にしよう！』をご確認ください。

 

宝箱開封可能期間：8月8日 17:00 ～ 9月2日 13:59

※ふしぎな宝箱の開封は、開封可能期間の間しか行うことができません。

　交換忘れにはご注意ください。

 

※一部のふしぎな宝箱は「あたりアイテム」が設定されており
　「あたりアイテム」を手に入れると、宝箱のリセットを行うかどうか
　選択できるようになります。


※「あたりアイテム」が設定されていないふしぎな宝箱は
　宝箱の全ての中身を空にするまでリセットを行うことができません。


※ふしぎな宝箱の各箱にて宝箱のリセットを行うと次の宝箱に進んでしまうため、
　以前の宝箱で残っていた中身を引き直すことはできません。
　宝箱のリセットを行う際はご注意ください。


※ふしぎな宝箱で入手できるアイテムにつきましては、
　ゲーム内の「ふしぎな宝箱」画面の「中身を確認する」ボタンからご確認ください。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。

★4 遥【水着】(そうりょ　土属性)に交換が可能です。

たくさん交換して、★4 遥【水着】(そうりょ　土属性)

を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。

　最大4回まで可能となっており

　チケットを複数枚所持していた場合でも1回ずつの交換となります。

豪華素材がドロップする「【超強敵】クエスト」！

【超強敵】クエストに挑戦するためには、専用の挑戦券が必要です。

※使用しない【超強敵】挑戦券 は、
　トレードショップ＞限定交換＞「パイレーツオブエトワリア」から
　「海賊の金貨」に交換することができます。

開放期間：8月10日 17:00 ～ 8月26日 13:59

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたら、
さらなる高難易度【極】クエストに挑戦してみましょう！
すべてのシナリオクエストをクリアしていると、 開放時刻以降に
【極】クエストに挑戦できます。挑戦するためにはスタミナを消費します。

開放期間：8月12日 17:00 ～ 8月26日 13:59

 

対決！海上の略奪者！「【超高難易度】クエスト」！

【極】クエストをクリアした後は
【超高難易度】クエストに挑戦してみましょう！
【極】クエストをクリアしていると、
開放時刻以降に【超高難易度】クエストに挑戦ができます。

挑戦するためにはスタミナを消費します。
【極】クエストよりも難易度が高い分、

クリアランクコンプリート時に入手できる報酬がより豪華となっています。

開放期間：8月15日 17:00 ～ 8月26日 13:59
