【開催】日替わりクラス限定召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

7月11日 15:00より、各クラスのキャラクターが日替わりでピックアップされた「日替わりクラス限定召喚」を開催いたします。

 

特定のクラスのキャラクターを入手する機会ですので、スケジュールをご確認のうえ、ぜひご利用下さい。

 

開催期間

全体開催期間　　：7月11日 15:00 ~7月23日 11:59 9:59　
各召喚の開催時間：各召喚開催日 0:00～ 23:59

※初日の開始時刻は7月11日 15:00～となります。
※最終日の終了時刻は7月23日 11:59 9:59となります。

※7月23日 10:00～のメンテナンス実施に伴い、終了時刻を7月23日 9:59に変更いたします。 ご迷惑をおかけいたしますが、何卒よろしくお願い申し上げます。（7月23日 19:50追記）

 

開催内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・各日替わり召喚の初回は、星彩石300個で「10回召喚」が可能です。

・本召喚のピックアップ対象キャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

日替わりクラス限定召喚開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

出現キャラクター一部紹介

※上記バナーのキャラクター以外の★5キャラクターは各日替わり召喚の出現対象キャラクターに含まれておりません。ご注意下さい。

 

 

注意事項

・提供割合の詳細はゲーム内の召喚画面＞提供割合からご確認下さい。

・本召喚では、ピックアップ対象キャラクターは存在せず、同一レアリティの他のキャラクターよりも提供割合が高く設定されているキャラクターは存在しません。ご注意下さい。

・キャンペーン内容は予告なく変更になる場合がございます。ご了承下さい。
