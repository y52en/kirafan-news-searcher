Ver1.15.0での改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.15.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・クエスト出撃時に「きらら装備」を変更できるよう改修
・ソート項目に「覚醒」を追加、その他ソート項目の改修
・おきにいり設定方法の改修
　・ユーザー情報画面から、おきにいりメンバーを設定できるように改修
　　※改修後は、入居者一覧を変更しても「おきにいりメンバー」は変更されません
・ルームアイテム設置上限数を、「メイン」「サブ」それぞれで管理できるよう改修


・細かな機能改修
　・ミッション期限の表記改修
　・トレードショップ表示の改修
　・サポート選択画面での各イベントボーナス値表示改修
　・アイテム獲得画面の表示改修 
　・一部キャラクターのスキル説明文言修正
　　※スキル効果内容は変更ありません。

 

不具合対応

　・軽微な不具合の修正


皆様により一層『きららファンタジア』をお楽しみいただけますよう、
スタッフ一同鋭意努めてまいります。

今後とも、『きららファンタジア』をよろしくお願い申し上げます。
