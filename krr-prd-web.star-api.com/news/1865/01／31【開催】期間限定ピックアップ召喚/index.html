【開催】期間限定ピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

1月31日 16:30より、新イベント「チョコレートクエスト」で活躍するキャラクターが登場する「期間限定ピックアップ召喚」を開催いたします。
バレンタインにちなんだ★5「きんいろモザイク」綾(CV.種田梨沙)、★5「ステラのまほう」椎奈(CV.村川梨衣)、★4「NEW GAME!」ツバメ(CV.大和田仁美)が期間限定で登場します。

 

 

 

 

開催期間

1月31日 16:30 ～ 2月16日 13:59

※期間限定キャラクターはこの期間中のみ召喚から登場いたします。

 

内容

・10回召喚では★4以上のキャラクターが1枚以上確定します。
・初回のみ星彩石300個で10回召喚が可能です。
・期間中、下記ピックアップ対象キャラクターの提供割合がアップしています。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、2月6日から開催されるイベント「チョコレートクエスト」にてイベントボーナスの効果を持っています。


イベントボーナスを持ったキャラクターは、イベント「チョコレートクエスト」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「チョコレートクエスト」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
