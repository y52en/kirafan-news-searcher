強敵チケットについて

 

「新春メカこけし挑戦券」は、イベント「きんいろNEW YEAR!」の強敵クエストに挑戦することができるチケットです。

※強敵クエストは2018年1月3日 17:00より開放されます。

 

見事強敵を倒してクエストをクリアすると、確定で「大吉こけし」が40個ドロップするほか、
低確率で★4キャラクターの限界突破アイテムである「まほうつかいの果実」を入手することができます。

 

詳しくはイベント「きんいろNEW YEAR!」のおしらせをご確認ください。
