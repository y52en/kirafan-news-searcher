イベント「リュウグウ・アドベンチャー」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。
7月24日 17:00より、イベント「リュウグウ・アドベンチャー」を
開催いたします。

 

開催期間

7月24日17:00 ～ 8月26日 13:59

 

イベント概要

本イベントは、シナリオクエストと収集クエスト、
【超強敵】クエスト、【極】クエストで構成されています。
イベントの報酬や遊び方の詳細は、
別途掲載のおしらせ「イベント『リュウグウ・アドベンチャー』 遊び方」または

イベント画面＞イベント詳細をご確認ください。

 

シナリオクエストでオリジナルストーリーを楽しもう！

シナリオクエストは全13節で構成されています。

節毎に開放日時が区切られており、

指定の日時以降に新しい節をプレイできるようになります。

また、すべてのシナリオクエストをクリアしていると

8月8日 17:00より開催予定のイベント「パイレーツオブエトワリア」

へ繋がるシナリオ「幕間劇」が開放されます。

「幕間劇」は全4節のシナリオで構成されています。

 

【シナリオ開放スケジュール】

※シナリオクエスト4節・8節・13節をそれぞれクリアすることで

　収集クエストが開放されます。

※「リュウグウ・アドベンチャー」の各シナリオを読むことで
　星彩石×5個を獲得できます。

※シナリオクエストの13節のみ、初回クリア報酬は星彩石×5個ではなく

　「マル秘の台本」×1個となります。

※本イベント「リュウグウ・アドベンチャー」より、
　シナリオクエストは開放日時ごとのバナーで分割されず、
　一つのバナー内にまとめて表示されるようになります。

 

収集クエスト、【超強敵】クエストで素材を集めよう！

クエストで集めた素材はイベント限定キャラクターや様々なアイテムと交換できます。

★5キャラクターの専用ぶきの進化に必要な「エトワリウム」も
イベント素材を集めることで獲得できます。
専用ぶきを進化させると、強力なスキルが使用できるようになるほか、

全編フルボイスの「メモリアルクエスト」の続きが読めるようになります。

※各キャラクターの「メモリアルクエスト」は、
　クエスト＞メモリアルクエストから閲覧できます。

 

また、専用の挑戦券を消費することで挑戦できる【超強敵】クエストでは
より多くのイベント素材や限界突破素材が入手できます。

 

さらなる挑戦！【極】クエスト！

【超強敵】クエストもクリアしたら

難易度の高いクエスト、【極】クエストに挑戦してみましょう！
初回クリアで星彩石×40個、クリアコンプリート報酬で、
イベント素材「マル秘の台本」×1個が入手できます。
