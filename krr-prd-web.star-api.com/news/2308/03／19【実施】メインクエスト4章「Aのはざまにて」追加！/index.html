女神ソラ様を救う旅は、まだまだ続く……！

草原や港町、砂漠を越え、女神ソラを救うための旅を続けるランプ、マッチ、きらら。

次に一行が立ち寄ったのは、渓谷の町。

一見したところ町は普通の様子だが、よくよく観察してみると町の人々はやっぱり様子がおかしくて…？

 

 

今回は「Aチャンネル」のキャラクターたちがエトワリアに登場！

 

 

新キャラクター紹介



新しく追加されるメインクエスト4章「Aのはざまにて」に登場する、はんざわかおり先生描き下ろしのオリジナルキャラクター・七賢者ソルトをご紹介！


普段は、同じ七賢者のシュガーと行動をともにすることも多いというソルト。どうやらクリエメイトを捕らえる以外にも狙いがあるようで……？　真相はメインクエスト4章で確かめよう！
