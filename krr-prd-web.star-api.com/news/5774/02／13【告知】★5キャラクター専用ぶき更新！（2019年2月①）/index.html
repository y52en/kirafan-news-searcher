更新日時

2月14日 17:00より

 

追加キャラクター

今回「★5キャラクター専用ぶき」が追加されるキャラクターは
以下のキャラクターです。

 

最終進化まで同時実装！

今回追加される★5キャラクター専用ぶきは最終進化まで同時に実装いたします。
エトワリウムを集めて最終進化を目指そう！

 

3回進化で「自動発動スキル」解放！

★5キャラクター専用ぶきを3回進化させると「自動発動スキル」が解放されます。
「自動発動スキル」はバトル開始時から自動的に効果が発動します。
自動発動スキルは以下をご参照ください。

 

★5リゼ専用スピアー
通常攻撃のダメージが小アップ＋
通常攻撃が全体攻撃になる

 

最終進化で「メモリアルクエスト4節」追加＆ぶきスキル変化！

専用ぶき最終（4回）進化では「メモリアルクエスト4節」の開放と
同時に「ぶきスキル」が変化！
より強力な効果になると共に発動時の演出も特別なものに変化します！

 

★5リゼ専用スピアー
　★変化前のぶきスキル
　メモリア・ドライブ
　「敵単体」に自属性の中ダメージ +
　「自身」の物理攻撃が一定ターン中アップ
　★変化後のぶきスキル
　ミッドナイト・オールアウト
　「自身」の物理攻撃が一定ターン大アップ+
　「敵全体」に水属性の特大ダメージ

 

★5キャラクター専用ぶき入手＆強化方法

対象のキャラクターをLv.100にすることで、
各★5キャラクター専用のぶきが入手できます。

「★5キャラクター専用ぶき」は他のぶきと同様に「強化」が可能です。
「★5キャラクター専用ぶき」を「強化」していき、以下の条件を満たすと
「ぶき進化」ができるようになります。

① 「★5キャラクター専用ぶき」のレベルを上限まで上げる
② 対象のキャラクターに「★5キャラクター専用ぶき」を装備させ、
　バトルでぶきスキルを使用してぶきスキルを上限まで上げる
③ 「ぶき進化」に必要な「エトワリウム」などのアイテム、コインを集める

「ぶき進化」に必要なアイテムの1つ「エトワリウム」は、
主にイベントにてイベント素材を集めることで入手できます。
