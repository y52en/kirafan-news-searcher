06/28
【実施】★5キャラクター専用ぶき＆ぶき進化機能実装！

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

6月28日 17:00より、「★5キャラクター専用ぶき」機能をリリースいたします。

 

機能リリース日時

6月28日 17:00より

 

新機能「★5キャラクター専用ぶき」機能とは

「★5キャラクター専用ぶき」機能では、対象の★5キャラクターが特定の条件を満たすと「キャラクター専用ぶき」が入手できます。

 

6月28日 17:00に「専用ぶき」機能が実装されるキャラクターは、以下3キャラクターです。

 

 

対象のキャラクターをLv.100にすることで、各★5キャラクター専用のぶきが入手できます。

※★5キャラクター専用ぶきは、★4、★3の同じキャラクターに装備させることはできません。

 

 

「★5キャラクター専用ぶき」は他のぶきと同様に「強化」が可能です。ぶきを「強化」していき、特定の条件を満たすと「ぶき進化」ができるようになります。

「ぶき進化」には、イベントなどで入手できる新アイテム「エトワリウム」が必要になります。

「エトワリウム」は、トレードショップ＞限定交換にて入手可能です。

 

さらに、「★5キャラクター専用ぶき」実装に伴い、専用ぶき対象キャラクターのフルボイスシナリオが実装された「メモリアルクエスト」が開放されます！

今回「メモリアルクエスト」が開放されるのは、専用ぶき機能が実装された3キャラクターです。

各キャラクターのフルボイスシナリオについては、クエスト＞メモリアルクエストから選択できます。

 

今回は各キャラクターの「メモリアルクエスト」のうち1，2節が開放されます。

※3節以降は順次リリース予定となりますので、今しばらくお待ち下さい。

 

メモリアルクエスト開放には、特定の条件を達成する必要があります。条件達成を目指して、キャラクターやぶきを強化させましょう。

※他の★5キャラクターの「メモリアルクエスト」も、各★5キャラクター専用ぶき実装に伴い、順次公開していく予定です。

 

注意事項

・機能の仕様や日程は、予告なく変更する可能性がございます。
