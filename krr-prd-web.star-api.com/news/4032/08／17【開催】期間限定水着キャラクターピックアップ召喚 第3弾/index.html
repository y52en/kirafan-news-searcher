08/17
【開催】期間限定水着キャラクターピックアップ召喚 第3弾

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

8月17日 17:00より、「期間限定水着キャラクターピックアップ召喚 第3弾」を開催いたします。

 

開催期間

8月17日 17:00 ～ 8月30日 11:59

 

ピックアップ対象キャラクター

・★5 紺【水着】　　　　（まほうつかい　風属性）

・★5 しずく【水着】　　（せんし　風属性）

・★5 なでしこ【水着】　（ナイト　風属性）

・★4 裕美音【水着】　　（せんし　月属性）

・★4 コトネ【水着】　　（まほうつかい　陽属性）

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

1日1回有償限定召喚

「期間限定水着キャラクターピックアップ召喚 第3弾」開催期間中、1日1回のみ、有償星彩石10個で1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、現在開催中のイベント「海の底の玉手箱」にてイベントボーナスの効果を持っています。

イベントボーナスの効果を持つキャラクターを編成すると、イベント「海の底の玉手箱」のクエストにおいて、イベント素材のドロップ数がアップします。


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

※「期間限定水着キャラクターピックアップ召喚 第3弾」では、水着キャラクターのうち、★5 紺【水着】、しずく【水着】、なでしこ【水着】、★4 裕美音【水着】、コトネ【水着】のみが出現、ピックアップ対象となります。

「期間限定水着キャラクターピックアップ召喚 第1弾」「期間限定水着キャラクターピックアップ召喚 第2弾」で出現する水着キャラクターは出現しませんのでご注意下さい。
