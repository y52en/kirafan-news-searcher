【開催】そうりょ限定キャラクター召喚（後半）

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

6月15日 17:00より、「そうりょ限定キャラクター召喚（後半）」を開催いたします。そうりょ限定キャラクター召喚（後半）は、そうりょのキャラクターのみが出現する特殊な召喚です。

 

開催期間

6月15日 17:00～ 6月22日 16:59

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

※「★5 ゆの（陽属性 そうりょ）」「★5 大宮 忍（炎属性 そうりょ）」「★5 棗 ノノ（風属性 そうりょ）」「★5 一之瀬 花名（水属性 そうりょ）」「★5 関谷 なる（陽属性 そうりょ）」は、本召喚での出現対象キャラクターに含まれておりません。ご注意ください。


1日1回有償限定召喚

そうりょ限定キャラクター召喚（後半）開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。
