【開催】イベント「Fairies Trick」


 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

3月28日 メンテナンス後より、「桜Trick」正式参戦に合わせた新イベント「Fairies Trick」を開催いたします。

 

 

開催期間

3月28日 メンテナンス後 ～ 4月9日 13:59

 

イベント内容

本イベントでは、ストーリーが楽しめ、イベント期間中様々なアイテムと交換可能な素材が手に入るシナリオクエスト、高難易度である代わりに、豪華な報酬が手に入る【強敵】クエスト・【超強敵】クエストをプレイできます。

また、【超強敵】クエストよりも更に手強い【極】クエストも登場します！

 

とめきち先生描き下ろしのオリジナルキャラクターが登場！

今回のシナリオにはとめきち先生描き下ろしのオリジナルキャラクター「オルバ・アモル」が登場します。
彼女たちの活躍にもご注目ください。

 

遊び方

シナリオクエストを進めることで、新たなストーリーを読むことができます。また、クエストクリアで手に入る素材を集めることで、トレードショップで様々なアイテムと交換できます！

 

 

イベントで入手できる各素材はトレードショップ＞限定交換＞「Fairies Trick」から以下のアイテムと交換可能です。

イベントボーナスについて

今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。

イベントボーナスを持ったキャラクターは、イベント「Fairies Trick」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

 

 



※「★4　高山 春香」「★4　園田 優」「★3　高山 春香」「★3　園田 優」は「Fairies Trick」キャラクターピックアップ召喚のピックアップ対象キャラクターに含まれておりません。ご注意ください。

 

豪華素材がドロップする「【強敵】クエスト」「【超強敵】クエスト」！

倒すことの困難な強敵モンスターの登場する「【強敵】クエスト」「【超強敵】クエスト」が、シナリオクエスト全クリアで開放されます。

【強敵】クエストに挑戦するためには、スタミナを消費する必要があります（「【超強敵】クエスト」とは異なり、「【強敵】クエスト」には挑戦券は必要ありません）。

【強敵】クエストをクリアすると、イベント素材と、まれに「【超強敵】Fairies Trick挑戦券」を入手することができます。

 

この「【超強敵】Fairies Trick挑戦券」を消費することで「【超強敵】クエスト」に挑戦することができ、見事【超強敵】クエストをクリアすると、イベント素材と、まれに★4限界突破素材を入手できます。

 

更に今回のイベントでは、【強敵】クエストと【超強敵】クエストの両クエストを、一人も撤退しないでクリアした場合、初回のみ「そうりょの果実×1個」を入手することができます。

パーティーを整えて、【強敵】クエスト・【超強敵】クエストに挑戦してみましょう。

 

※【強敵】クエスト・【超強敵】クエストの発生期間は3月28日 メンテナンス後 ～ 4月9日 13:59です。
※使用しない【超強敵】Fairies Trick挑戦券は、トレードショップ＞限定交換＞「Fairies Trick」から、イベント素材「妖精の首飾り」に交換することができます。

 

 

【極】クエスト開催！

更に、【超強敵】クエストよりも難易度が高い「【極】クエスト」が登場！

「【極】クエスト」は、イベント開始後の4月3日 17:00から開催されますが、参加するためには各シーズンのシナリオクエストを全てクリアしている必要があります。

【極】クエストに挑戦するためには、スタミナを消費する必要があります（「【超強敵】クエスト」とは異なり、「【極】クエスト」には挑戦券は必要ありません）。

【極】クエストでは、特定の条件でクリアすると報酬を入手することができます。

 

【極】クエストを初めてクリアした時：星彩石×40個

【極】クエストにおいて、一人も撤退しないで初めてクリアした時：そうりょの果実×2個

 

※【極】クエストをクリアした際には、アイテムのドロップはございませんのでご注意ください。

※【極】クエストの発生期間は4月3日 17:00 ～ 4月9日 13:59です。
