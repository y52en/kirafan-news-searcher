大沖先生描き下ろしマンガが読める！作家クエスト「からみねーしょん」

『きららファンタジア』をご利用いただき、誠にありがとうございます。
10月8日 17:00より、
作家クエスト第3弾「からみねーしょん」が追加されます。

 

作家クエストは常設のクエストとなり、追加以降いつでも遊ぶことができます。

 

「はるみねーしょん」参戦！

『きららファンタジア』に「はるみねーしょん」が参戦！ 細野 はるみが登場します。

また、「はるみねーしょん」のコンテンツ施設「すげえ学校」が追加されます。
「すげえ学校」は、作家クエスト「からみねーしょん」追加と同時に里で
建設可能になります。

 

作家クエスト第3弾は「はるみねーしょん」大沖先生！

 

作家クエスト第3弾として、「はるみねーしょん」の原作者、

大沖先生による描き下ろしマンガが読めるクエストが追加されます！

また、クエストの追加を記念して、大沖先生に特別インタビューを行いました。
開発裏話を伺っておりますので、ぜひご覧ください。

 

作家クエスト第3弾 追加日時

10月8日 17:00 ～ 
※第4弾以降も順次追加予定ですので楽しみにお待ちください！

 

「からみねーしょん」あらすじ

全32節からなるシナリオクエストは、全編マンガ形式となっており
大沖先生による描き下ろしマンガが楽しめます。

また、シナリオクエストを全てクリアすると
特別なルームアイテムを入手することができます。

 

収集クエストでアイテムを集めよう！

シナリオクエストを全てクリアした後は、
ぶき素材の収集がしやすい収集クエストや、
高難易度の【超強敵】クエスト、【極】クエストにも挑戦することができます。

クエストの詳細は、別途掲載のおしらせ
作家クエスト「からみねーしょん」遊び方をご確認ください。

 

ミッション追加！

作家クエスト第3弾「からみねーしょん」の追加を記念して

また、トータルミッションにもミッションが追加されます。
