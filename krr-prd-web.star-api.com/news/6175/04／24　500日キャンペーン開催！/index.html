リリース500日記念★5選べるチケット付き！有償限定10回召喚

開催期間：4月25日 17:00 ～ 5月7日 23:59


『きららファンタジア』リリース500日を記念して、
「リリース500日記念★5選べるチケット付き！有償限定10回召喚」を
開催いたします。

■内容

・期間中、お一人様一回限りの有償星彩石限定召喚となります。

・本召喚を実行することで、★5キャラクター1人を指定して仲間にすることができる
　「スペシャルコールチケット」が入手できます。

 

■スペシャルコールチケット召喚とは

・「スペシャルコールチケット召喚」とは、

　★5キャラクター1人を指定して仲間にすることができる召喚です。


・指定することのできる★5キャラクターについては、
　下記「スペシャルコールチケット指定対象キャラクター」を
　ご確認のうえ、「スペシャルコールチケット」をご使用ください。


・既に加入済みのキャラクターが存在する場合に、その加入済みのキャラクターを
　「スペシャルコールチケット召喚」で仲間にすると、
　そのキャラクターのクラスの限界突破素材の入手に加えて、
　最大覚醒していない場合はキャラクターが「覚醒」し、
　最大覚醒している場合は「輝星のかけら　金」を入手できます。


・4月27日より開催される
　「ゴールデンウィーク★5選べるチケット付き！有償限定10回召喚」でも
　「スペシャルコールチケット」を入手できます。
※本召喚と合わせて最大2枚「スペシャルコールチケット」の入手が可能です。

 

 

スペシャルコールチケット指定対象キャラクター

キャンペーン注意事項

・所持している星彩石の内訳は、召喚画面下部の表示、もしくはホーム画面右上の
　MENUボタン＞設定・その他ボタン＞購入履歴ボタンよりご確認いただけます。
・「リリース500日記念★5選べるチケット付き！有償限定10回召喚」は、
　有償星彩石400個で召喚可能です。無償星彩石では召喚できません。
・「リリース500日記念★5選べるチケット付き！有償限定10回召喚」
　の提供割合詳細は召喚画面＞提供割合からご確認ください。
・「リリース500日記念★5選べるチケット付き！有償限定10回召喚」では、
　ピックアップ対象キャラクターは存在せず、
　同一レアリティの他のキャラクターよりも提供割合が高く設定されている
　キャラクターは存在しません。ご注意ください。
・本召喚で入手できる「スペシャルコールチケット」の
　使用期限は2019年5月19日 23:59です。
　期限が過ぎますと「所持アイテム一覧」から自動的に削除され、
　使用不可となりますので、必ず期限内にご使用ください。
・本キャンペーンの対象以外の★5キャラクターは、
　「スペシャルコールチケット召喚」の召喚候補には追加されません。
　ご注意ください。
・10回召喚を行った際には、特別アイテム「星屑のかけら」は入手できますが、
　「スペシャルコールチケット」使用時の召喚では、
　特別アイテム「星屑のかけら」は入手できません。
 ・キャンペーン内容は予告なく変更になる場合がございます。ご了承ください。
