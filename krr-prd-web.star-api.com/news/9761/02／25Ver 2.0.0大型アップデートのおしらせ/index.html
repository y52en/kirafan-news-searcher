Ver 2.0.0大型アップデートのおしらせ

『きららファンタジア』をご利用いただき、誠にありがとうございます。

2月28日に、Ver 2.0.0大型アップデートを実施いたします。

大型アップデート内容

里の施設に「コンテンツルーム」が登場！

新機能「クリエクラフト」実装！

新きらら装備「クリエオーブ」追加！

「称号」が設定可能に！

大型アップデート その１

里に設置可能なコンテンツ施設に訪問できるようになります。
また、コンテンツ施設の中に各作品をイメージしたコンテンツルームが登場します。
初めは引越し直後のような内装ですが、
クリエクラフトという新機能で遊ぶことで徐々に改装され、
その作品ならではのルームが完成していきます。

※コンテンツ施設への訪問は、
　該当作品のキャラクターが1人以上加入している場合に
　里画面＞該当作品のコンテンツ施設を選択＞訪問するボタン もしくは、
　MENU＞作品リスト＞該当作品のボタン より可能です。

大型アップデート その２

クリエクラフトとは、キャラクターからの依頼をこなして
コンテンツルームを充実させていく新しい遊びです。

クリエクラフトの遊び方

作品ごとにクリエを一定量貯めることでキャラクターからの依頼が発生します。
クリエは、該当作品のキャラクターをパーティーに編成し
クエストをクリアすることで獲得できます。

キャラクターからの依頼を受注したら、
達成条件となっているクラフトクエストに挑戦しましょう。
クエストをクリアし、依頼を達成することでコンテンツルームが一部改装されます。
また、依頼の達成報酬として新きらら装備「クリエオーブ」や、
新アイテム「クラフトメダル」を入手することもできます。
次々に依頼をこなし、コンテンツルームの完成を目指しましょう。

詳細は、「クリエクラフトの遊び方」をご覧ください。

大型アップデート その３

きらら装備に作品ごとの「クリエオーブ」が追加されます。
クリエオーブは、各作品のクリエクラフトの1つ目の依頼を達成することで
入手できます。

クリエオーブの特徴

クリエオーブは、クリエクラフトを進めることでレベルが上がります。
一定レベルに到達するとスキルが追加で解放され、最大で3つのスキルが備わります。
なお、解放されたスキルは1バトル中に1回ずつ使用できます。

また、きらら装備に設定したクリエオーブと同作品のキャラクターが
自分のパーティーにいる場合、該当キャラクターの各種ステータスがアップします。

※クリエオーブ追加に伴い、クラスオーブの経験値が廃止され、
　初めから3つのスキルが使用できるようになります。

パーティーごとにきらら装備を設定しよう

パーティー編成画面できらら装備の切り替えができ、
パーティーごとに設定を保存できるようになります。
色々なパーティーを作成してバトルを楽しみましょう。

※仕様変更により、編成画面内の「きらら装備変更」のボタンはなくなります。

大型アップデート その４

ユーザー情報画面で称号が設定できるようになります。
称号には、キャラクターに関連したものや、
クエストに関連したものなど様々な種類があります。
おきにいりの称号を設定してみましょう。

称号は、
MENU＞設定・その他＞ユーザー情報＞称号の設定 もしくは、
ホーム画面下部＞ユーザー＞称号の設定 より設定できます。
また、称号の取得状況もこの画面で確認できます。

※「はじめてミッション」を完了していない場合、
　称号の設定はMENU＞設定・その他＞ユーザー情報＞称号の設定 からのみ行えます。
※既に取得条件を満たしている称号は
　Ver 2.0.0アップデートのメンテナンスにて付与対応を行いますが、
　対応完了までに数日かかる可能性があります。
※「きららファンタジア 2nd Anniversary Fes.開催記念クエスト
　『ランプの限界チャレンジ！』」を
　クリアした方へ贈られる特別な称号「ランプのおすみつき」も
　Ver 2.0.0アップデートのメンテナンスにて付与対応を行います。
　また、アイテム「ランプからの贈り物」は所持アイテム一覧から後日削除されます。
※既にクリア済みのクエストで取得可能な称号は、
　該当のクエストを再度クリアすることで取得できます。
