「幸腹グラフィティ」参戦決定！

『きららファンタジア』に、「幸腹グラフィティ」の新規参戦が決定しました！
また、2月13日より参戦記念イベント「勇者のグルメ～エトワリア食紀行～」を
開催予定です。

「幸腹グラフィティ」とは

「幸腹グラフィティ」は、芳文社「まんがタイムきららミラク」にて連載されていた
川井マコト先生による4コマ漫画作品です。

芳文社公式マンガアプリ『COMIC FUZ』にてキャンペーン開催！

「幸腹グラフィティ」の『きららファンタジア』参戦を記念して
芳文社公式マンガアプリ『COMIC FUZ』にて、
原作コミックス第1巻～第3巻を無料で読めるキャンペーンが開催されています。
未読の方は、この機会に是非お読みください！

開催期間

2月7日 0:00 ～ 2月27日 14:00

キャンペーンの内容は、予告なく変更になる場合があります。ご了承ください。
