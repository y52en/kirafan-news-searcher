イベントで素材をGETしてトレードショップで交換しよう！

「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。

★5キャラクター専用ぶきの進化に必要なエトワリウムを入手しましょう。
 
その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「復刻・スティーレ2号店オープン！」からご確認下さい。
 
また、イベント素材の交換期限は10月31日 13:59までとなっておりますので
交換期限までに交換をお願いいたします。

「オススメスイーツセット」と交換できます。

 「オススメスイーツセット」は、対応する作品の★5キャラクターなら
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

【超強敵】クエストに挑戦するためには、
「【超強敵】挑戦券 」が必要です。
※過去に開催した「スティーレ2号店オープン！」で入手した
名称/デザインが同一の「【超強敵】挑戦券 」は使用できませんのでご注意下さい。

開放期間：
10月18日 17:00 ～ 10月24日 13:59

 

※使用しない【超強敵】挑戦券は、
　トレードショップ＞限定交換＞「復刻・スティーレ2号店オープン！」から
　「ブレンドクッキー」に交換することができます。

【極】クエストに挑戦ができます。挑戦するためにはスタミナを消費します。

 

開放期間：
10月19日 17:00 ～ 10月24日 13:59
