【開催】「異世界でもうさぎですか？」ピックアップ召喚


『きららファンタジア』をご利用いただき、誠にありがとうございます。

11月6日 16:00より、 「異世界でもうさぎですか？ピックアップ召喚・ココア編」
「異世界でもうさぎですか？ピックアップ召喚・チノ編」を開催いたします。

 

開催期間

11月6日 16:00 ～ 11月21日 13:59

 

 

2つの召喚が同時開催！

今回は「ココア編」と「チノ編」の2つの召喚を同時開催！
ぞれぞれピックアップされるキャラクターが異なり
自分の好みのキャラクターを狙って召喚がしやすくなっております。

 

「ココア編」ピックアップ対象キャラクター

★5 ココア（せんし　風属性）
★4 チノ（まほうつかい　水属性）

 

「チノ編」ピックアップ対象キャラクター

★5 チノ（まほうつかい　陽属性）
★4 チノ（まほうつかい　水属性）

 

※「ココア編」「チノ編」でピックアップされる、
　★4 チノ（まほうつかい　水属性）は同一のキャラクターです。

 

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「異世界でもうさぎですか？ピックアップ召喚」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、11月6日 16:00から開催されるイベント
「異世界でもうさぎですか？」にてイベントボーナス効果を持っています。

イベントボーナス効果を持つキャラクターを編成すると、
イベント「異世界でもうさぎですか？」のクエストにおいて、
イベント素材のドロップ数がアップします。

イベントボーナスに関する詳細は、
イベント「異世界でもうさぎですか？」のイベント詳細をご確認下さい。
