【開催】イベント「チョコレートクエスト」

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。
2月6日 メンテナンス後より、新イベント「チョコレートクエスト」を開催いたします。

 

 

開催期間

2月6日 メンテナンス後 ～ 2月16日 13:59

 

イベント内容

 

 

本イベントでは、ストーリーが楽しめるシナリオクエストと、イベント期間中様々なアイテムと交換可能な素材が手に入る素材収集クエスト、高難易度である代わりに、豪華な報酬が手に入る乱戦クエストをプレイできます。

 

 

遊び方

 

シーズン開放スケジュール：
シーズン1：2月6日   メンテナンス後 ～  2月16日 13:59
シーズン2：2月8日   17:00 ～  2月16日 13:59
シーズン3：2月10日 17:00 ～  2月16日 13:59


イベントは全部で3つのシーズンで構成されています。
シーズンは日ごとに開放され、そのシーズンのシナリオクエストを進めることで新たなストーリーを読むことができます。
また、クエストクリアで手に入る素材を集めることで、トレードショップで様々なアイテムと交換できます！

 

 

 

全シーズン共通の「バレンタインチョコレート」と、各シーズンのクエストでしか入手できない「ラッピングリボン」「バレンタインシール」「バレンタインボックス」を集めて豪華アイテムを手に入れましょう。

 

 

交換可能期間：
2月6日 メンテナンス後 ～ 2月23日 13:59

新機能：イベントボーナスについて




今回のイベントでは、イベントボーナスの効果を持ったキャラクターが存在します。
イベントボーナスを持ったキャラクターは、イベント「チョコレートクエスト」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。


※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。


イベントボーナス対象キャラクター：
★5　綾【バレンタイン】　　※期間限定ピックアップ召喚にて入手可能
※イベントボーナス：イベント「チョコレートクエスト」において、「バレンタインチョコレート」「ラッピングリボン」「バレンタインシール」「バレンタインボックス」のドロップ数＋１

★5　椎奈【バレンタイン】　※期間限定ピックアップ召喚にて入手可能
※イベントボーナス：イベント「チョコレートクエスト」において、「バレンタインチョコレート」「ラッピングリボン」「バレンタインシール」「バレンタインボックス」のドロップ数＋１

★4　ツバメ【バレンタイン】※期間限定ピックアップ召喚にて入手可能
※イベントボーナス：イベント「チョコレートクエスト」において、「ラッピングリボン」「バレンタインシール」「バレンタインボックス」のドロップ数＋１

★4　千矢【バレンタイン】　※イベント「チョコレートクエスト」の限定交換にて交換可能
※イベントボーナス：イベント「チョコレートクエスト」において、「バレンタインチョコレート」のドロップ数＋１

 

 

イベントボーナスの効果が発動しているキャラクターは、「クエスト選択時のキャラクター選択画面」「クエスト選択時の編成画面」「出撃パーティー確定画面」にて、「BONUS」の表示がされ、ドロップ数がアップする対象イベント素材とアップする数値を確認することができます。

また、クエスト終了時のリザルトポップアップの獲得アイテム画像では、「×入手したイベント素材の個数（＋イベントボーナスとしてアップした獲得個数）」と、イベントボーナスでの増加分が表示されるようになります。

 

新クエスト：豪華素材が手に入る「乱戦クエスト」！

 

倒すことが困難な強敵モンスターが複数登場する「乱戦クエスト」が登場！
「乱戦クエスト」は、各シーズンのシナリオクエストを全てクリアすると開放されます。


「乱戦クエスト」に挑戦するためには、「乱戦チケット」を消費する必要があります。乱戦チケットは、イベント開催中のイベントミッションから入手することができます。

 

 

 

乱戦クエストをクリアすると、イベント共通の素材と、まれに★4限界突破素材を入手することができます。
その他、特定の条件でクリアすると追加で報酬を入手することができます。



乱戦クエストを初めてクリアした時：星彩石×40個
乱戦クエストにおいて、一人も撤退しないで初めてクリアした時：アルケミストの果実×1個

 

パーティーを整えて、乱戦クエストに挑戦してみましょう。

※乱戦クエストの発生期間は2月10日 17:00 ～ 2月16日 13:59です。

イベントミッション開催！

イベント開催にあわせて、イベント素材や乱戦チケットを報酬としてもらえるイベントミッションを開催します。
イベントを有利に進めるために、ミッションクリアを目指しましょう！
