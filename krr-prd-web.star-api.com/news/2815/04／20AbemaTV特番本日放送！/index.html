AbemaTV特番本日放送！

 

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

4月20日（金）22:00よりAbemaTVウルトラゲームスチャンネルにて、楠木ともりさん、高野麻里佳さんらが出演する「みんなで声優を脱出させよう！ 放課後！謎解きららファンタジア」を放送いたします。

 

ゲスト出演する内山夕実さん、田村睦心さん、山口愛さん、和氣あず未さんがそれぞれ部屋に閉じ込められてしまいます。

 

視聴者と一緒に謎を解いて見事声優たちを脱出成功へと導くことができれば、1人脱出するごとに『きららファンタジア』の初出し情報を番組内で発表いたします。

詳しくは公式サイトのニュースをご覧ください。
