イベント「三者三葉ぶらり旅」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「特産きゅうり」や
「マヨネーズ」が入手できます。

イベントで手に入る素材は★4 西川 葉子（まほうつかい　月属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。

その他のイベント報酬については、
トレードショップ画面＞限定交換内の 「三者三葉ぶらり旅」からご確認ください。

また、イベント素材の交換期限は4月17日 13:59までとなっておりますので
期限までに交換をお願いいたします。

限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。

「抽選マシーン」と交換できます。

 「抽選マシーン」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。
