Ver1.13.0での改修予定箇所について

『きららファンタジア』をご利用いただき、誠にありがとうございます。

Ver1.13.0アップデートにおいて、
実装予定の各機能改善内容についてご案内いたします。

 

機能アップデート

・メインシナリオ2章まで フルボイスアップデート

　メインシナリオのうち、以下をフルボイス仕様へアップデートいたします。

　・プロローグ「はるかなる旅立ち」
　・1章「この世界でも、笑顔を」（ひだまりスケッチ）
　・2章「続きから▶港町」（NEW GAME!）

　3章以降につきましても順次フルボイス仕様へ
　アップデート予定ですのでお楽しみに！


・ログインボーナスの改修
　報酬内容の調整及び最大累積14日間のログインボーナスへ改修いたします。
　※12/10に実施予定のメンテナンス開始までにログインしている方につきましては、
　　メンテナンス終了後に再度ログインする事で
　　改修後の「ログインボーナス1日目」の報酬が受け取れます。
　※ログインボーナスリセットに伴う補填といたしまして、
　　メンテナンス開始までにゲームを開始していたすべてのお客様へ
　　無償星彩石20個をプレゼントリストへ配布いたします。


・ユーザーランク上限解放
　ユーザーランクを最大100から最大120まで解放いたします。
　また、既にユーザーランク100に達しているお客様については
　アップデート後、ユーザーランク100（経験値ゼロ）の状態からの
　スタートとなります。

・デイリー、ウィークリーミッションの追加
　各ミッションの追加と変更及び、報酬の個数や内容を調整いたします。
　※デイリー、ウィークリーミッション追加、改修に伴い、
　　メンテナンス開始までにゲームを開始していたすべてのお客様へ
　　無償星彩石60個をプレゼントリストへ配布いたします。

・はじめてミッションの実装
　ゲームを始めたばかりの方向けのミッションを実装いたします。
　すべてのミッションクリアで10回召喚1回分の無償星彩石400個が入手できます。
　※既にゲームをプレイしている方も対象です。

・召喚関連機能改修
　無料召喚機能など召喚に関する機能を実装いたします。

・細かな機能改修
　・バトル中のAUTO解除方法を画面全体からオートボタンのみに変更
　・サポート編成で同作品のキャラクターを設定できるよう変更
　・前回のメインクエストプレイ位置を記憶するよう変更
　・強化、進化、限界突破時の演出改修
　・その他、機能改修や今後実装予定のスキル効果実装など
