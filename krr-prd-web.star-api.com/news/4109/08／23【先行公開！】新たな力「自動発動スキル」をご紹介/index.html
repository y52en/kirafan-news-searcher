【先行公開！】新たな力「自動発動スキル」をご紹介
『きららファンタジア』をご利用いただき、誠にありがとうございます。

本日は★5キャラクター専用ぶきを3回進化させると
解放される「自動発動スキル」の情報と、
次回追加予定の★5キャラクター専用ぶき情報を先行して少しだけ公開いたします。

「自動発動スキル」を解放して新たな力を手に入れよう！


★5 ゆの（そうりょ　陽属性）は使用する回復スキルが、
最大HPの200%まで回復できるようになるため
事前に回復スキルを使用してHPを増加させるなど、より戦略の幅が広がります！

★5 カレン（せんし　月属性）は通常攻撃の攻撃力アップに加えて、
クリティカルダメージ量が上昇しアタッカーとしてより強力に！

また、他のキャラクターの自動発動スキルでは、
常時狙われやすさアップや状態異常無効、
与えたダメージに応じて自身のHPを回復など強力な効果ばかり。

自動発動スキルは★5キャラクター専用ぶきが実装済みの
すべてのキャラクターで解放可能ですので、
★5キャラクター専用ぶきの3回進化を目指してみましょう！
※各キャラクターの自動発動スキル一覧はアップデートおしらせで公開予定です。
