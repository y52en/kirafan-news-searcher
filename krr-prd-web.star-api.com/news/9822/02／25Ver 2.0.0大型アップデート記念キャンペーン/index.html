Ver 2.0記念★5選べるチケット付き！有償限定10回召喚

開催期間中、1回限り行える有償星彩石限定の召喚です。
「10回召喚」を行うことで「スペシャルコールチケット」を入手でき、
召喚画面に「スペシャルコールチケット召喚」が出現します。

「スペシャルコールチケット召喚」では、
好きな★5キャラクター1人を指定して仲間にすることができます。
指定対象キャラクターは下記をご確認ください。

■開催期間
2月28日 17:00 ～ 3月12日 23:59
※「スペシャルコールチケット」の使用期限：3月19日 23:59

■注意事項

・本召喚は、有償星彩石400個で行えます。無償星彩石では行えません。
・所持している星彩石の内訳は、召喚画面下部の表示 または、
　MENU＞設定・その他＞購入履歴 よりご確認ください。
・2月18日より開催の
　「リリース800日記念★5選べるチケット付き！有償限定10回召喚」でも
　「スペシャルコールチケット」を入手でき、
　本召喚と合わせて最大2枚「スペシャルコールチケット」を入手できます。
・本召喚で入手できる「スペシャルコールチケット」の
　使用期限は2020年3月19日 23:59です。
　期限が過ぎますと「所持アイテム一覧」から自動的に削除され、
　使用不可となりますので、必ず期限内にご使用ください。
・本召喚の提供割合は、
　MENU＞召喚＞本召喚のバナー＞提供割合 よりご確認ください。
・本召喚では、ピックアップ対象キャラクターは存在しません。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・「10回召喚」では「チャンスアップキーホルダー」を使用できますが、
　「スペシャルコールチケット召喚」では使用できません。
・「10回召喚」では「星屑のかけら」を入手できますが、
　「スペシャルコールチケット召喚」では入手できません。
・「スペシャルコールチケット召喚」で既に加入済みのキャラクターを仲間にすると、
　そのキャラクターのクラスの限界突破素材の入手に加えて
　最大覚醒していない場合はキャラクターが覚醒し、
　最大覚醒している場合は「輝星のかけら　金」を入手できます。

1日1回きらきら召喚

開催期間中、1日1回行える有償星彩石限定の召喚です。
有償星彩石40個を消費して「3回召喚」を行うと、
いずれの3体も★4以上確定で召喚できます。

■開催期間
2月28日 17:00 ～ 3月6日 23:59

■注意事項
・本召喚は、有償星彩石40個で行えます。無償星彩石では行えません。
・所持している星彩石の内訳は、召喚画面下部の表示 または、
　MENU＞設定・その他＞購入履歴 よりご確認ください。
・本召喚には「1回召喚」や「10回召喚」が存在せず、「3回召喚」のみとなります。
・本召喚を行った際に「星屑のかけら」を3個入手できます。
・本召喚で出現するキャラクターの詳細は、
　MENU＞召喚＞本召喚のバナー＞出現一覧 よりご確認ください。
・本召喚では、ピックアップ対象キャラクターは存在しません。
・本召喚は、1日1回限りとなっております。
　召喚を行うと召喚一覧から本召喚が消え、翌日0時に再度出現します。
