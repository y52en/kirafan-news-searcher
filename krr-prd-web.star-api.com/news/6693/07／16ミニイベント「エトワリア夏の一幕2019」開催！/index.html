ミニイベント「エトワリア夏の一幕2019」開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。
7月18日 0:00より、ミニイベント「エトワリア夏の一幕2019」を
開催いたします。

 

開催期間

7月18日 0:00 ～ 7月24日 23:59

 

イベント概要

本イベントは、全5話からなるシナリオクエストと
各属性の新芽・大種などの強化素材や交換メダルが獲得できる

収集クエスト・【乱戦】クエストで構成されています。

シナリオクエストでは、7月24日より開催予定となっている
イベント「リュウグウ・アドベンチャー」の前日譚を楽しむことができます。


シナリオは1日1話公開され、シナリオが公開されると同時に
公式Twitterにてシナリオに登場するキャラクターの新規水着イラスト

が公開されます！是非ご覧ください。

 

■シナリオクエスト追加スケジュール

1話追加：7月18日 0:00 ～ 7月24日 23:59 
2話追加：7月19日 0:00 ～ 7月24日 23:59 
3話追加：7月20日 0:00 ～ 7月24日 23:59 
4話追加：7月21日 0:00 ～ 7月24日 23:59 
5話追加：7月22日 0:00 ～ 7月24日 23:59 

 

また、シナリオクエスト1話目をクリアすることで、
収集クエスト・【乱戦】クエストが開放されます。

※「エトワリア夏の一幕2019」の各シナリオを読むことで、

　星彩石×5個を獲得できます。

 

 収集クエストで素材を集めよう！

収集クエストでは、新芽や大種といった強化アイテムの他
「交換メダル」がドロップします

交換メダルは「【乱戦】挑戦券」と交換できる他、
トレードショップにて特別な育成セットと交換できます

 

ドロップする素材
炎の新芽
水の新芽
風の新芽
土の新芽
陽の新芽
月の新芽
炎の大種
水の大種
風の大種
土の大種
陽の大種
月の大種
交換メダル
【乱戦】挑戦券

 

さらなる挑戦！「【乱戦】クエスト」開催！

【乱戦】クエストに挑戦するためには、
専用の「【乱戦】挑戦券 」が必要です。

※使用しない「【乱戦】挑戦券」は、
　トレードショップ＞限定交換＞「エトワリア夏の一幕2019」から
　「交換メダル×3」に交換することができます。

交換可能期間：7月18日 0:00 ～ 7月31日 13:59

 

トレードショップにお得な進化セット追加！

期間限定で、トレードショップに各属性・各クラスのキャラクターの

進化に必要なアイテムがセットになったお得な進化セットを追加いたします。

 

▼お得なクラス進化セット 例

 ★5あるいは★4キャラクターの進化に必要な量の
各クラスのスタチューと限界突破素材が詰まったセットです。
交換に必要なアイテム：交換メダル×300
限度数：1　

▼お得な属性進化セット 例

★5あるいは★4キャラクターの進化に必要な
各属性の進化珠と強化素材が詰まったセットです。
交換に必要なアイテム：交換メダル×100
限度数：1　

交換可能期間：7月18日 0:00 ～ 7月24日 23:59
※クラス用進化セットは
　全5種（せんし・まほうつかい・そうりょ・ナイト・アルケミスト）が登場します
※属性用進化セットは、全6種（炎・風・土・水・月・陽）が登場します
※期間中、各セットにつき1回限り交換可能です
