今日から始めるエトワリアの世界の歩き方

『きららファンタジア』を始めたばかりのみなさまに、
エトワリアの世界の歩き方（遊び方）をご紹介！

「どのクエストを遊んだらいいかわからない…」
「強い敵が出てきて負けてしまったけどどうしたら…」

など困った時に参考にしてみてください！

まずはメインクエストに挑戦しましょう！

メインクエストでは「まんがタイムきらら」の
各作品キャラクターたちがエトワリアの世界を舞台に大活躍！


それぞれ『きららファンタジア』だけでしか読めないオリジナルシナリオ。
きららやランプたちと一緒に世界を救う冒険にでかけましょう！

メインクエストの中ではバトルが発生することも…
出撃前にエネミーの属性を確認してパーティーを編成してみましょう！


他にもクラスがそうりょのキャラクターでHP回復を使ってみたり
ナイトで自分の狙われやすさをあげて他のキャラクターへ
攻撃がいかないようにしてみたりなど
キャラクターのクラスの組み合わせを工夫して編成することが重要です！

それでも勝てない強敵が現れたらキャラクターの育成をしてみましょう！

★ポイント
バトルは画面右上の「AUTOボタン」（巻き矢印のボタン）をタップすることで
自動戦闘に切り替えられます！
強い敵が出現するクエストでは画面をタップして自動戦闘から手動に切り替えて
とっておきやスキルを使うタイミングを工夫してみましょう！

 

 

キャラクターを育成しましょう！

キャラクターの育成の最初の一歩はキャラクターのレベル上げです。
レベルアップする方法は以下の3通りがあります。

・パーティーに編成してクエストをクリアする
・強化素材を使って強化を行う
・トレーニングを行う


また、限界突破素材を消費することで限界突破ができレベル上限が上昇します。

限界突破素材はキャラクターを重複して入手した際に手に入るほか、
トレードショップやイベントの報酬でも入手ができます。

さらにレアリティが★4以上のキャラクターは
限界突破を4回すると進化ができます！


レベルアップ以外にもキャラクターのステータスは
なかよし度をあげたり里の施設を強化することでも上昇します。

キャラクターを強化、進化して、高難易度のクエストにも挑戦してみましょう！

★ポイント
バトル中に右下のキャラクターを右フリックして使用できる
「きららスキル」は活用していますか？
戦況を変える一発逆転のチャンスを作ることができるかもしれません！

 

 

イベントクエストに挑戦しましょう！

イベントクエストでも色々な「まんがタイムきらら」の
キャラクターたちがエトワリアの世界を大冒険！
ここでしか見られないキャラクターたちの意外な一面も見られるかもしれません…！？

また、イベント参戦作品のイベント限定★4キャラクターや
進化・限界突破素材アイテム、キャラクター召喚に使える星彩石、
ぶき進化に必要なエトワリウムなどを入手できます。

▼イベントでもらえるアイテムの一例

シナリオを楽しみながらお気に入りのキャラクターをより強くしていこう！

さらなる試練や強さを求めて…

より強くなるにはスキルレベルやとっておきの強化、ぶきの強化、
★５キャラクター専用ぶき入手やぶき進化などに挑戦してみましょう。

より強くなったキャラクターたちで
★5キャラクターの限界突破素材と交換できるメダルが手に入る
毎週土・日曜日に開催されている強敵チャレンジクエスト、
星彩石や限界突破素材が手に入るイベントクエスト内の乱戦クエストや極クエスト、
超強敵クエストなどにも挑戦してみましょう！
