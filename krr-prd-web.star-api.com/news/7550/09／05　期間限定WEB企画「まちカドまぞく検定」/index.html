　期間限定WEB企画「まちカドまぞく検定」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

「まちカドまぞく」の参戦を記念して、
期間限定WEB企画「まちカドまぞく検定」を実施いたします♪

実施期間

9月5日12:00 ～ 9月9日23:59

概要

「まちカドまぞく」原作コミックス1巻のエピソードから出題される
クイズに挑戦しましょう！

クイズは全20問のうち毎回ランダムで4問が出題され、
全問正解すると★5シャミ子のイラストを見ることができます。

さらに、キャンペーンページ上の「みんなの合計ポイント」が
目標ポイントを超えると、「まちカドまぞく」の参戦日（9月11日）に
『きららファンタジア』にて豪華報酬が貰えます！

なお、出題範囲の「まちカドまぞく」原作コミックス1巻は、
芳文社公式マンガアプリ『COMIC FUZ』で期間限定無料公開中です♪
2巻・3巻もイッキ読みキャンペーンを実施中ですので、
どんどん先まで読み進めましょう！
