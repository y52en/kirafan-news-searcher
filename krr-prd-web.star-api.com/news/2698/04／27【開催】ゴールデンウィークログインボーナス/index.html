【開催】ゴールデンウィークログインボーナス


『きららファンタジア』をご利用いただき、誠にありがとうございます。

4月27日より、通常開催しておりますログインボーナスに加え、ゴールデンウィークログインボーナスを開催いたします。

ゴールデンウィークログインボーナスでは、下記の開催期間のうち、10日間ログインすることで、星彩石を合計300個入手することができます。

 

 

ゴールデンウィークログインボーナス

開催期間：4月27日 0:00 ～ 5月7日 23:59

 

●ログインボーナス内容

1日目　           星彩石×80個
2日目　　　　  星彩石×40個
3日目～9日目　星彩石×20個
10日目　　　　星彩石×40個
