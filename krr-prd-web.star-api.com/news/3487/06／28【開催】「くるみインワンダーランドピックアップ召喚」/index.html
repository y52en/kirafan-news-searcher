【開催】「くるみインワンダーランドピックアップ召喚」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

6月28日 17:00より、「くるみインワンダーランドピックアップ召喚」を開催いたします。

 

召喚前半期間には、「がっこうぐらし！」より★5 恵飛須沢 胡桃（CV.小澤亜李）の出現率がアップ！
召喚後半期間には、『きららファンタジア』に初登場のあの作品のキャラクターの出現率がアップ！

 

開催期間

開催期間：6月28日 17:00 ～ 7月11日 11:59
※ピックアップ召喚前半と後半で、ピックアップ対象が異なりますのでご注意下さい。

 

■ピックアップ召喚前半：6月28日 17:00 ～ 7月11日 11:59
※ピックアップ召喚後半開始後も、ピックアップ召喚前半は継続して実施されます。

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。
※ピックアップ召喚後半も、初回のみ星彩石300個で『10回召喚』が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「くるみインワンダーランドピックアップ召喚」開催期間中につき、1日1回のみ、有償星彩石10個を使用することで1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、6月28日 17:00から開催されるイベント「くるみインワンダーランド」にてイベントボーナスの効果を持っています。

イベントボーナスを持ったキャラクターは、イベント「くるみインワンダーランド」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「くるみインワンダーランド」のおしらせをご確認下さい。


※★4アリス・カータレット（まほうつかい　炎属性）は、イベント「くるみインワンダーランド」の期間中のみ仲間にできるキャラクターです。イベント中に「サイコロ 」を集めることで、交換可能になります。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。
