07/23
【開催】「けいおん！　エトワリアライブ！！ピックアップ召喚」

『きららファンタジア』をご利用いただき、誠にありがとうございます。

7月23日 15:00 16:00 より、「けいおん！　エトワリアライブ！！ピックアップ召喚」を開催いたします。

 

開催期間

7月23日 15:00 16:00 ～ 8月3日 16:59

※メンテナンス終了時刻延長のため開始時刻を変更いたしました。（7/23 15:00追記）

 

ピックアップ対象キャラクター

・★5＆★3 平沢 唯　（CV.豊崎愛生）

・★4＆★3 中野 梓　（CV.竹達彩奈）

・★4＆★3 秋山 澪　（CV.日笠陽子）

・★4＆★3 田井中 律（CV.佐藤聡美）

・★4＆★3 琴吹 紬　（CV.寿美菜子）

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

・召喚を行った際に、特別アイテム「星屑のかけら」を追加で入手することができます。（1回召喚で1個、10回召喚で10個入手できます）

・本召喚で出現するキャラクターの詳細につきましては、ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「けいおん！　エトワリアライブ！！ピックアップ召喚」開催期間中、1日1回のみ、有償星彩石10個で1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。

※通常の1回召喚、10回召喚では、無償星彩石から使用されます。

※1日1回有償限定対応の召喚が複数同時に開催されている場合、それぞれの召喚につき1回までご利用いただけます。

 

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、7月23日 15:00 16:00から開催されるイベント「けいおん！　エトワリアライブ！！」にてイベントボーナスの効果を持っています。

イベントボーナスを持ったキャラクターは、イベント「けいおん！　エトワリアライブ！！」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。
イベントボーナスに関する詳細は、イベント「けいおん！　エトワリアライブ！！」のおしらせをご確認下さい。

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

※★4 平沢 唯（せんし　土属性）は、イベント「けいおん！　エトワリアライブ！！」の期間中に仲間にできるキャラクターです。イベント中に「ティーカップ 」を集めることで、交換できます。
