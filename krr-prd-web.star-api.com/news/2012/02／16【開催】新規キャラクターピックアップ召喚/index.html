【開催】新規キャラクターピックアップ召喚

『きららファンタジア』をご利用いただき、誠にありがとうございます。

2月16日 メンテナンス後より、「新規キャラクターピックアップ召喚」を開催いたします。

 

「夢喰いメリー」よりメリー・ナイトメア（CV.佐倉綾音）と橘 勇魚（CV.茅野愛衣）が『きららファンタジア』に登場！

 

 

「夢喰いメリー」が新規参戦！


『きららファンタジア』に、「夢喰いメリー」が新規参戦！
メリー・勇魚が新規キャラクターピックアップ召喚に登場します。

 

さらに、「夢喰いメリー」のコンテンツ施設「夢と幻想の境界」が追加されます。
「夢と幻想の境界」は新規キャラクターピックアップ召喚と同時に里で建設可能になります。

 

開催期間

2月16日 メンテナンス後 ～ 2月27日 13:59

 

内容

・「10回召喚」では★4以上のキャラクターが1体以上確定します。

・初回のみ星彩石300個で「10回召喚」が可能です。

・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。

 

今回のピックアップ対象キャラクターは、イベントボーナス効果持ち！

今回のピックアップ対象キャラクターは、2月19日メンテナンス後から開催されるイベント「鍛冶師の夢を喰らう者」にてイベントボーナスの効果を持っています。

 

イベントボーナスを持ったキャラクターは、イベント「鍛冶師の夢を喰らう者」のクエストにおいて、イベント素材のドロップ数アップのボーナスが付与されます。

 

※イベントボーナスの効果は、サブメンバー・サポート枠に編成されたキャラクターからも受けることができます。

★5　メリー・ナイトメア
★4　メリー・ナイトメア

★3　メリー・ナイトメア
