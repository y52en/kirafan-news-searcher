【開催】期間限定2017クリスマスキャラクターピックアップ召喚


『きららファンタジア』をご利用いただき、誠にありがとうございます。

11月30日 17:00より、「期間限定2017クリスマスキャラクターピックアップ召喚」を
開催いたします。

 

また、近日中に「過去に登場した期間限定★5キャラクター」を選んで召喚できる
チケットが付いた有償限定10回召喚を開催予定です！
こちらの召喚は開催期間中1回のみ行うことができる特別な召喚です。
詳細につきましては後日掲載のおしらせをご確認ください。【11月30日16:45追記】

 

開催期間

11月30日 17:00 ～ 12月10日 23:59

 

 

ピックアップ対象キャラクター

★5 ユー子【クリスマス】（ナイト　月属性）
★5 アリス【クリスマス】（まほうつかい　土属性）
★5 はじめ【クリスマス】（せんし　水属性）
★5 千矢【クリスマス】（そうりょ　風属性）
★4 るん【クリスマス】（アルケミスト　水属性）
★4 たまき【クリスマス】（そうりょ　風属性）

 

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に特別アイテム「星屑のかけら」を追加で入手することができます。
　（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　ゲーム内の召喚画面＞出現一覧からご確認下さい。

 

 

1日1回有償限定召喚

「期間限定2017クリスマスキャラクターピックアップ召喚」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。

 

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。
