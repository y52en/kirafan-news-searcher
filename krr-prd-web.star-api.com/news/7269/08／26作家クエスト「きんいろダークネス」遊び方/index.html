作家クエスト「きんいろダークネス」遊び方

原悠衣先生描き下ろしマンガを楽しもう！

作家クエスト「きんいろダークネス」では、

「きんいろモザイク」原悠衣先生による描き下ろしマンガを読むことができます。

原悠衣先生描き下ろし４コママンガは全編フルカラーとなっております。

 

8月29日より原悠衣先生による開発裏話を公開予定！

描き下ろしマンガへのこだわりなど、様々な開発裏話を伺っております。

ぜひご覧ください！

 

 

収集クエストでぶき素材や【超強敵】チケットを集めよう！

シナリオクエストをクリアすると、収集クエスト・【超強敵】クエスト

【極】クエストが開放されます。

収集クエストでは、まれに【超強敵】クエストに挑戦するために

必要な専用の挑戦券がドロップします。

収集クエストをクリアして、【超強敵】クエストに挑戦しましょう！

 

■ぶき素材ドロップ内容一覧
・木の根

・光輝岩

・硬い石

・古木の小枝

・太古の骨

・黄金の枝

・銀鉱石

・古の岩石
etc.

 

豪華素材がドロップする「【超強敵】クエスト」開催！

【超強敵】クエストではぶき強化素材がドロップする他、

まれに「エトワリウムのかけら」がドロップします。

「エトワリウムのかけら」を4つ集めると、トレードショップの通常交換で
★5キャラクター専用ぶきの進化に必要な「エトワリウム」と交換できます。

【超強敵】クエストをクリアし、ぶきを強化しましょう！

 

さらなる挑戦！「【極】クエスト」！

【超強敵】クエストをクリアしたら
さらなる高難易度【極】クエストに挑戦してみましょう！
【極】クエストに挑戦するためにはスタミナを消費します。
