「はるみねーしょん」参戦決定！大沖先生による描き下ろし漫画も登場！

『きららファンタジア』に、「はるみねーしょん」の新規参戦が決定しました！
さらに、大沖先生による描き下ろし漫画が読める新形式のクエストが登場します！

「はるみねーしょん」とは

トボケた女子高生、細野はるみは宇宙人。
浮いたり飛んだり、時には土中に埋まったり、地球ライフを満喫中。
宇宙人の彼女にとっては地球での暮らしは
＼すげえ／で＼やべえ／なことがいっぱいです。
一度読むと止められない、シュールギャグ4コマ！

本日7月25日に、原作コミックス最終巻の8巻が発売されます。
この機会にぜひお読みください♪
原作は完結となりますが
大沖先生が描くはるみの異世界生活にご期待ください！

「はるみねーしょん」の細野はるみ参戦と、
大沖先生による漫画は2019年秋に実装予定です！
