「配信のすゝめ」が入手できます。


イベントで手に入る素材は★4 リゼ（ナイト　月属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。

その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「エトワリア冒険譚 前編 リゼ教官の秘密特訓」からご確認ください。

また、イベント素材の交換期限は3月19日 13:59までとなっておりますので
期限までに交換をお願いいたします。

ミッションクリアで集められます。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。

★4 リゼ（ナイト　月属性）に交換が可能です。

 

たくさん交換して、★4 リゼ（ナイト　月属性）を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで5枚、
　後日談シナリオ「特訓、その後」の各シナリオを読むことで5枚、
　合計10枚獲得することができます。

※★4 リゼ（ナイト　月属性）の覚醒は、最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。

※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

イベント素材「配信のすゝめ」を集めると、
「スタンプカード」と交換できます。


 「スタンプカード」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。
