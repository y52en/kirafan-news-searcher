　イベント「甘兎庵の野望」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「海に映る月と星々」や
「3羽の看板パン」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「甘兎庵の野望」からご確認ください。

また、イベント素材の交換期限は5月21日13:59までとなっておりますので
期限までに交換をお願いいたします。

 

イベント限定キャラクターを仲間にしよう！

イベント素材「3羽の看板パン」を100個集めると
★4 千夜（そうりょ　炎属性）を仲間にできます。

また、キャラクターを進化させることで
キャラクターのイラストも変化します。ぜひ進化させてみましょう。

キャラクターの進化は★4、★5のキャラクターが4回「限界突破」した後、
キャラクターのレベルが一定以上になると行えます。
限界突破に必要なアイテムやコインは、イベントクエストや
ミッションクリアで集められます。

「イベント限定コールチケット」をGETしてキャラクターを覚醒させよう！

イベントミッションをクリアすると「イベント限定コールチケット」が入手できます。


イベント限定コールチケットは、トレードショップで
★4 千夜（そうりょ　炎属性）に交換が可能です。

たくさん交換して、★4 千夜（そうりょ　炎属性）を覚醒させてみましょう！

 

※「イベント限定コールチケット」は、イベントミッションクリアで
　合計10枚獲得することができます。

※★4 千夜（そうりょ　炎属性）の覚醒は、最大4回まで可能となっており
　同一キャラクターを10回重複させることで最大覚醒となります。

※「イベント限定コールチケット」での交換は、
　チケットを複数枚所持していた場合でも1回ずつの交換となります。

 

 

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

 「お手製のテーブルセット」は、対応する作品の★5キャラクターの限界突破時に
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

 

イベントボーナス効果について

イベントボーナス効果を持つキャラクターをパーティーに編成することで、
イベント「甘兎庵の野望」のクエストにおいて
イベント素材のドロップ数がアップします。
