【更新】★5キャラクター専用ぶき更新！（2019年3月②）

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

3月28日 17:00より
以下キャラクターの、専用ぶき最終進化、メモリアルクエスト4節が開放されました。

 

開放日時

3月28日 17:00～

 

対象キャラクター

・★5 ユー子【クリスマス】（ナイト）
・★5 志摩 リン（まほうつかい）
・★5 一之瀬 花名（そうりょ）

 

開放条件

対象キャラクターの専用ぶきを4回進化させる

 

最終進化で「メモリアルクエスト4節」追加＆ぶきスキル変化！

専用ぶき最終（4回）進化では「メモリアルクエスト4節」の開放と同時に
「ぶきスキル」が変化！
より強力な効果になると共に発動時の演出も特別なものに変化します。

 

・★5ユー子【クリスマス】専用スピアー
　★変化前のぶきスキル
　メモリア・エッジ
　　「敵単体」に自属性の中ダメージ +
　　「自身」の狙われやすさが一定ターン中アップ
　　
　★変化後のぶきスキル
　オバケイチゴのスペシャルケーキ
　　「自身」に四度だけ攻撃を完全にカットする
　　バリアを張る +「敵全体」に月属性の大ダメージ +
　　「自身」の狙われやすさが一定ターン大アップ

 

・★5志摩 リン専用クリスタル
　★変化前のぶきスキル
　メモリア・スペル
　　「敵全体」に自属性の中ダメージ +
　　「自身」の魔法攻撃が一定ターン中アップ

 

　★変化後のぶきスキル
　職業：異世界キャンパー
　　「自身」の魔法攻撃が一定ターン大アップ +
　　「敵全体」に水属性の特大ダメージ

 

・★5一之瀬 花名専用ロッド
　★変化前のぶきスキル
　メモリア・ブースト
　　「味方単体」の物理防御と魔法防御が
　　一定ターン小アップ +「味方単体」の
　　物理攻撃と魔法攻撃が一定ターン小アップ
　　
　★変化後のぶきスキル
　秘めた想いを胸に
　　「味方全体」に一度だけ攻撃をすこしカットする
　　バリアを張るスキルカードを設置 +「味方全体」の
　　物理攻撃と魔法攻撃が一定ターン中アップ

 

注意事項

・機能の仕様や日程は、予告なく変更する可能性がございます。

　装備させることはできません。
・各キャラクターのフルボイスシナリオについては、
　クエスト＞メモリアルクエストから閲覧できます。
・他キャラクターのぶき最終進化、「メモリアルクエスト」の4節は
