イベントで素材をGETしてトレードショップで交換しよう！


イベントでは、イベント素材「ねこ歌の譜面」や「ねこ村の釣り竿」が入手できます。
素材はトレードショップで報酬と交換できます。
 
「★5キャラクター専用ぶき」を進化させるためのアイテム
「エトワリウム」もイベント素材と交換可能です。
「エトワリウム」を入手して、対象のキャラクター専用ぶきを進化させてみましょう。
 

イベントで手に入る素材を集めて、★4  平沢 憂（ナイト　陽属性）や
★5キャラクター専用ぶきの進化に必要なエトワリウムを入手しましょう。
 
その他のイベント報酬については、
トレードショップ画面＞限定交換内の「宮子とねこのうた」からご確認下さい。
 
また、イベント素材の交換期限は10月16日 13:59までとなっておりますので
交換期限までに交換をお願いいたします。

イベント素材を「コンテンツ限定限界突破素材」に交換しよう！

「ねこの不思議なオルゴール」と交換できます。

 「ねこの不思議なオルゴール」は、対応する作品の★5キャラクターなら
クラスを問わず使用できる「コンテンツ限定限界突破素材」と交換できます。

豪華素材がドロップする「【乱戦】クエスト」！

今回のイベントでは、クリアすることで豪華報酬が手に入る「【乱戦】クエスト」が発生します。
【乱戦】クエストは、イベント開始後の2018年10月1日 17:00から開催されます。

 

【乱戦】クエストに挑戦するためには、
【乱戦】マウス・アタック挑戦券を消費する必要があります。
【乱戦】マウス・アタック挑戦券は、
「素材収集クエスト上級」で入手することができます。

 

乱戦クエストをクリアすると、イベント共通の素材と、
まれに★4限界突破素材を入手することができます。
その他、クエスト初クリア時や、一人も撤退せずにクリアした時などに特別な報酬を入手することができます。

 

※使用しない【乱戦】マウス・アタック挑戦券は、
　トレードショップ＞限定交換＞「宮子とねこのうた」から
　「ねこ村の釣り竿」に交換することができます。
