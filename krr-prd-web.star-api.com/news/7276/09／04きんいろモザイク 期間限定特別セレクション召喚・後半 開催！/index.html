きんいろモザイク 期間限定特別セレクション召喚・後半 開催！

『きららファンタジア』をご利用いただき、誠にありがとうございます。

 

9月5日 17:00より、「きんいろモザイク」のキャラクターが
ピックアップされたセレクション召喚・後半を開催いたします。

 

開催期間

9月5日 17:00 ～ 9月11日 13:59

 

ピックアップ対象キャラクター

★5 カレン【舶来の魔術師】（まほうつかい　陽属性）

★5 アリス【クリスマス】（まほうつかい　土属性）

★5 大宮 忍（そうりょ　炎属性）

★5 綾【水着】（ナイト　陽属性）

 

内容

・星彩石40個で1回召喚、400個で10回召喚を行うことができます。
・「10回召喚」では★4以上のキャラクターが1体以上確定します。
・初回のみ星彩石300個で「10回召喚」が可能です。
・期間中は、ピックアップ対象キャラクターの提供割合がアップしています。
・召喚を行った際に、特別アイテム「星屑のかけら」を追加で
　入手することができます。（1回召喚で1個、10回召喚で10個入手できます）
・本召喚で出現するキャラクターの詳細につきましては、
　MENU＞召喚＞召喚バナー＞出現一覧 からご確認ください。

 

1日1回有償限定召喚

「きんいろモザイク 期間限定特別セレクション召喚・後半」開催期間中、
1日1回のみ、有償星彩石10個で1回召喚が行えます。

※1日1回有償限定召喚は、毎日0時に回数がリセットされます。 
※通常の1回召喚、10回召喚では、無償星彩石から使用されます。 
※1日1回有償限定対応の召喚が複数同時に開催されている場合、
　それぞれの召喚につき1回までご利用いただけます。
