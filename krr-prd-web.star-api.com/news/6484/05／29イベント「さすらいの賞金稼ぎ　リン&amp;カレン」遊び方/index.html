イベント「さすらいの賞金稼ぎ　リン&カレン」遊び方

イベントで素材をGETしてトレードショップで交換しよう！

イベントでは、イベント素材「蹄鉄」や
「拳銃型飛び道具」が入手できます。

イベントで手に入る素材は
★5キャラクター専用ぶきの進化に必要なエトワリウムなどと交換できます。


その他のイベント報酬については、
トレードショップ画面＞限定交換内の
「さすらいの賞金稼ぎ　リン＆カレン」からご確認ください。

また、イベント素材の交換期限は6月20日13:59までとなっておりますので
期限までに交換をお願いいたします。
